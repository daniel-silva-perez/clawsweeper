The current moderation code also supports the reported behavior: non-allowlisted static findings such as dangerous command execution and potential exfiltration remain part of the top-level suspicious verdict even when the OpenClaw/LLM verdict is benign with high confidence.

Keep this issue open for maintainer follow-up.
