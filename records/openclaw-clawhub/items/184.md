## Summary

Keep open. The cleanup policy for this repository does not allow auto-closing ClawHub issues, and current main does not implement the requested paid expedited human-review certificate or badge program. The existing implementation has ordinary skill badges, moderator manual-clean overrides, and public wording for cleared security findings, but no payment state, expedited queue, certificate issuance, review criteria, SLA/refund handling, or public paid-review badge semantics.

## What This Changes

Request a paid expedited human review process that awards a certificate or badge to skills that pass staff security review.

## Best Possible Solution

Maintainers should first decide whether ClawHub should offer paid expedited human review at all. If approved, define the trust model separately from #192, then implement it end-to-end across schema and payment state, review queue, moderator workflow, public badge/API display, audit trail, and docs.
