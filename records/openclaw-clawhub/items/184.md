## Summary

Keep open. This is an open ClawHub issue, and the repository cleanup policy only allows auto-closing pull requests that current main definitely implements; substantively, current main has staff moderation overrides and existing badge types, but no paid expedited human-review certificate/badge program.

## What This Changes

Add a paid expedited human security review flow for skills that grants a certificate or badge after manual approval.

## Best Possible Solution

Maintainers should first decide whether ClawHub should offer a paid expedited review program at all. If approved, define the trust model separately from #192, then implement it end-to-end across schema, moderation queue/workflow, payment state, management UI, public API/docs, and badge display copy.
