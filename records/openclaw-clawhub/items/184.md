## Summary

Keep open. This is a ClawHub issue, and the cleanup policy for this repository only allows auto-closing pull requests that current main definitely implements. Substantively, current main has staff manual-clean overrides, official/highlighted/deprecated badge infrastructure, and public review wording for cleared findings, but it does not implement a paid expedited human-review certificate or badge program.

## What This Changes

The issue requests a paid expedited approval pipeline and public certificate/badge for skills that pass human security review.

## Best Possible Solution

Maintainers should first decide whether ClawHub should offer paid expedited human review at all. If approved, define the trust model separately from #192, then implement it end-to-end across schema/payment state, review queue, moderator workflow, public badge/API display, auditability, and docs.
