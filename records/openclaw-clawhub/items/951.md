## Summary

Keep open. Current main still advertises onlycrabs.ai as the SOUL.md/SoulHub registry in README/docs, runtime defaults, and tests, while live checks on April 29, 2026 show onlycrabs.ai is still not connected to the ClawHub/SoulHub app. ClawHub cleanup policy also keeps issues open for maintainer follow-up rather than auto-closing them.

## What This Changes

The issue reports that the README links onlycrabs.ai as the SOUL.md registry even though the domain resolves to parking or unavailable behavior, and proposes removing that section.

## Best Possible Solution

Maintainers should decide the intended product direction: either connect onlycrabs.ai to the production SoulHub experience and add a smoke check, or consistently de-advertise/replace onlycrabs.ai across README, docs, runtime defaults, tests, and user-facing SoulHub copy until the domain is connected.
