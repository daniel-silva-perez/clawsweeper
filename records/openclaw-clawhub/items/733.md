## Summary

Keep open. This is a ClawHub issue, and the repository-specific cleanup policy only permits auto-closing pull requests that current main definitely implements. Substantively, both named skills still report a live suspicious moderation state for `suspicious.potential_exfiltration`; current main intentionally preserves that non-allowlisted static finding even when VT and LLM/OpenClaw are clean.

## Best Possible Solution

The closed #1688 PR is useful prior art but is not merged.
