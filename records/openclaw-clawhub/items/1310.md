Keep open. This is an issue, and the ClawHub cleanup policy only allows auto-closing pull requests that are certainly implemented on current main. ## What This Changes

The issue proposes adding public quality signals or scoring for skills, covering dimensions like documentation completeness, claim accuracy, maintenance, dependency hygiene, complexity, and alternatives awareness.

## Best Possible Solution

Define a public quality-signal model distinct from security/moderation verdicts, then add the agreed schema/API/search support, backfill path, and UI presentation on browse/detail surfaces while keeping #288 as related security-signal context.
