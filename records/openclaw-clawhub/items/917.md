## Summary

Keep open. This is an open ClawHub issue, and the cleanup policy only permits automatic closure for pull requests that current main certainly implements; current main also still blocks deleted GitHub-linked users from signing in again and does not provide an account restore or GitHub identity unlink path.

## What This Changes

The issue requests restoration of an accidentally deleted ClawHub account or unlinking the reporter's GitHub identity so the same GitHub account can register again.

## Best Possible Solution

Define an audited support/admin recovery path for deleted GitHub-linked accounts, or document and perform an operational unlink/restore process for affected users, while preserving ban protections and adding tests for existing deleted-account records.
