## Summary

Keep open. Current main still contains the broad standalone `swap` crypto capability matcher and lacks the proposed `display=swap` and token swap regression tests, while this draft PR is the active focused implementation for related issue #1524 rather than behavior already implemented on main. A security review pass of the provided PR surface found only deterministic matcher/test changes, with no dependency, workflow, lockfile, secret, permission, release, artifact-download, or package-publishing changes.

## What This Changes

This PR narrows crypto capability tagging by replacing the standalone `swap` regex with contextual crypto-adjacent swap patterns and adds regression tests for Google Fonts `display=swap` and real token swap wording.

## Best Possible Solution

Keep this draft PR open for maintainer review. The best path is to land this PR or an equivalent focused change that removes the generic standalone `swap` trigger, preserves real crypto swap detection, adds regression coverage for both `display=swap` and representative DeFi/token swap wording, and then lets related issue #1524 close through the normal merge path.
