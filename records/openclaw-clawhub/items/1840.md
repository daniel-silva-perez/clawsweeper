## Summary

Keep this PR open. Current main does not yet implement the PRâ€™s intended REST/CLI support for publishing org-owned skill versions: the V1 skill publish handler still sends parsed payloads directly to `publishVersionForUser`, the shared skill publish schema/parser still lacks `ownerHandle`, and the skill CLI publish commands still lack `--owner`. Keep #1814 open until this PR or an equivalent fix merges.
