Keep this PR open. The PR is also paired with open related issue #1345 by the same author, and the provided PR diff still has unresolved functional, validation, and data-sync concerns, so ClawHub cleanup policy does not permit auto-closing it.

## Best Possible Solution

Keep this PR open for contributor or maintainer revision if the feature is still wanted. It should also coordinate with related issue #1345.
