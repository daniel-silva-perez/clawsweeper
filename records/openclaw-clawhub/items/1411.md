## Summary

Keep open. Current main does not implement owner-editable skill summaries, and this PR is the open implementation path for the same user problem as related #1345. The proposed branch is not merge-ready because the provided diff and review comments identify authorization, validation, invalid HTML, search digest, and moderation/rescan questions that need maintainer or contributor revision.

## What This Changes

This PR adds a public Convex `updateSummary` mutation plus inline skill-detail summary editing state, header form controls, and CSS so skill summaries can be edited from the skill detail page.

## Best Possible Solution

Keep this PR open for contributor or maintainer revision. A mergeable version should add a validated owner-facing summary update path, align backend authorization with the intended owner/publisher-member/moderator model, trim and cap the summary, update `skillSearchDigest`, make an explicit moderation/rescan policy for changed summary text, render valid JSX, and add focused Convex/frontend tests while coordinating with related #1345.
