Keep open. This is a ClawHub issue, and the repository-specific cleanup policy only permits auto-closing ClawHub pull requests that are already implemented on current main. ## What This Changes

The issue asks why the Ikana/time skill was flagged as suspicious and whether distributing source JavaScript instead of Bun-compiled binaries would resolve it.

## Best Possible Solution

Keep this open for maintainer/moderator follow-up. Recommend removing or strongly de-emphasizing those examples, or replacing them with pinned, checksummed download instructions, then request a rescan.
