Keep open. ## What This Changes

The issue reports an npm installation failure from `curl -fsSL https://molt.bot/install.sh | zsh` caused by a zsh unmatched-glob error in `cleanup_npm_clawdbot_paths`.

## Best Possible Solution

Resolve the installer support boundary explicitly: if the legacy hosted installer remains supported, make its source traceable and harden the zsh cleanup glob for unmatched Homebrew npm paths; if it is retired, redirect or update user-facing install guidance to maintained OpenClaw/ClawHub commands and state the supported Node range.
