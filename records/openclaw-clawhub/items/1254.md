Keep open. Current main still lacks the indexed normalized slug/display-name prefix recall path requested here, and open PR #1256 is the focused implementation candidate with closing syntax for this issue.

## Best Possible Solution

Keep this issue open and review/land PR #1256, or land an equivalent focused implementation.
