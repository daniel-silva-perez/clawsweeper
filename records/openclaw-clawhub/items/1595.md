## Summary

Keep open. The linked `apple-notes-native` skill is already published on ClawHub, and current main has registry, install, and metadata surfaces that can represent a macOS osascript-based skill. The remaining request is maintainer product judgment about whether that community skill should replace or be recommended over an integrated memo-based Apple Notes path, and this repository’s review policy keeps ClawHub issues open unless they are PRs already implemented on main.

## Best Possible Solution

Keep this issue open for maintainer product review. The published `apple-notes-native` ClawHub skill can remain the community-distributed path now; maintainers can decide separately whether to document it as a recommended alternative or track an OpenClaw core change to replace the memo-based Apple Notes default after evaluating reliability, setup friction, and macOS automation permissions/security.
