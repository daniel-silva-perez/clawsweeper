Keep this issue open for maintainer moderation follow-up. If the installer is acceptable, apply the existing manual clean override with an audit note; otherwise ask the owner to publish a safer version that avoids pipe-to-shell/`iex` or adds reviewable package/checksum/source guidance, then request a rescan for the latest release.
