## Summary

Keep open. This is an open ClawHub security issue, and the repository policy only permits auto-closing ClawHub PRs that current main definitely implements. Current main has meaningful registry-side mitigations for obfuscated install-prompt payloads and CLI blocking for malware-flagged installs, but the report also asks for OpenClaw runtime installer sandboxing, explicit prerequisite-script approval, outbound-network controls, and keychain-backed local credential storage, which are not resolved in this ClawHub source tree.
