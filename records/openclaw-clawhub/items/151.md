## Best Possible Solution

The likely implementation is in provider message conversion or session replay: when one assistant turn emits multiple tool_use blocks, coalesce all corresponding tool_result blocks into the immediately following Anthropic user message, or constrain Anthropic-compatible providers to one tool call per assistant turn, with a regression test based on the reporter's JSONL shape.
