## Summary

Keep this PR open. Current main still lacks the PR’s intended indexed direct prefix lookup for skill slug/display-name recall, and the PR is paired with open bug #1254 plus later related search-recall reports. It is not obsolete or implemented on main.

## Best Possible Solution

Keep this PR open for maintainer review, or replace it with an equivalent implementation that adds indexed direct prefix lookup for skill slug/display name, normalized digest fields and indexes, migration-safe backfill for existing digest rows, and regression tests for the Midscene case plus the later `kling` -> `klingai` and ScienceClaw case-stability reports. Close #1254 only after an implementation lands on main.
