## Summary

Keep this PR open. Current main still lacks the PR’s intended indexed direct prefix lookup for skill slug/display-name recall, and the PR is paired with open bug #1254 via closing syntax. The provided discussion also includes a later `kling` -> `klingai` reproduction and related open reports, so this is not obsolete or implemented on main.

## Best Possible Solution

Keep this PR open for maintainer review, or replace it with an equivalent patch that adds indexed direct prefix lookup for skill slug/display name, normalized digest fields and indexes, migration-safe backfill, and regression coverage for Midscene plus the reported `kling` -> `klingai` and case-stability examples.
