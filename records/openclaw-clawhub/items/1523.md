## Summary

Current main does not implement this PR's central production smoke hardening. The current HTTP smoke test still lacks the PR's 429-aware retry and shared skill-detail fetch behavior, and the signed-out upload assertion remains stale against the current /upload redirect flow.

## Best Possible Solution

Keep this PR open for maintainer review and rebase/fix it before merge. Also remove the unreachable retry-helper post-loop code noted in review.
