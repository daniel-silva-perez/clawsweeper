Keep open. This is a ClawHub issue, and the review policy says ClawHub issues should stay open unless maintainers have explicitly handled them. Keep this open for maintainer/security follow-up. Review jimcollinson/x0x v0.17.4 in the live moderation tools; if the reporter’s analysis checks out, apply a clean manual override and reset capability tags for the skill/latest version through the existing moderation paths.
