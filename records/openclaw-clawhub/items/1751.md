Keep open. ## Best Possible Solution

Keep this issue open for maintainer follow-up. A maintainer should review x0x v0.17.4 and the current latest version, use the existing moderator tools to remove false capability tags and apply a clean/manual override if appropriate, then decide whether to harden static scan scoping and capability heuristics with focused tests for cryptographic identity versus cryptocurrency wallets, payments, OAuth, and credentials. Any data correction should also sync affected version, skill, moderation, and search digest records.
