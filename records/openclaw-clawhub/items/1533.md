## Summary

Keep this PR open. Current main still lacks the proposed shared `convex/lib/math.ts` helper and test, the duplicate `clampInt` implementations remain, and the PR has an unresolved behavior question because it would change `skills.ts` and `souls.ts` non-integer pagination limits from `Math.round` to `Math.trunc`.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to deduplicate the helpers while preserving the existing `Math.round` behavior for `skills.ts` and `souls.ts`, or explicitly approve and document the switch to a single `Math.trunc` plus `Number.isFinite` helper before merging.
