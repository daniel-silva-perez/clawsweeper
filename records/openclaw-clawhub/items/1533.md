## Summary

Current main does not implement this PR’s intended deduplication. The shared `convex/lib/math.ts` helper and test are absent, and duplicate `clampInt` helpers remain with differing rounding behavior across Convex modules. The PR still needs maintainer review because it also changes `skills.ts` and `souls.ts` non-integer pagination limits from `Math.round` to `Math.trunc`.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land the deduplication only after explicitly deciding whether `Math.trunc` is acceptable everywhere, or to preserve the previous `Math.round` behavior for `skills.ts` and `souls.ts` while sharing a helper for modules whose semantics match.
