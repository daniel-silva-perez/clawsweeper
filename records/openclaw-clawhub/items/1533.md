## Summary

Keep this PR open. Current main at 3701733797866be20db584cab84f4ba3572ad7ff still does not implement the PR’s intended deduplication: there is no shared `convex/lib/math.ts` or `convex/lib/math.test.ts`, and the module-local `clampInt` copies remain across the Convex files. The existing review concern also remains valid because the PR’s canonical helper would change `skills.ts` and `souls.ts` non-integer limit behavior from `Math.round` to `Math.trunc`.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land the deduplication only after making an explicit decision on the `Math.round` to `Math.trunc` behavior change, or adjust the PR so `skills.ts` and `souls.ts` preserve their current rounding semantics while the shared helper covers the modules where truncation/flooring is intended.
