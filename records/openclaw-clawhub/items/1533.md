## Summary

Keep this PR open. Current main still has the duplicated local clampInt helpers and no shared convex/lib/math.ts helper, so the PR is not implemented on main; the PR also still has an unresolved functional review question because it changes skills.ts and souls.ts pagination rounding from Math.round to Math.trunc.

## What This Changes

This PR adds a shared convex/lib/math.ts clampInt helper with tests and replaces local clampInt definitions across 11 Convex modules with imports.

## Best Possible Solution

Keep this PR open. The best path is to deduplicate clampInt into a shared helper while either preserving the existing Math.round behavior for skills.ts and souls.ts pagination limits or getting explicit maintainer approval for switching those call sites to Math.trunc, then rebase and run the standard checks.
