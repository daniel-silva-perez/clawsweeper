## Summary

Keep open: current main still has duplicate local clampInt helpers, and this PR is not already implemented on main. The proposed shared helper also changes existing finite non-integer limit behavior in skills and souls from rounding to truncation, matching the open review concern already raised in the discussion.

## What This Changes

The PR adds `convex/lib/math.ts` and a test file, then replaces local `clampInt` definitions in 11 Convex modules with imports from the shared helper.

## Best Possible Solution

Keep the deduplication direction, but land it only after preserving existing call-site semantics or explicitly standardizing the integer coercion contract with focused tests for public pagination and maintenance batch-size behavior.
