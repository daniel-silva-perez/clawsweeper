## Summary

Keep open: current main still has the duplicated local helpers, and the proposed shared helper would change existing rounded public pagination limit handling in skills and souls to truncation. This needs maintainer judgment before merge.

## What This Changes

The branch adds a shared `convex/lib/math.ts` `clampInt` helper with tests and replaces local helper definitions across 11 Convex modules with imports.

## Best Possible Solution

Keep the deduplication direction, but land it only after preserving existing call-site semantics or explicitly standardizing the integer coercion contract with focused tests for public pagination and maintenance batch-size behavior.
