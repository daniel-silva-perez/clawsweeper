## Summary

Keep this PR open. Current main still does not include the proposed shared clampInt helper or its tests, and the duplicate local helpers remain across the Convex modules. The existing review concern is also still relevant: the PR would change skills.ts and souls.ts pagination limit behavior from Math.round to Math.trunc for non-integer numeric inputs.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to either preserve the current Math.round semantics for skills.ts and souls.ts while deduplicating the remaining helpers, or document and explicitly approve the switch to a single Math.trunc plus Number.isFinite helper before merging.
