## Summary

Keep open. The repository policy for this ClawHub cleanup pass keeps issues open, and current main still contains the reported publish-to-search disappearance path for ordinary publishers while a new version is pending review.

## What This Changes

Keep already-approved skills searchable while a newly published version is under Instruction Scope and VirusTotal review, and avoid re-triggering review for unchanged content when safe.

## Best Possible Solution

Add a version-aware public projection for skills: keep the last approved card, digest, and searchable embedding visible while a newer version is pending review, then promote the new version atomically after moderation and scanning clear it.
