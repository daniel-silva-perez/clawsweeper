## Summary

Keep open. This is an issue in ClawHub, so this cleanup pass should not auto-close it; current main also still supports the reported publish-to-search disappearance for ordinary publishers during pending scan/review.

## What This Changes

The issue asks for already-approved skills to remain searchable while newly published versions are scanned or reviewed, and for unchanged content not to retrigger review flags unnecessarily.

## Best Possible Solution

Implement a version-aware public projection for skills: preserve the last approved card, digest, and searchable embedding while a newer version is pending review, then promote the new projection only after scanning and moderation clear it.
