## Summary

Keep this issue open. Current main still supports the reported publish-to-search drop for ordinary publishers, and the ClawHub cleanup policy for this review does not auto-close issues.

## What This Changes

The issue requests that previously approved skills remain searchable while a newly published version is pending moderation review, VirusTotal scanning, or LLM evaluation.

## Best Possible Solution

Design a version-aware moderation/search path that keeps the last approved public skill card, digest, and searchable embedding visible while a new version is pending review, then promotes the new projection only after the review clears. Add regression coverage for republishing an already approved skill and decide separately whether unchanged-content review reuse is safe.
