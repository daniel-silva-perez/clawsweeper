## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Expose current installs without relying on title:** `src/components/SkillHeader.tsx:224`
  - body: When `installsAllTime > 0`, the rendered text shows only the all-time install count while `installsCurrent` is available only through the native `title` tooltip. Native title tooltips are not reliably reachable on touch devices or keyboard-only paths, so the current install metric regresses from visible text on main to inaccessible hover-only detail. Keep both values visible or use an accessible tooltip pattern with proper ARIA wiring.
  - confidence: 0.88
