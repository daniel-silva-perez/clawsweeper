## Summary

Keep this issue open. ClawHub cleanup policy keeps issues open, and current main still allows an ordinary publisher update to recompute the skill-level moderation status to hidden, which public search/listing/detail paths filter out.

## What This Changes

A publisher reports that updating a previously visible skill through the web UI/server publish path made it disappear from search and listings, with concern about preserving the existing slug and account ownership.

## Best Possible Solution

Keep this open for maintainer moderation-policy follow-up. The best implementation should preserve slug ownership and avoid hiding already-active skills on clean ordinary updates, while still hiding static-malicious uploads, quality-quarantined releases, publishers under moderation, removed skills, and manual moderation locks; add regression coverage for publish updates plus search, listing, detail, and owner-visible moderation state.
