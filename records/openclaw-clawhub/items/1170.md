Fix the remaining owner-publisher publish pagination hardening gap. Current main schedules package and skill digest syncs separately via scheduleOwnerPublisherDigestSync, but syncPackageSearchDigestsForOwnerPublisherId and syncSkillSearchDigestsForOwnerPublisherId still loop over .paginate() inside one Convex mutation invocation.
