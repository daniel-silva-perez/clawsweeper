## Summary

Keep this issue open. EcoCompute is already available as a published ClawHub skill, but the request asks maintainers to promote it to recommended, official, built-in, or runtime GPU-monitoring behavior. Current main provides publishing, discovery, official/highlighted curation, plugin/package catalog, and runtime metadata surfaces, but no EcoCompute-specific bundled behavior. The repository policy for ClawHub also says open issues should remain open for maintainer judgment.

## Best Possible Solution

Keep this issue open for maintainer triage. EcoCompute can remain available as the published ClawHub skill at https://clawhub.ai/hongping-zh/ecocompute. If maintainers want to promote it, use the existing highlighted/official curation paths; if they want automatic GPU monitoring or bundled runtime behavior, track that as an explicit OpenClaw core/plugin design decision with security and maintenance review.
