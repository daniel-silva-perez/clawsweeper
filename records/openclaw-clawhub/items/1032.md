## Summary

Keep this issue open. The EcoCompute skill is already published on ClawHub, but the request is for maintainer promotion to recommended, official, built-in, or runtime GPU-monitoring behavior. Current main provides registry, metadata, package catalog, and staff/admin curation paths, but it does not bundle EcoCompute or implement automatic GPU energy monitoring. ClawHub review policy also keeps issues open for maintainer judgment.

## Best Possible Solution

Keep this issue open for maintainer triage. EcoCompute can remain available as the published ClawHub skill; if maintainers want to endorse it, use the existing Highlight/Mark official curation path or a documented recommendation channel. If automatic GPU monitoring or bundled runtime behavior is desired, track that as explicit OpenClaw core/plugin design work with security and maintenance review.
