Keep open. This is a ClawHub issue, and the repository-specific cleanup policy only permits auto-closing ClawHub pull requests that are certainly implemented on current main. ## Best Possible Solution

Keep this issue open for maintainer/moderator review of the Overlay Market listing. If the appeal is accepted, use the existing manual override flow with an audit note.
