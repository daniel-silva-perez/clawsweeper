Keep this issue open. Source review also does not prove the deleted-account OAuth/session problem is resolved: current main intentionally deactivates and purges self-deleted users, blocks their reauthentication, returns null from users.me for inactive accounts, and the header can remain in the signed-out UI when no active user document is loaded.
