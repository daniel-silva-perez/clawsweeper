Keep open. This is a ClawHub issue, and the cleanup policy only permits auto-closing ClawHub PRs that are certainly implemented on main. Current main also still lacks catalog-level canonical package/alias de-duplication for the reported duplicate plugin cards; recent related work constrained plugin endpoints to plugin families but did not collapse historical aliases like `openclaw-wework` and `@openclaw/wework`.

Keep this issue open for maintainer follow-up.
