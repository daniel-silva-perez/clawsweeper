## Summary

Keeping this open. The repository policy for this ClawHub cleanup run only allows auto-closing pull requests, not issues. Current main also only partially addresses the request: it exposes structured scanner evidence in some surfaces, but still lacks the requested stable remediation/blocking contract and local `clawhub scan <path> --format json` workflow.

## Best Possible Solution

Keep this issue open. A complete fix should define one stable scanner findings contract covering rule/category, severity, file/line range, redacted snippet or evidence policy, remediation guidance, and whether each finding blocks publish/install or is informational. Then wire it through backend schema/API, owner/staff evidence redaction, web UI, docs, tests, and a local `clawhub scan <path> --format json` CLI workflow for pre-publish remediation.
