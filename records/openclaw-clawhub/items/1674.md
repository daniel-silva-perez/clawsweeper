## Summary

Keep open. This is a live ClawHub production account/moderation appeal, and the repository-specific review policy says ClawHub issues should remain open rather than auto-closed. Current main has ban/unban and automatic publisher moderation-hold machinery, but it does not prove that PHY041 has been restored, and I did not find a shipped admin flow that clears publisher moderation holds or restores skills hidden as user.moderation. The related open PR #1133 remains the clearest recovery-tooling candidate from the provided discussion.
