## Summary

## What This Changes

Publishing or importing a skill should not fail with Convex's multiple-paginated-query error in `syncSkillSearchDigestsForOwnerPublisherId`.

## Best Possible Solution

Verify the deployed scheduler mitigation against production Convex failures and smoke-test CLI publish, upload publish, and GitHub import for affected accounts. If failures remain, refactor both owner-publisher digest sync helpers into cursor-bounded scheduled continuations and then consolidate the related publish/import reports around the same fix.
