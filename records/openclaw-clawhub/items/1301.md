Keep open. This is a ClawHub issue, and the cleanup policy for this repo only allows auto-closing PRs that are certainly implemented on main. Substantively, current main has package trust, official-channel, trusted GitHub Actions publishing, and async scan primitives, but it does not document a community verified-publisher application path or answer whether the reporter's production code-plugin scans were operational after 24+ hours.
