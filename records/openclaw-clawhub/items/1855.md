## Summary

Keep open. Current main still does not implement this PR's intended `clawhub inspect` behavior: the skill detail API returns skill-level moderation and the CLI schema can parse it, but inspect JSON does not include `skillResult.moderation` and plain inspect only prints version-level `security` when a version fetch occurs. This remains a focused draft implementation candidate for #1483.

## Best Possible Solution

Keep this PR open and review or land it, or an equivalent change, as the implementation path for #1483. The final change should pass `skillResult.moderation` through inspect JSON, print a concise skill-level moderation summary in plain `clawhub inspect <slug>` without requiring a separate version fetch, type version `security` in the CLI schema, and retain focused regression tests for both plain and JSON output.
