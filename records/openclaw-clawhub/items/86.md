## Summary

Keep open. The repository policy for this cleanup pass keeps ClawHub issues open, and current main still has no first-class cron/automation-template catalog, preview model, or one-click install path. The current catalog contract is limited to skills, code plugins, and bundle plugins; the related cron form issue is only a tangential OpenClaw dashboard UI bug.

## What This Changes

Request to add a shareable Cron Jobs or Automations catalog where users can browse templates, preview cron configuration, and install scheduled jobs into OpenClaw.

## Best Possible Solution

Keep this issue open for maintainer product judgment. Decide the product model for cron templates first, including manifest shape, preview/install semantics, runtime integration, trust labels, and moderation; then implement the agreed schema/API/UI/CLI/docs path in a scoped follow-up.
