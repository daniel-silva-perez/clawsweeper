## Summary

Keep open. Current main does not provide a shareable cron-job or automation-template catalog, and the ClawHub cleanup policy keeps issues open unless a PR is certainly implemented on main. The existing catalog surfaces are limited to skills, code plugins, and bundle plugins; the cross-referenced Cron form report is a separate dashboard layout bug, not a superseding implementation path.

## What This Changes

The issue requests a Cron Jobs or Automations catalog where users can browse, preview, share, categorize, and one-click install reusable scheduled-job templates.

## Best Possible Solution
