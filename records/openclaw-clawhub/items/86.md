## Summary

Keep open. This is an open ClawHub issue, and the repo-specific cleanup policy keeps ClawHub issues open unless maintainers explicitly resolve them; substantively, current main still has no first-class cron/automation-template catalog, preview model, or one-click install path.

## What This Changes

Add a browseable, shareable catalog of cron job or automation templates with descriptions, cron config preview, categories, and one-click installation into OpenClaw.

## Best Possible Solution

Keep this issue open for maintainer product judgment. Define whether cron templates belong as a new package family, a skill capability/tag, a bundle plugin format, or an OpenClaw runtime/plugin feature, then specify manifest shape, preview/install semantics, trust labels, security prompts, moderation, API/UI support, and runtime integration before implementation.
