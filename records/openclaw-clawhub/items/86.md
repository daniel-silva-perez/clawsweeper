## Summary

Keep open. Current main still lacks a shareable cron-job or automation-template catalog, and the ClawHub cleanup policy for this repository keeps issues open unless they are pull requests already implemented on main. The existing catalog, API, CLI, and UI surfaces support skills, code plugins, and bundle plugins only; the cross-referenced Cron form report is a separate dashboard layout bug rather than a superseding implementation path.

## What This Changes

The issue requests a Cron Jobs or Automations catalog where users can browse, preview, share, and one-click install reusable scheduled-job templates.

## Best Possible Solution

Keep this open until maintainers choose the cron-template product model. If approved, implement the agreed schema, API, UI, CLI, docs, and OpenClaw runtime install contract with explicit preview, consent, trust labels, and moderation semantics.
