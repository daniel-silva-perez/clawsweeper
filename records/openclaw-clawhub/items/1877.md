Substantively, the report is concrete: the staff-cleared moderation state is preserved, but current main still has public security summary and scanner-detail UI paths that read raw latest-version scan fields without applying the clean manual override.

## Best Possible Solution

Keep this issue open and have maintainers decide the public semantics for staff-cleared scan findings.
