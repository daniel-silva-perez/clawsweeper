## Summary

This should stay open. The target-repo policy says ClawHub issues are not auto-closed by this review path, and current main still does not explicitly support .ps1 uploads across the shared allowlist, browser validation, Convex publish validation, and CLI publish/hash paths. Related issue #290 remains open for the same PowerShell upload gap.

## Best Possible Solution

Keep this issue open and either consolidate it with open issue #290 or implement explicit PowerShell support if maintainers accept the policy tradeoff. A complete fix should update the shared and CLI text-file allowlists, browser upload validation, Convex publish validation, archive/hash behavior, tests, and allowed-file documentation.
