## Summary

Keep open. This is an open ClawHub issue, and the repository policy only allows auto-closing pull requests that are certainly implemented on current main. Current main also still lacks the requested reliable registry discovery path for exact slug or slug-prefix enumeration from the CLI.

## Best Possible Solution

Keep this issue open. The best implementation would add an indexed registry discovery path for exact slug and slug-prefix queries, expose it through CLI flags such as `search --exact`, `search --prefix`, or a registry-backed `skills list --source hub --cursor --json` mode, and update API/CLI tests plus public docs.
