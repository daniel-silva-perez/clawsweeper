## Summary

Keep open. ClawHub cleanup policy keeps issues open, and current main still does not provide deterministic exact-slug or slug-prefix enumeration through the skill CLI/API path; the existing surfaces remain relevance search or bounded browsing.

## What This Changes

Add CLI/API support for exact-slug and prefix/list-style registry skill discovery so users can enumerate matching skills instead of relying only on ranked search.

## Best Possible Solution

Add a narrow registry discovery surface that preserves ranked search while exposing exact slug lookup and slug-prefix enumeration through indexed, cursor-paginated backend queries, public v1 API support, CLI output including JSON, and focused docs/tests.
