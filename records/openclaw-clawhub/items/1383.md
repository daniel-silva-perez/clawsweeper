## Summary

Keep open. This is an open ClawHub issue, and the repository policy for this cleanup pass only permits auto-closing pull requests that are certainly implemented on current main. Current main still has ranked CLI search plus bounded registry browsing, but no reliable exact-slug or slug-prefix enumeration path for hub skills.

## Best Possible Solution

Keep this issue open and implement a narrow indexed registry discovery path: exact slug lookup and slug-prefix enumeration on the backend, exposed through CLI flags such as `search --exact`, `search --prefix`, or a hub-list mode, with JSON/cursor support plus API/CLI docs and tests.
