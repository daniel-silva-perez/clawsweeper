## Summary

Keep open. This is an open ClawHub issue, and the repository cleanup policy only permits auto-closing pull requests that are certainly implemented on current main. Current main still lacks reliable exact-slug or slug-prefix registry discovery from the CLI.

## Best Possible Solution

Keep this issue open. The best implementation would add an indexed, cursor-paginated registry discovery path for exact slug and slug-prefix queries, expose it through CLI flags such as `search --exact`, `search --prefix`, or `list --source hub --cursor --json`, and update API/CLI tests plus public docs.
