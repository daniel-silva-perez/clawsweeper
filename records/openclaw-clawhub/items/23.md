## Summary

Keep this issue open. ## What This Changes

The issue requests adding `undici` as a package dependency so Node.js installs of the legacy `clawdhub` CLI do not fail with `ERR_MODULE_NOT_FOUND`.

## Best Possible Solution

A maintainer should make an explicit legacy package distribution decision: publish or retag a fixed stable `clawdhub` package, deprecate `clawdhub@0.3.0` with migration guidance to `clawhub`, or otherwise ensure affected install paths no longer resolve to a package missing `undici`; then run a fresh Node install smoke test and close this with #33 and #51 if the legacy path is resolved.
