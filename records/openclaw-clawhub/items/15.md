## Summary

Keep open. Current main contains partial search-relevance fixes, but it does not prove the reported hosted ranking problem is resolved across skill search, plugin search, all-results, and header typeahead; ClawHub cleanup policy also keeps issues open for maintainer handling.

## What This Changes

The issue reports poor ClawHub search relevance for the query “Remind Me”, where the intended plugin ranked 71st behind results that did not mention the phrase.

## Best Possible Solution

Make exact display-name, package-name, and runtime-id matches reliably outrank semantic-only matches on the intended search surfaces, with a maintainer decision on whether all-results and header typeahead should remain grouped or become globally interleaved.
