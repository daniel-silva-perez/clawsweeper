## Summary

Keep open. ClawHub cleanup policy does not auto-close issues, and current main only partially addresses the reported relevance failure: exact-token filtering and plugin exact-match scoring exist, but the hosted rank for "Remind Me" is not proven and unified all-results ordering still keeps skills and plugins as separate groups.

## What This Changes

The issue reports that searching ClawHub for "Remind Me" ranks the actual plugin far down the list while unrelated skills appear first.

## Best Possible Solution

Keep the issue open until maintainers verify the production rank for "Remind Me" on each search surface; if the rank is still poor, land a focused cross-surface relevance update that makes exact name/display-name matches win consistently without regressing broader semantic search.
