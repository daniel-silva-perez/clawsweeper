## What This Changes

New and existing publishers report that CLI and web publish fail because personal-publisher resolution triggers Convex’s multiple-pagination error and leaves the upload owner dropdown empty.

## Best Possible Solution

A maintainer should deploy or confirm the current backend, smoke-test CLI and web publishing for both a first-time account and an affected existing publisher, then close or consolidate the related reports if production passes. If large-owner digest pagination still fails, the remaining code fix should convert those rebuilds into cursor-safe scheduled batches rather than reopening the original trigger transaction bug.
