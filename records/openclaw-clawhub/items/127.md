## What This Changes

The issue requests fixing public skill detail pages where README/content overflows horizontally instead of fitting responsively within the browser window.

## Best Possible Solution

Add deterministic desktop and mobile coverage for skill-detail README/content overflow using the reported pages or an equivalent fixture, then tighten only the skill-detail and Markdown layout so content wraps or scrolls inside the intended inner container without document-level horizontal overflow or clipping.
