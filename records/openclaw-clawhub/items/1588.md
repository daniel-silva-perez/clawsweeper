Keep open. This is an open ClawHub issue, and the repository policy for this review only permits auto-closing ClawHub pull requests that are certainly implemented on current main. ## Best Possible Solution

Keep this issue open for ClawHub staff triage. Staff should inspect the production `guytogay` user record, audit logs, token state, `deletedAt`/`deactivatedAt`, `requiresModerationAt`, and affected skills. If it is a ban, use the existing unban path and have the user recreate revoked tokens; if it is a moderation hold, land or adapt a #1133-style recovery path that clears the hold and restores only eligible `user.moderation` skills while leaving genuinely malicious or separately moderated skills hidden.
