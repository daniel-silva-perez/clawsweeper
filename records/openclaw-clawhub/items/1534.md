## Summary

Current main still has the reported UI inconsistency: publisher dashboard downloads use ArrowDownToLine, while public skill header/list/shared stats still use Package for download counts. This is also a ClawHub issue, and repository policy says issues should remain open; additionally, open draft PR #1854 is a closing implementation candidate and should be reviewed, merged, or superseded first.

## Best Possible Solution

Keep this issue open and review, land, or supersede #1854. The fix should use a download-specific icon such as ArrowDownToLine or Download for public download metrics in SkillHeader, SkillStats, SkillListItem, and any related public stat surfaces, while preserving Package for versions/package concepts.
