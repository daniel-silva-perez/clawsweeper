## Summary

Keep this issue open. Current main still renders the publisher dashboard download count with ArrowDownToLine, while the public skill header and shared public skill stat/list components still render downloads with Package. The provided GitHub context also shows an open draft implementation PR #1854 with closing syntax, so the right path is to review and land or supersede that PR rather than close the issue now.

## Best Possible Solution

Keep this issue open and review/land PR #1854 or an equivalent focused UI patch that replaces `Package` with `ArrowDownToLine` or `Download` for public download metrics in `SkillHeader`, `SkillStats`, `SkillListItem`, and any remaining public stat surfaces while preserving `Package` for versions/packages.
