## Summary

Keep open. Current main still reproduces the reported UI inconsistency: publisher dashboard download metrics use `ArrowDownToLine`, while public skill header, shared stats, and list rows still use `Package` for downloads. There is also an open draft PR (#1854) whose body says it fixes this issue, so the best path is to review and land that PR or an equivalent patch before closing.

## Best Possible Solution

Keep this issue open and review/land PR #1854 or an equivalent small core UI patch that replaces `Package` with `ArrowDownToLine` or `Download` for public download metrics in `SkillHeader`, `SkillStats`, `SkillListItem`, and any remaining public stat surfaces, while keeping `Package` for versions/packages.
