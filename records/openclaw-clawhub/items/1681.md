## Summary

Keep open. This is an open ClawHub issue, and repo policy only permits automated close proposals for ClawHub PRs that current main certainly implements. Substantively, current main still renders skill README links without a skill-file resolver, keeps non-directory owner/slug skill URLs, and has open draft PR #1858 as the implementation candidate.

## Best Possible Solution

Keep this issue open and review or land PR #1858, or an equivalent current-main fix. Related issue #1001 should be treated as the same fix cluster.
