## What This Changes

Fix publish/import flows failing on first publish with Convex's "multiple paginated queries" error in `syncSkillSearchDigestsForOwnerPublisherId`.

## Best Possible Solution

Implement cursor-rescheduled owner-publisher digest rebuilds: each internal mutation should process at most one indexed paginated page, schedule a continuation with the returned cursor when needed, and cover both package and skill digest rebuilds with regression tests before consolidating the duplicate publish-failure reports.
