## Summary

Keep this draft PR open. Current main still does not implement the intended fix: `clawhub publish` discovers upload files directly through `listTextFiles`, and that discovery path can ignore a root `SKILL.md` before publish validation checks for it. The PR remains a focused implementation candidate for the linked open bug #1509, not obsolete work already shipped on main.

## Best Possible Solution

Keep this draft PR open for maintainer review. The likely path is to land or adapt the localized `cmdPublish` fallback, preserve normal `listTextFiles` ignore semantics, keep the broad-markdown-ignore regression test, and close #1509 only after the fix merges and ships.
