## Summary

Keep this draft PR open. Current main still lacks the PR's intended root-manifest recovery: `cmdPublish` publishes only the `listTextFiles(folder)` result, while `listTextFiles` applies ignore rules before the publish command checks for `SKILL.md`/`SKILLS.md`. The PR remains a focused implementation candidate for the linked open bug #1509.

## Best Possible Solution

Keep this draft PR open for maintainer review. The likely path is to land or adapt the localized `cmdPublish` root-manifest fallback, preserve generic `listTextFiles` ignore semantics, keep the broad markdown ignore regression test, and close #1509 only after the fix merges and ships.
