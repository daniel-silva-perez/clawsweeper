## Summary

This issue should remain open. Current main still lacks the requested ChatGPT/Claude-style left conversation list management: persistent manual ordering, folder/tag/custom grouping, move-between-group actions, and double-click/right-click rename in the chat sidebar.

## Best Possible Solution

Keep this open as a Control UI session-management enhancement. The best path is to design an additive persistent session organization contract in the Gateway/session store for user-defined folders or tags/categories, manual ordering, and move/update operations, then implement the left conversation list UI with drag ordering, group/move actions, and double-click/context-menu rename. Reuse current label support where it fits, and add focused protocol, session-store, Control UI tests, and docs after the product contract is settled.
