## What This Changes

The issue asks OpenClaw to stop replaying an agent's previous tool call after a failed tool result and allow the model to choose a new action, especially in the Feishu/Qwen embedded runtime path.

## Best Possible Solution

Keep this issue open until #74434, #74503, or a narrower successor merges.
