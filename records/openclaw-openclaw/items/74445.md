## Summary

Close as duplicate/superseded: the DeepSeek V4 thinking-profile gap is real on current main, but the same remaining work is already covered by open PR #73008, which predates this PR, targets the same xhigh/max DeepSeek V4 behavior, and includes the provider hook, regression tests, and docs updates. This PR is a focused equivalent implementation, not the canonical review path.

## What This Changes

The PR adds a DeepSeek provider `resolveThinkingProfile` hook and tests so `deepseek-v4-flash` and `deepseek-v4-pro` expose `xhigh` and `max`, while `deepseek-chat` and `deepseek-reasoner` keep base thinking levels.

## Best Possible Solution

Use #73008 as the canonical implementation, or explicitly consolidate the two contributor PRs into one branch, then land a single DeepSeek-owned fix with provider-profile exposure, regression tests, and docs for `/think xhigh|max`.
