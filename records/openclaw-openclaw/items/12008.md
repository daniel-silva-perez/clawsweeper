Keep open. Current main still lacks a typed/schema-validated Gemini safety-settings config surface, and the direct Google Generative AI request path still builds payloads without `safetySettings`. The concrete path is a typed and schema-validated `models.providers.*.safetySettings` config surface, generated schema/help/docs updates, runtime mapping only for `google-generative-ai` requests, explicit per-request precedence, and focused regression tests for config validation and payload injection.
