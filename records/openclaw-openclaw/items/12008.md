Current main still lacks a typed, schema-validated, documented Gemini safety-settings configuration surface, and the direct Google Generative AI request builder still emits payloads without `safetySettings`. ## What This Changes

Add configurable Gemini safety-setting thresholds for Gemini providers or models and map them into Google Generative AI request payloads.
