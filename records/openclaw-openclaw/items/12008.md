Current main still lacks a typed or schema-validated Gemini safety-settings config surface, and the direct Google Generative AI payload builder still returns requests without `safetySettings`. ## What This Changes

Add configurable Gemini safety-setting thresholds for Gemini providers/models and map them into Google Generative AI request payloads.

## Best Possible Solution
