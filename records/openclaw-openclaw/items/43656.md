## Summary

Keep this PR open. Current main still lacks `gateway.peers` and cross-gateway targeting on `sessions_send`/`sessions_spawn`, and this PR is the same-author implementation candidate for the still-open cross-gateway sessions issue #43605. The PR changes remote gateway execution/auth routing, so the right maintainer path is review and hardening, not cleanup closure.
