## Summary

Keep this PR open. Current main still lacks the Feishu reply-path mediaLocalRoots plumbing this PR adds, while the Feishu media sender and core media loader already require scoped roots for local file reads. The PR is a focused implementation candidate; overlapping Feishu media-root PRs exist, but none is merged or clearly canonical yet.

## Best Possible Solution

Keep this PR open for maintainer review or supersede it with one canonical Feishu reply-path fix. The chosen patch should add mediaLocalRoots to CreateFeishuReplyDispatcherParams, compute roots with getAgentScopedMediaLocalRoots at both Feishu bot dispatcher call sites, forward them into sendMediaFeishu, and add a targeted reply-dispatcher regression test for the forwarding. After one canonical fix lands, close or supersede the overlapping Feishu mediaLocalRoots PRs.
