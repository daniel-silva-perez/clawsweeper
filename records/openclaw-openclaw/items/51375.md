## Summary

Keep this PR open. Current main still lacks the Feishu reply-path mediaLocalRoots plumbing this PR adds, while the Feishu media sender and core media loader require scoped roots for local file reads. The related open Feishu media-root PRs corroborate the same gap, but none is merged or clearly canonical yet.

## Best Possible Solution

Keep this PR open for maintainer review, or supersede it with one canonical Feishu reply-path fix. The chosen patch should add mediaLocalRoots to CreateFeishuReplyDispatcherParams, compute roots with getAgentScopedMediaLocalRoots at both Feishu bot dispatcher call sites, forward them into sendMediaFeishu, and add a focused reply-dispatcher regression test for the forwarding. After one canonical fix lands, close or supersede the overlapping Feishu mediaLocalRoots PRs.
