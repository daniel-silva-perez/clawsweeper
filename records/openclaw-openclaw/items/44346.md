## Summary

Keep this PR open. Normal selected-skill reads and sandbox `workspaceAccess: rw` still point agents at the original `SKILL.md`, so the documented feature would be inconsistent if merged as-is.

Public docs and changelog should land only once normal reads, sandbox sync reads, and `workspaceAccess: rw` reads all see the same supported behavior.
