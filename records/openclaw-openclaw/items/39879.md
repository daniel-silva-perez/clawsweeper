## Summary

Current main still does not implement native WhatsApp outbound @mentions. The bundled WhatsApp send path builds plain text/media payloads and quote options, but there is no typed outbound mention option, no @number extraction/resolution, and tests still assert plain payloads. The related implementation PR #56863 remains the right review vehicle if maintainers want to land this feature.

## Best Possible Solution

Keep this issue open until the bundled WhatsApp plugin ships native outbound mention support. A robust fix should add a typed outbound mention path, extract or accept @+<digits>/@<digits> safely, resolve group/LID participant mappings where possible, pass Baileys mention metadata for text and supported media captions, handle the separate visible-text send used for voice notes, and add focused unit coverage. PR #56863 can remain the implementation vehicle if maintainers choose to revive or merge it.
