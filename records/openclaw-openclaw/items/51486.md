## What This Changes

The PR removes the global `schtasks /Query` preflight from `readScheduledTaskRuntime()`, catches task-scoped `execSchtasks` rejections for fallback handling, and adds focused startup-fallback regression tests and fixtures.

## Best Possible Solution
