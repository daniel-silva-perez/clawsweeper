## Summary

Keep this PR open. Current main still lacks the direct-file normalization the PR proposes, and the config/help contract says QMD paths may be files. The PR is a plausible focused bug fix, though it still needs maintainer review for basename-as-glob handling and the source/package mirror split.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is a focused resolver fix that detects existing file targets, converts them to parent-directory QMD collections, uses a literal-safe exact filename pattern, preserves existing directory and missing-path behavior, updates the package mirror if it is still published/consumed, and adds regression coverage for normal filenames and glob-metacharacter filenames.
