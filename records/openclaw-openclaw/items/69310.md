## Summary

Keep this PR open. Current main still lacks the PR's intended dropped-media pipeline: partial media drops are not surfaced to users, catastrophic normalization still clears media without structured diagnostics, and outbound JSON/delivery/transcript/routing/followup paths have no dropped-media carrier. The related #70109 candidate is closed and unmerged, so it does not supersede this active PR.

## Best Possible Solution

Keep this PR open for maintainer review. The best outcome is to land this PR after resolving current head issues and validating the touched lanes, or replace it with an equivalent maintainer-approved implementation that preserves sanitized per-item dropped-media details through normalization, catastrophic fallback, routing/followup filters, outbound JSON/delivery, hooks/audit behavior, and transcript mirroring.
