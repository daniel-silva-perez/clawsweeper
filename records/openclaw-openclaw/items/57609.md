## What This Changes

The PR changes the Mattermost monitor so non-allowlisted group senders can fall through to pending-history recording and then return without generating a bot response.

## Best Possible Solution

Land a narrow Mattermost plugin fix that treats group allowlist as trigger authorization for this path, records visible but non-triggering group messages into bounded pending history, preserves the no-response access-control behavior, and ships with regression coverage plus changelog credit.
