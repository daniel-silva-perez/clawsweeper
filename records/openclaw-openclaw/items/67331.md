## Summary

Keep this PR open. Current main still lacks the proposed standardized filesystem permission-denied surface and mutation payload guards, while the PR diff has unresolved review blockers in sandbox boundary classification and missing-file read semantics. The provided diff does not show supply-chain-sensitive changes such as workflow, dependency, lockfile, package-script, or release metadata edits, but the touched behavior is security-sensitive file mutation and permission reporting, so it needs maintainer review before merge.

This PR adds an agent filesystem permission-denied helper, wraps apply_patch and sandbox filesystem boundary errors with standardized reason metadata, adds write/edit text payload size and NUL-byte guards, and updates related tests.

## Best Possible Solution
