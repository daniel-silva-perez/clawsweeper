## What This Changes

Request to bound `OpenClawChannelBridge` pending Claude permission and approval maps with TTL/sweeping, close cleanup, and a cap so long-running `openclaw mcp serve` sessions do not retain stale entries indefinitely.

## Best Possible Solution
