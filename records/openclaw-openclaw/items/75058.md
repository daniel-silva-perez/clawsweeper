## What This Changes

Fix context compaction so slash-separated file paths survive unchanged instead of being transformed into spaced path fragments after long-context compression.

## Best Possible Solution

Land PR #75093 or an equivalent narrow change that deterministically carries exact path-like literals through provider and LLM compaction, with regression coverage for absolute paths, Windows paths, URL exclusion, and hidden tool-result detail filtering.
