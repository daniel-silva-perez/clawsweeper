## Summary

Keep this PR open. Current main implements the duplicate-webhook part through the shared inbound dedupe path, but it still does not implement the PR's stated Google Chat reply-to default change, and ADDED_TO_SPACE/add-on lifecycle handling is still incomplete. Keep this PR open for maintainer judgment, but do not merge it as-is.
