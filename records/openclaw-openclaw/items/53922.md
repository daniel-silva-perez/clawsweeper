## Summary

Keep this PR open. Current main has the normal active-run WebChat stop path and some stream handling coverage, but it still lacks the PR's heartbeat/session-activity stop flow, Web UI set-heartbeats wiring, sessions.activity protocol/API surface, and explicit progress/milestone prompt guidance. ## What This Changes

This PR adds a WebChat heartbeat/session stop control backed by sessions.activity and broader sessions.abort behavior, updates Web UI/OpenClawKit stream and progress handling, and adds progress/milestone guidance to the agent system prompt.

## Best Possible Solution

Keep this PR open for maintainer review or author revision.
