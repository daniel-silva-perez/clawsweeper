## Summary

Keep this PR open. Current main has absorbed the generic conversation-binding direction and some Feishu ACP/session-binding foundation from #46819, but it still does not implement the full user-visible change this PR targets: Feishu ACP block payloads are still suppressed outside streaming-card handling, Feishu lacks channel config/lifecycle hooks for `threadBindings`, command/docs copy still omits Feishu for `/focus` and `/unfocus`, and the ACP environment change needs a current-path security review before merge.

## Best Possible Solution

Keep this PR open for a focused rebase or replacement against current main: use the generic conversation-binding and plugin SDK surfaces already present, fix non-streaming Feishu ACP block delivery with targeted regression tests, add Feishu `threadBindings` schema/config metadata and lifecycle/update hooks through `conversationBindings`, update `/focus` and `/unfocus` command/docs text, and redo the ACP environment change as a current-path security-reviewed patch.
