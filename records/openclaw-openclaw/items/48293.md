## Summary

This draft PR should stay open. Current main still lacks the proposed pure transcript-tail detector and the prerequisite `agents.defaults.compaction.guard` config surface. The existing compaction safeguard and tool-loop detector are adjacent protections, but they do not provide the PR's detector contract for repeated tool failures, duplicate assistant clusters, stale system recurrence, and trailing user turns without grounded replies. The PR is also part of an active same-author stack for #48238, with #48278 still open as the prerequisite config PR.

## Best Possible Solution

Keep this PR open as the PR2 implementation candidate for #48238. Review or land prerequisite #48278 first, then review this detector on a clean stack diff; close only if current main later lands an equivalent transcript-tail detector or maintainers explicitly abandon the loop-aware compaction guard stack.
