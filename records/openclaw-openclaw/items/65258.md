## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Surface exhausted auth-token mismatches:** `ui/src/ui/app-gateway.ts:375-377`
  - body: This reconnect branch treats `AUTH_TOKEN_MISMATCH` as recoverable because `isNonRecoverableAuthError` intentionally excludes it. Once the browser client has used its device-token retry budget, it stops scheduling reconnects, but this path clears `lastError`/`lastErrorCode` and leaves the UI in a permanent reconnecting state instead of telling the operator to fix credentials.
  - confidence: 0.95
- **[P2] Mark first result-only tool events complete:** `ui/src/ui/app-tool-stream.ts:573`
  - body: When the first event seen for a tool call is a successful `result`, the new entry is initialized as `running` here and never passes through the later existing-entry branch that sets `complete`. Reconnect or seq-gap streams can therefore show already-finished tools as running indefinitely.
  - confidence: 0.92
- **[P2] Preserve attachments when retrying prompts:** `ui/src/ui/chat/grouped-render.ts:1198-1202`
  - body: The retry button forwards only the extracted markdown text. Retrying a multimodal user turn silently drops its files, changing the prompt semantics; either hide retry for attachment-bearing turns or pass the original attachments through the retry path.
  - confidence: 0.9
- **[P2] Route preserved function-call IDs through dedupe:** `src/agents/tool-call-id.ts:456-458`
  - body: Returning canonical `callId|fc_*` IDs before `resolveAssistantId` bypasses the occurrence-aware collision handling. Duplicated replay turns can keep duplicate assistant/tool-result IDs, which is the provider rejection class this sanitizer is meant to prevent.
  - confidence: 0.88
- **[P2] Distinguish shutdown closes from restarts:** `ui/src/ui/app-gateway.ts:382-383`
  - body: `priorShutdownMessage` is set for shutdown events with and without a restart expectation. Using its presence as the restart signal makes an intentional shutdown close show `Gateway restarting.` and suppresses the disconnected error.
  - confidence: 0.88
- **[P2] Honor user-edited session spaces:** `ui/src/ui/chat-spaces.ts:47-50`
  - body: `resolveChatSpace` always prefers the backend `sessionSpace` over the stored local override. For sessions that already have a space, edits snap back on the next render and cannot change grouping.
  - confidence: 0.9
- **[P2] Persist space edits before rerenders:** `ui/src/ui/app-render.helpers.ts:195-199`
  - body: The space field is controlled by `.value=${currentSpace}` but only writes on `change`. Live chat/session updates can rerender before blur and reapply the old value, losing the operator's in-progress edit.
  - confidence: 0.84
- **[P2] Use attachment MIME type in labels:** `ui/src/ui/views/chat.ts:684-690`
  - body: The new context rail hard-codes `Image N` before computing `kindLabel`, so non-image attachments are labelled as images in accessibility labels, titles, and chips.
  - confidence: 0.86
