## Summary

Keep open. Current main does not implement this PR's intended resolved-model reasoning gate for the shared Google thinking sanitizer. The PR is stale against the current shared helper location, but the underlying behavior still needs maintainer follow-up rather than cleanup closure.

## Best Possible Solution

Keep this PR open and ask for a current-main rebase/adaptation. The likely maintainer path is to update `src/plugin-sdk/provider-stream-shared.ts` and the wrapper call path so the sanitizer can use resolved model capability where appropriate, preserve the current Gemini 3/Gemma/adaptive behavior, align Gemini 2.5 fixed-level handling with the native Google transport, and add focused regression coverage for `gemini-2.5-flash` plus a non-reasoning negative case.
