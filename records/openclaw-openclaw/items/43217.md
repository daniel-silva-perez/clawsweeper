## Summary

Keep open. Current main has moved the Google thinking sanitizer into the shared provider stream helper, but it still does not use resolved model reasoning capability and still lacks a fixed-level Gemini 2.5 Flash regression path. The PR is stale and should be rebased, not closed as implemented.

## Best Possible Solution

Keep this PR open and ask for a rebase onto the shared Google thinking helper. The maintainer path is to update `src/plugin-sdk/provider-stream-shared.ts` and its wrapper call path so sanitization can use the resolved model capability, preserve current Gemini 3/Gemma/adaptive behavior, repair fixed-level Gemini 2.5 payloads with the model-family-correct budget mapping, and add a `gemini-2.5-flash` fixed-level regression test plus a non-reasoning negative case.
