Keep open. Current main covers part of the report by always sending Responses requests with array `input` and `stream: true`, and it has a Codex-only guard for system-prompt-only turns, but generic `openai-responses` proxy models still lack stable opt-in controls for fallback top-level `instructions` and for omitting `max_output_tokens`.
