- Add a bounded dynamic action such as `set-suggested-prompts` only through the existing channel message-action contract if it can be typed and gated without introducing Slack-specific core policy.
- Update Slack config types/schema/help hints and Slack docs. Add a user-facing changelog entry with `Thanks @leslielesleyles` if behavior ships.
