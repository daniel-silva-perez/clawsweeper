## Summary

Keep this issue open. Current main still only wires Slack Assistant thread status; it does not accept Slack suggested-prompt config, does not call `assistant.threads.setSuggestedPrompts`, and does not expose a dynamic message action for suggested prompts.

## What This Changes

Add Slack Assistant suggested prompts through validated Slack config and an optional dynamic message action so agents can set contextual prompt suggestions in Slack assistant threads.

## Best Possible Solution

Implement this in the bundled Slack plugin. Add a validated `channels.slack.assistantPrompts` config shape for static suggestions, call Slack `assistant.threads.setSuggestedPrompts` after eligible assistant-thread replies, and add a bounded `set-suggested-prompts` message action only as an additive channel contract. Update Slack docs/config help and focused Slack plugin tests for validation, prompt limits, API calls, and dynamic dispatch.
