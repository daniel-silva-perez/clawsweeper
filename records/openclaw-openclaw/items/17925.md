## Summary

Keep this issue open. Current main implements the Gemini side as a bundled `gemini` web_search provider using Google Search grounding, but the ZAI/GLM native web_search support requested here is still missing. The ZAI plugin has no `webSearchProviders` manifest contract, no `registerWebSearchProvider(...)` call, no ZAI web-search runtime/config/docs/tests, and the related Zhipu/ZAI PRs were closed unmerged.

## Best Possible Solution

Keep this issue open until the bundled ZAI plugin implements GLM/ZAI native search through the existing plugin web-search contract, or an explicitly chosen provider-owned passthrough path, with manifest metadata, config/SecretRef handling, docs, and tests. Gemini is already implemented; any remaining Gemini freshness semantics should stay in the narrower #66498 follow-up.
