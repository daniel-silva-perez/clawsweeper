## Summary

Keep open: the underlying Control UI display URL bug is still valid on current main, but this PR is not ready to merge because it mixes the useful narrow URL fix with stale status-surface hunks and unrelated security/privacy-sensitive agent error-copy and document-intake changes.

## What This Changes

The branch coerces `lan` to `loopback` in configure/status Control UI URL callsites, updates embedded-agent error formatting and tests to add contextual diagnostics and transport hints, and adds a bundled `dropbox-intake` skill with a PDF rendering helper.

## Best Possible Solution

Land a narrow current-main fix that centralizes display-only LAN-to-loopback normalization for browser Control UI URLs and applies it to dashboard, configure completion, status overview/status-all, daemon status, and setup finalization while preserving runtime bind, probe, custom-bind, tailnet, basePath, and TLS semantics. Split the embedded-agent error-copy work and Drop Box document-intake skill into separate reviews.
