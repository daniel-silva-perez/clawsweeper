## Summary

Keep open. Current main still has the sandbox bridge path-mangling bug this PR targets, but the latest PR head rewrites unvalidated `media://` URIs by basename before the existing media-reference validation can reject unsupported hosts or nested IDs, so it should not merge as-is.

## What This Changes

The PR adds early `media://` handling to `resolveSandboxedBridgeMediaPath`, tries inbound fallback resolution through the sandbox bridge, enforces host-path workspace checks, and adds sandbox helper tests.

## Best Possible Solution

Update the PR so sandbox fallback only probes staged inbound files for a validated `media://inbound/<single decoded id>` reference, returns the raw URI for unsupported hosts, malformed URLs, nested IDs, or fallback misses, preserves workspace-boundary enforcement, and adds regression coverage for those rejected cases.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.89

Full review comments:

- **[P2] Validate media URIs before fallback rewrite:** `src/agents/sandbox-media-paths.ts:46-47`
  - body: The new branch rewrites any `media://` URL to an inbound fallback file by taking only `path.basename(new URL(filePath).pathname)`. That lets inputs such as `media://outbound/foo.png` or `media://inbound/a/b.png` load `media/inbound/foo.png` or `media/inbound/b.png` if present, bypassing the existing `resolveInboundMediaReference` checks for hostname and single-segment IDs. Validate the authority and decoded ID first; otherwise return the raw URI for downstream rejection.
  - confidence: 0.91
- **[P3] Handle malformed media URLs before probing fallback:** `src/agents/sandbox-media-paths.ts:46`
  - body: `new URL(filePath)` runs before the fallback probe's `try`, so malformed values like `media://[bad` throw a raw `TypeError` from the sandbox helper instead of falling through to the media resolver's typed invalid-media path. Guard the parse and return the raw URI, or convert the error through the existing media-reference validation path.
  - confidence: 0.76
