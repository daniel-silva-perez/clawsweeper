This PR should stay open: current main still appends the heartbeat workspace-path hint to any resolved heartbeat prompt that mentions `HEARTBEAT.md`, while the docs say configured heartbeat prompts are sent verbatim. The PR adds a user-custom-prompt guard to heartbeat workspace-hint injection and adds focused unit coverage for custom, default, unrelated, and already-hinted prompts.
