## Summary

Keep open: current main and v2026.4.27 still append the workspace HEARTBEAT.md hint to configured heartbeat prompts, despite the docs saying custom heartbeat prompts are sent verbatim. This PR targets a real regression linked to an open issue, but the branch is currently out of date and reported non-mergeable against current main.

## What This Changes

The PR exports the heartbeat workspace-hint helper, adds an isUserCustomPrompt guard derived from configured heartbeat prompt values, and adds focused unit coverage for custom, default, unrelated, and already-hinted prompts.

## Best Possible Solution

Preserve user-authored heartbeat prompts verbatim while retaining workspace-path guidance for built-in default and internally generated heartbeat prompts; refresh or replace the branch so the current due-commitment path uses the same guard, then land with targeted regression coverage.
