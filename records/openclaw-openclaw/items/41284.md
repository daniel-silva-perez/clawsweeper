## Summary

Keep open. Current main still lacks direct `IDEProvisioningTeamByIdentifier` parsing, so the PR is not obsolete and addresses a real documented iOS signing setup path. The PR updates `scripts/ios-team-id.sh` to parse Xcode team data from both `IDEProvisioningTeamByIdentifier` and `IDEProvisioningTeams`, and adjusts the iOS team-id test fixture and expectations.
