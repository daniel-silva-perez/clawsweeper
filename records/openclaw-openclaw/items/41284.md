The PR updates `scripts/ios-team-id.sh` to parse Xcode preferences exported by `defaults` for both `IDEProvisioningTeamByIdentifier` and `IDEProvisioningTeams`, preserves fallback behavior, and adjusts the iOS team-id test fixture and expected selection.

Land the focused signing-helper support after the identifier-key regression case runs the actual inline Python parser against mocked `defaults` and `plutil`, while preserving explicit env overrides, existing `IDEProvisioningTeams`, managed-profile fallback, legacy defaults, optional keychain fallback, and current team-selection order.
