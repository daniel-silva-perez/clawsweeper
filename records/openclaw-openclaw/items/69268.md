## Summary

Keep this PR open. Current main has adjacent shipped fixes for Anthropic adaptive-thinking payloads and outbound assistant-prefill stripping, but it still preserves explicit custom stream functions for supported APIs and lacks the PR's runner-level missing-user-tail/state-cleanup helpers. ## What This Changes

This PR changes pi-embedded Anthropic stream resolution to prefer OpenClaw boundary-aware transports for supported APIs even when a session stream wrapper is present, adds helpers to drop trailing assistant/non-user turns and short-circuit no-input assistant-tail runs, and expands regression coverage around stream resolution and Anthropic turn validation.
