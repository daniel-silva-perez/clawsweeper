## Summary

Keep this PR open. Current main still injects and conditionally strips the Anthropic 1M beta header that this PR is trying to remove, and the PR targets an old embedded-runner file path rather than the active Anthropic plugin implementation. It is also paired with same-author open #45550 via closing syntax, while #45613 is another open implementation candidate but not merged or clearly selected as canonical.

## Best Possible Solution

Keep this PR open for maintainer consolidation. The useful path is to rebase or replace it with a patch against `extensions/anthropic/stream-wrappers.ts`, update `extensions/anthropic/stream-wrappers.test.ts`, the applicable `src/agents/pi-embedded-runner-extraparams.test.ts` coverage, and the Anthropic/token-use docs; preserve `context1m` for context-window sizing in `src/agents/context.ts`; and make an explicit compatibility decision for legacy configured `context-1m-2025-08-07` beta headers under OAuth auth.
