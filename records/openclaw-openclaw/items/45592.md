## Summary

Keep this PR open. Current main still has the beta-header behavior this PR is trying to remove, and the PR patch targets an old embedded-runner file path rather than the active bundled Anthropic plugin implementation. The same-author tracking issue is still open, and #45613 is another open implementation candidate, so closing this PR now would drop a plausible contributor path while the migration remains unresolved on main.

## Best Possible Solution

Keep this PR open for maintainer consolidation. The useful path is to rebase or replace it with a patch against `extensions/anthropic/stream-wrappers.ts`, update `extensions/anthropic/stream-wrappers.test.ts`, `src/agents/pi-embedded-runner-extraparams.test.ts`, and the Anthropic/token-use docs, preserve `context1m` for context-window sizing in `src/agents/context.ts`, and make an explicit compatibility decision for legacy configured `context-1m-2025-08-07` beta headers under OAuth auth.
