## Security Review

Status: needs_attention

Summary: The diff adds no dependency, CI, or supply-chain change, but the new prompt guardrail depends on unresolved file immutability and path-trust semantics.

Concerns:

- **[medium] Define the immutable prompt-file boundary:** `src/agents/system-prompt-file.ts:14`
  - body: The feature is framed as agent-unmodifiable operator guidance, but `readSystemPromptFile()` rereads any configured path per session and does not reject workspace-relative or agent-writable paths.
