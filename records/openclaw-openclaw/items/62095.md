The issue reports post-upgrade v2026.4.5 stability regressions across doctor config repair, subagent announce timeouts, plaintext private-LAN Gateway WebSockets, Slack stale-socket restarts, and Gateway memory growth.

## Best Possible Solution

Keep this open or split it into linked narrow tracks: reduce or justify the subagent announce default, verify Slack Socket Mode cleanup under idle/NAT timeout with diagnostics, keep the private ws:// security contract documented, and close the broad report only after the unresolved stability pieces are either fixed or explicitly superseded by narrower canonical issues.
