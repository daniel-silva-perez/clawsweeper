## Summary

Keep open. Current main still lacks Anthropic advisor support and still has the underlying Anthropic server-side block gaps, while the same reporter has an open implementation PR tracking this exact work. The issue should remain attached to that PR until the PR lands, is replaced, or maintainers explicitly choose a different product direction.

## What This Changes

Support Anthropic's advisor beta by injecting an advisor tool and beta header from model params, plus handling Anthropic server-side tool stream/result blocks in transport replay and display.

## Best Possible Solution

Land #64064 or a maintainer-approved replacement that keeps the owner boundary intact: core transport should handle Anthropic server-side result/display/replay semantics generically, while the bundled Anthropic plugin owns advisor config parsing, beta-header/tool injection, model validation, docs, and focused tests. Follow-up issues should track any intentionally deferred encrypted-content round-tripping or cost-accounting behavior.
