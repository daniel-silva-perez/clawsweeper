## Summary

Current main still has the provider-id guard this PR removes, so `params.context1m: true` still does not unlock 1M context for `github-copilot` or other proxied Claude Opus/Sonnet model refs. The PR remains a focused implementation candidate, but it needs maintainer review/revision rather than cleanup closure.

## What This Changes

The PR removes the provider-id condition from the Anthropic 1M context resolver, adds proxied GitHub Copilot regression tests, and adds a changelog entry for the linked bug.

## Best Possible Solution

Keep this PR open as the canonical implementation surface for the linked bug.
