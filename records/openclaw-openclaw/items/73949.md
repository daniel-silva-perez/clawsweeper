## Summary

## Best Possible Solution

Keep the fix in the Discord plugin lifecycle boundary: land this PR or an equivalent narrow change in `extensions/discord/src/monitor/provider.lifecycle.ts` with the focused regression test and existing contributor credit, without changing the shared gateway health-monitor policy.
