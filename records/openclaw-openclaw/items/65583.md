## Summary

Current main does not implement this PR's central feature. There is no `openclaw agents optimize` command, no optimizer module or command implementation, no Dashboard Optimize/Optimized control, and the GitHub Copilot `auto` virtual model work from the PR is also absent on main. The PR still contains useful unmerged work, but it is large and mixed with security-sensitive packaging/shell changes that need maintainer review rather than cleanup closure.

## Best Possible Solution

Keep this PR open for maintainer review, but require it to be narrowed or split. The model optimization feature should land with focused CLI/UI/docs/tests and browser-safe optimizer code; the GitHub Copilot `auto` work should use a plugin-owned virtual-model seam or an approved temporary exception; Docker, Matrix bundling, and Windows shell execution changes should be separated or strongly justified with targeted security and packaging validation.
