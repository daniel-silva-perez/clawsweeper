## Summary

Current main still does not implement this PR's central feature: no `openclaw agents optimize` command, no optimizer module, no Dashboard Optimize/Optimized control, and no GitHub Copilot `auto` virtual model behavior. The PR remains potentially useful, but it is large and mixes model optimization with Copilot virtual-model behavior plus security/packaging-sensitive script and Docker changes that need maintainer review rather than cleanup closure.

## Best Possible Solution

Keep this PR open for maintainer review, but ask for it to be narrowed or split. The per-agent optimization feature should land as focused CLI/UI/docs/tests with browser-safe optimizer logic; GitHub Copilot virtual-model behavior should use a plugin-owned seam or an approved temporary exception; Docker, Matrix bundling, VS Code, and Windows shell execution changes should be separate PRs or explicitly justified with targeted security and packaging validation.
