## Summary

Keep open. Current main still prepends drained `System:` event blocks into reply prompt text, so the requested bug fix is not already implemented on main. ## What This Changes

The PR routes queued runtime system events out of reply user prompt fields into `followupRun.run.extraSystemPrompt`, updates the prompt-body helper and tests, and currently also changes lazy plugin CLI loading to include the configured memory-slot plugin.
