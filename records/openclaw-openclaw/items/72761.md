Keep open. Current main still routes drained `System:` event blocks into reply prompt text, so the requested bug fix is not already implemented. This PR moves drained queued system-event text from reply user prompt bodies into `followupRun.run.extraSystemPrompt`, updates prompt-body tests, and also changes lazy plugin CLI loading to include the configured memory-slot plugin.
