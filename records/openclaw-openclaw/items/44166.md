## Summary

Keep this issue open. Current main still does not retry the reported transient embedding transport failures or split exhausted multi-item batches, and the same-author implementation PR #44167 is still the active candidate for this exact fix.

## Best Possible Solution

Keep this issue open and use #44167 as the immediate implementation candidate. Review or update it against `extensions/memory-core/src/memory/manager-embedding-policy.ts` and `extensions/memory-core/src/memory/manager-embedding-ops.ts`, then land focused tests for transient transport retry classification, retry-before-split behavior, recursive split ordering, and single-item terminal failure behavior.
