## Summary

Keep this PR open. Current main still awaits `reader.cancel()` on the shared bounded-response reader overflow path, so the intended fail-fast behavior for oversized inbound media is not implemented. The PR remains useful, but it is stale because Matrix now delegates to the shared reader and the unrelated import cleanup is obsolete on current main.

## What This Changes

The branch changes the shared bounded response reader and the old Matrix helper to cancel overflowed streams without awaiting `reader.cancel()`, adds a regression test for a never-resolving cancel, and removes two unused agent imports.

## Best Possible Solution
