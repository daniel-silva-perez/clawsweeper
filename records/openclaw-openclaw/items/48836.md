## Summary

Keep this PR open. Current main still has hardcoded English chat UI labels and does not contain the proposed chat translation keys, so the intended Chinese chat UI localization is not implemented on main. The PR also needs a refresh for the current generated Control UI i18n workflow and the now-separated run controls file.

## Best Possible Solution

Keep the PR open for a refresh against current main: move the remaining hardcoded chat labels to t(...) calls, add English source keys, update ui/src/ui/chat/run-controls.ts, remove or wire closeSessionDetails, then run the Control UI i18n sync/check workflow so generated locale bundles and metadata stay aligned.
