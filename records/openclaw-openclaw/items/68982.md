## Summary

Keep this PR open. Current main still does not implement the PR's intended live chat reasoning path: gateway chat deltas/finals are text-only, Control UI chat stream state is still string-only, and the streaming renderer explicitly disables reasoning. The PR also has useful active review context, including the author's mergeability updates and bot review notes that still need maintainer review before landing.

## Best Possible Solution

Keep this PR open as an implementation candidate. The best path is an end-to-end patch that carries structured assistant stream payloads from gateway chat events through Control UI state into rendering, respects `showThinking` and session `reasoningLevel`, preserves reading indicators for hidden reasoning-only streams, adds focused regression tests, and resolves the existing review comments before landing.
