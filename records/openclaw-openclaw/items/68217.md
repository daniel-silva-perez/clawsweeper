## Summary

Keep this PR open. Current main still has the case-sensitive localized-docs scheme guard without `vbscript:` and no regression coverage for the PR's new `vbscript:` or mixed-case scheme cases. Static review also shows the cited examples already fall through unchanged today because only root-relative paths are localized, so the PR is best treated as a small defensive hardening and coverage improvement for maintainer review rather than an obsolete change.

## Best Possible Solution

Keep this PR in the normal review queue. If maintainers want the defensive invariant, land the bounded `EqualFold` helper plus the `vbscript:` and mixed-case regression tests; if they only want behavior-changing fixes, ask for a current-main failing case or close after maintainer judgment.
