## Summary

Keep this PR open. Current main still lacks the proposed bounded case-insensitive scheme check, still does not treat `vbscript:` as a recognized absolute scheme in `hasURLScheme`, and still has no localized-docs regression cases for `vbscript:` or mixed-case `Javascript:` hrefs. The cited hrefs appear to fall through unchanged today because non-root paths are returned unchanged, so the PR is best handled as a focused defensive hardening and coverage improvement rather than an obsolete PR.

## Best Possible Solution

Keep this PR in the normal review queue. If maintainers want the defensive invariant, land the bounded `EqualFold` prefix helper plus the `vbscript:` and mixed-case regression tests; if they only want behavior-changing fixes, ask for a current-main failing case or close after maintainer judgment.
