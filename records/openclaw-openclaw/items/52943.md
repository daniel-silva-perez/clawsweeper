Keep this PR open. Current main still does not implement iMessage typing indicators, and the PR remains a plausible implementation candidate, but it needs the existing review comments addressed and the upstream imsg RPC/CLI contract rechecked before it should land.

Keep this PR open as the current iMessage typing-indicator implementation candidate.
