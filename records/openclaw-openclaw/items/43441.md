## Summary

Keep open. Current main still has the bundled dist fallback gap, and the PR's production direction is useful, but the regression test still needs a small isolation fix before merge because argv1 package-root discovery can bypass the synthetic fallback path.

## What This Changes

This PR changes workspace template resolution to add module-relative source/dist fallback candidates before cwd and adds a Vitest regression case for bundled dist fallback resolution.

## Best Possible Solution

Land this PR or an equivalent narrow patch after the bundled-layout regression test forces package-root discovery to return null, proving the `dist/../docs/reference/templates` fallback without broadening root-resolution behavior.
