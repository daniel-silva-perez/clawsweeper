## Summary

Keep this PR open. Current main at 5828dcdb05aad587bff831467e7fa51f73312ff8 still has the reported bundled-layout gap: workspace template fallback is a single module-relative ../../docs/reference/templates path, cwd is checked before that fallback, and there is no packageRoot-null bundled-dist regression test. The PR remains a focused implementation candidate, and its latest described head addresses the earlier testability review by deriving fallback directories from the injected moduleUrl.

## Best Possible Solution

Keep this PR open as the implementation candidate. Maintainers should review the focused resolver change, keep package-root lookup first, add a bundled ../docs module-relative fallback before cwd, retain the packageRoot-null bundled-layout regression test, and land only after the relevant changed gate passes.
