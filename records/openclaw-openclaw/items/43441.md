## Summary

Keep this PR open. Current main still has the reported bundled-layout gap: workspace template lookup has only the source-tree `../../docs/reference/templates` fallback, checks `cwd` before that fallback, and lacks a packageRoot-null bundled-dist regression test. The PR remains a focused candidate; its follow-up commit also addresses the earlier testability concern by deriving fallback directories from the resolver's `moduleUrl`.

## What This Changes

The PR updates workspace template directory resolution to try module-relative source and bundled `dist/` fallback paths before `cwd`, and adds a regression test for bundled layout resolution when package-root discovery returns null.

## Best Possible Solution

Land this PR or an equivalent narrow fix that keeps package-root lookup first, computes module-relative fallback directories from the active module URL, checks the bundled `../docs/reference/templates` fallback before `cwd`, and preserves a regression test for packageRoot-null bundled `dist/` resolution.
