## Summary

Keep this PR open. Current main has an adjacent configured-model view for the Control UI picker, but it does not implement this PR's central auth-gated model selector feature: no gateway.controlUi.modelSelector.filter config, no authenticated/configured filter enum, no models.list _meta response, and no credential pre-flight in sessions.patch. The PR also has unresolved security review findings, so it is not a clean obsolete/implemented-on-main close candidate.

## Best Possible Solution

Keep this PR open pending maintainer review. The best path is to rebase or rework it against the current view-based models.list implementation, decide whether gateway.controlUi.modelSelector.filter is the desired product/API shape, and fix the security review issues before considering merge. If maintainers prefer the narrower current-main configured-view approach, close this PR explicitly after pointing contributors to the canonical follow-up rather than marking it implemented now.
