Current main still does not implement native outbound WhatsApp mention metadata. This PR is a real implementation candidate for the open #39879 request and carries forward the closed, unmerged #56863 approach, but it should stay open for maintainer review because automated review found LID resolution edge cases and the behavior can turn outbound @digit text into native group notifications.
