## Summary

Keep this PR open. Current main still lacks `partialJson` transcript repair, and the PR is a focused fix, but it should not merge as-is because provider-owned signed-thinking turns can still preserve complete tool-call blocks with `partialJson` before the new strip path runs.

## What This Changes

The PR updates session transcript repair to strip `partialJson` from complete tool-call blocks, continue dropping incomplete tool-call artifacts, and adds regression tests for retention, dropping, and `sessions_spawn` redaction.

## Best Possible Solution

Merge a version that strips `partialJson` for mutable replay turns and treats `partialJson` as replay-unsafe for provider-owned signed-thinking turns, with regression coverage for both ordinary tool calls and `sessions_spawn` redaction.
