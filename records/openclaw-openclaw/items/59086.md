## What This Changes

The branch adds gateway chat thinking deltas and run mapping, changes embedded agent event ordering and reasoning stream emission, and updates the Control UI to preserve and render structured thinking/text/tool stream blocks.

## Best Possible Solution

Land one current-main-compatible fix that forwards session-scoped thinking deltas through chat, preserves structured stream payloads in the UI, keeps text/reasoning/tool ordering, and suppresses reasoning-only UI rows when reasoning is disabled; use the narrower #68982 path if it covers the intended behavior, otherwise rebase and fix this branch.

## Reproduction Assessment

Yes. Current-main inspection gives a focused reproduction path: a gateway thinking agent event is not converted into a chat delta, and the PR-specific rendering regression can be reproduced by rendering a thinking-only streamMessage with showThinking disabled.

## Solution Assessment

No. The direction is appropriate, but this patch is not the best mergeable solution as-is because it introduces an empty live-stream row when reasoning is hidden and overlaps with a newer active implementation candidate.
