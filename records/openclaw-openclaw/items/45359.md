## Best Possible Solution

Keep the issue open, collect current Codex request IDs/timestamps on v2026.4.27 or newer, and if confirmed add a narrowly gated Codex post-refresh refresh/rebind plus one-time same-selected-model retry only for replay-safe turns with targeted regression coverage.
