Yes. The source-level reproduction path is high-confidence: the gateway broadcasts cumulative assistant text, `app-tool-stream` commits full snapshots before tool cards, and `build-chat-items` renders the cumulative live stream unchanged after committed segments.

## Solution Assessment

No, not as-is. The offset/suffix strategy is the right narrow direction, but this branch needs to be rebased to the current render path, adjusted to preserve whitespace-only deltas, and cleaned of the unrelated binary before it is a maintainable fix.
