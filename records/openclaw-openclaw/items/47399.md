## Summary

Keep open. Current main still lacks the Control UI stream-segment dedupe behavior this PR is trying to add, so the PR is not obsolete; however, the patch is not merge-ready because it drops whitespace-only streamed suffixes, includes an unrelated binary workspace artifact, and needs to be rebased onto the current `buildChatItems` render split.

## What This Changes

The PR adds `chatStreamSegmentOffset` state across Control UI chat/tool-stream code, slices committed and live stream text by that offset, updates test fixtures, adds a chat-view regression test, and includes a new workspace PNG artifact.

## Best Possible Solution

Land one corrected current-main implementation that stores and renders only uncommitted streamed suffixes across tool interruptions, preserves whitespace-only separators, removes the local artifact, and adds focused regression coverage in `app-tool-stream` plus `buildChatItems`.
