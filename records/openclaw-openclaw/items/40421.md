## What This Changes

The PR adds a channel-agnostic bilingual waiting-tip controller, messages.waitingTips config/schema/UI copy, tests, and a Telegram dispatch hook to send, cancel, and delete a temporary tip while the assistant is thinking.

## Best Possible Solution
