Keep this PR open. This PR adds a low/standard/high resource-profile utility and applies low-memory/ARM defaults across FFmpeg/image processing, memory embedding and LanceDB recall, plugin-loader caching, systemd cgroup limits, and Raspberry Pi documentation.
