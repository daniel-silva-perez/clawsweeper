## Summary

Keep this PR open. Current main still does not implement the proposed weighted-recall feedback loop or the three SkillFoundry skills, so it cannot close as implemented. It also should not close as ClawHub-only because the memory-core ranking and feedback behavior is a real bundled memory-plugin feature request, while the new skill documents likely need to be split or moved through ClawHub.

## Best Possible Solution

Keep this PR open, but ask for it to be split or narrowed. The three SkillFoundry skills should go through ClawHub or a bundle-style plugin unless maintainers approve bundling. The memory-core work should continue as a focused PR with a supported way to identify feedback targets, a stable search-result or selector contract, score reordering across FTS/vector/hybrid paths, concurrency-safe persistence, docs, and targeted tests.
