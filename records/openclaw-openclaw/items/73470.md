The issue is also paired with open unmerged PR #73618 using closing syntax, so it should stay open until that PR is reviewed, landed, or closed.

## What This Changes

Requests a mandatory completeness scan guardrail for agent read/search operations before final answers are accepted.

## Best Possible Solution

Keep this open while maintainers define the enforcement scope and review PR #73618. A durable implementation should identify the covered read/search surfaces, preserve incomplete or truncated result state where needed, and add tests proving the agent either performs follow-up scanning or clearly reports blocked/partial evidence before finalizing.
