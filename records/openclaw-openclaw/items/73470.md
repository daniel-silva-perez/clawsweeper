## What This Changes

Request a mandatory guardrail that makes agents complete or explicitly account for read/search operations before finalizing.

## Best Possible Solution

Keep this open while maintainers review PR #73618 and decide the enforcement level. A durable fix should define the covered read/search surfaces, preserve partial/truncated/no-result state, and add tests proving the agent either follows up or clearly reports blocked/partial evidence before finalizing.
