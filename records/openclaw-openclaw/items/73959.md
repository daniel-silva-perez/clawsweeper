## Summary

Keep this PR open. Current main still lacks the proposed Bedrock `modelContextWindowOverrides` config surface, and the linked Bedrock context-window bug remains unresolved by a merged canonical change. The PR is a plausible implementation, but it needs maintainer review against the competing #73382 family-fallback approach and the admin-regex/config-schema tradeoffs before merge.

## What This Changes

This PR adds a Bedrock discovery `modelContextWindowOverrides` regex map, wires it into context-window resolution and cache keys, adds tests, and updates the Amazon Bedrock plugin manifest, exported config type, and provider docs.

## Best Possible Solution
