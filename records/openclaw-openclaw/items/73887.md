## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Preserve final TTS payload before retrying:** `src/auto-reply/reply/dispatch-acp-replay.ts:352-364`
  - body: If `recordProjectorPreparedSyntheticFinal()` throws before it persists the payload, the retry starts with a fresh coordinator and skips already-checkpointed block effects, so `resolveSyntheticFinalPayload()` returns null and final TTS is silently lost. Persist reconstructable payload before this failure window, or replay enough block state to rebuild it, and add a regression for that write-failure path.
  - confidence: 0.84
