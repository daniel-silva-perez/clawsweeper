## Summary

Current main still does not implement the PR's intended TUI user-message color env overrides, while the PR remains a focused source/docs/test change with an unresolved readability edge case around low-contrast override combinations. It should stay open for maintainer review or author update rather than be closed as obsolete.

## What This Changes

This PR adds `OPENCLAW_TUI_USER_BG` and `OPENCLAW_TUI_USER_TEXT` env vars for TUI user-message card colors, documents them, and adds tests for valid and invalid hex override values.

## Best Possible Solution

Keep this PR open as the implementation vehicle. A maintainer or the author should resolve the contrast edge case by deriving, rejecting, or warning on unreadable override combinations, rebase if needed, then run the targeted TUI theme test and docs checks before landing if the env-var surface is still desired.
