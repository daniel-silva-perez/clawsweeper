## Summary

Keep open. Current main still lacks Chat Completions client-tool passthrough and OpenAI-compatible tool_calls output, and PR #66278 is an open, unmerged implementation candidate that explicitly closes this issue.

## Best Possible Solution

Keep this issue open and use PR #66278 as the active implementation candidate. The maintainer path is to review, revise, and land that PR or an equivalent change in src/gateway/openai-http.ts, with focused tests in src/gateway/openai-http.test.ts for tools/tool_choice parsing, non-stream message.tool_calls, streaming delta.tool_calls, and role:"tool" follow-up messages with tool_call_id.
