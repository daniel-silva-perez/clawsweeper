## Summary

Keep this PR open. Current main still has untranslated Chat UI copy, so the requested i18n work is not implemented, but the branch is now stale against the split Chat modules and the generated locale pipeline. The PR needs revision before it can merge: cover the current `ui/src/ui/chat/*` renderers, sync all supported locale bundles, and remove or implement the no-op fallback parameter pattern.

## What This Changes

The PR replaces selected hard-coded Chat view strings in `ui/src/ui/views/chat.ts` with `t("chat.*")` calls and adds matching `chat.*` keys to six locale files.

## Best Possible Solution

Remove these params or add a real typed fallback API with tests; otherwise the code implies graceful fallback behavior that does not exist.
  - confidence: 0.92
