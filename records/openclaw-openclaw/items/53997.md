## Summary

Keep this PR open. Current main still does not implement the PR's wrapper-owned terminal-truth artifact contract, and the PR is stale against the current ACPX architecture, which delegates turns to the packaged `acpx/runtime` instead of the older in-repo CLI wrapper shape. The work is still plausibly useful for post-mortem ACPX debugging, especially #66465, but it needs maintainer rework and a security pass before it can merge.

Keep this PR open for maintainer review, but require a current-main rebase/rework.
