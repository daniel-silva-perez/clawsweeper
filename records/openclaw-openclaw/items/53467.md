Keep this PR open. Current main still lacks Slack `ignoreOtherMentions` in the public Slack config type, strict Zod schema, channel-config resolver, runtime message gate, docs, and Slack tests. The provided PR remains a plausible focused implementation candidate, and the related target-repo PR #55570 is closed unmerged rather than a canonical shipped replacement.

## Best Possible Solution

Keep the PR open and review it as the implementation candidate. The right landed shape is to add `ignoreOtherMentions` to Slack's public config type, strict schema, generated config metadata/docs, channel-config resolution with wildcard fallback, and `prepareSlackMessage` gating, with tests for other-user mentions, explicit bot mentions, DMs, no-mention messages, and degraded bot identity. No new core API appears necessary.
