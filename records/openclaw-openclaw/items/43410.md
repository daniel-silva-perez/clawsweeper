## Summary

Keep open. Current main still has representative bundled skills with capability-only descriptions, canvas still lacks YAML frontmatter and therefore a description, and gh-issues still measures over the reported 5,000-word guide threshold. The open closing PR #44329 covers only three skills and is unmerged, while the broader subset PR #40453 was closed unmerged, so this should remain the umbrella until maintainers decide how to consolidate the metadata pass.

## What This Changes

The issue asks for bundled skill descriptions to include explicit trigger phrases, for canvas to gain a description field, and for the large gh-issues skill to avoid the referenced guide's word-limit edge.

## Best Possible Solution

Use this as the umbrella until maintainers review or close #44329, decide whether to revive or consolidate the #40453-style changes, then land one coherent bundled-skill metadata pass that matches skill-creator guidance, decides whether canvas should be model-loadable frontmatter, and trims or splits gh-issues if the guide limit is adopted.
