## Summary

Keep this PR open. Current main still does not implement the PR's intended usage-only discovery path for preserved deleted-agent transcript directories, and the latest PR branch remains a scoped implementation candidate rather than something superseded by main.

## What This Changes

This PR adds a usage-only disk scan of `agents/<id>/sessions`, lets `discoverAllSessions` accept an explicit `sessionsDir`, resolves preserved transcript files for usage detail endpoints, and adds regression tests for deleted-agent transcript history.

## Best Possible Solution

Review and land this PR, or a maintainer-owned successor, after validating that usage discovery stays separate from `listAgentsForGateway`, carries the real `sessionsDir` through list/detail/timeseries/log paths, handles expected filesystem errors, and has acceptable path/symlink containment for state-dir transcript reads.
