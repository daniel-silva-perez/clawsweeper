## Summary

Keep this PR open. Current main still has the Copilot Claude replay gap the PR targets: Copilot Claude enables thinking-block dropping, but both session-history replay and tool-continuation replay still use the generic helper that preserves latest assistant thinking. The PR remains useful, though it needs maintainer review around normalized provider policy and the bundled ACP/guard-script changes.

## Best Possible Solution

Keep this PR open. The best maintainer path is to refresh it against current main, model Copilot's stricter replay behavior as an explicit normalized provider-policy sanitizer, apply it consistently to session-history replay, tool-continuation replay, and context-pruning estimates, and split or clearly justify the ACP/guard-script allowlist changes before review.
