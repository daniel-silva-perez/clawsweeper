Current main still does not provide official prebuilt Android APK downloads: the public docs say Android is not publicly released, release tooling builds signed AAB bundles, CI validates only debug APK/test lanes, and the latest GitHub release assets are macOS archives only. Prior discussion and related closures already treat this as the canonical tracker for official Android distribution rather than something implemented or safely closeable.
