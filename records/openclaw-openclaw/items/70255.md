## Best Possible Solution

Yes. The source path is high-confidence: hooks reporting builds plugin diagnostics, diagnostics can execute plugin `register()`, and current main does not force-exit after successful hooks output, so a register handler that leaves an open handle can keep the CLI alive.

## Solution Assessment
