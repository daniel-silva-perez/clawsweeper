## Summary

Keep open. Current main still sends Feishu document write, append, and insert content through Feishu's Markdown conversion API without an equation-aware preservation path, so the reported LaTeX backslash loss remains a plausible, narrow bundled Feishu plugin bug with a concrete reproduction.

## What This Changes

Preserve LaTeX backslashes inside Feishu document block formulas and `<equation>` content when writing or updating documents through the Feishu document tool.

## Best Possible Solution

Handle this in the Feishu plugin by preserving raw formula bodies before Markdown conversion and emitting Feishu's supported equation representation for block formulas and `<equation>` tags, with regression tests for the reported formula and no core Markdown contract change unless another plugin needs the same seam.

## Solution Assessment
