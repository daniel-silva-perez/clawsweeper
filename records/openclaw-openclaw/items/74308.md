## Summary

Keep open: current main still documents the opposite completion contract for the bundled coding-agent skill. The PR is a focused docs change that replaces worker self-notification through `openclaw message send` with parent-owned `sessionId` tracking and a structured stdout result block, so it needs normal maintainer review rather than cleanup closure.

This PR rewrites `skills/coding-agent/SKILL.md` so background coding-agent workers print a structured stdout result block while the parent assistant captures the `sessionId`, inspects process logs/status, and reports completion or failure itself.
