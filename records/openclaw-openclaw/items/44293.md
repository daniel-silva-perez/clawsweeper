## Summary

Keep open. Current main removes the direct `xargs` dependency from `format:docs:check`, but the formatter still passes every temporary docs path to a single `spawnSync` call; with 463 tracked docs files, native Windows command-line length remains a real verification gap. The latest release still ships the old `xargs` pipeline, and #48887 remains an open related fix/verification PR, so this should be reconciled before closure.

## Best Possible Solution

Keep this issue open while maintainers reconcile the docs formatter path. The best outcome is to either add chunking/native PowerShell proof to the current `scripts/format-docs.mjs` implementation or explicitly close #48887 as superseded with equivalent proof; after that, close this issue as implemented on main or in the first release that ships the fix. The broader native PowerShell smoke-coverage request in #44291 can remain a separate follow-up.
