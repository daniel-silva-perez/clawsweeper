## Summary

Keep open. Current main already covers the file-picker/drag-drop attachment work and the chat-header model picker, and it also exposes runtime skill/plugin commands through slash completion and the topbar command palette. The remaining requested capability is still not implemented as a discoverable composer-visible skill/capability menu or product-level category picker, so closing would drop a meaningful part of the feature request.
