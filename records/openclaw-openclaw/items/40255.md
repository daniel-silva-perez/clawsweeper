## Summary

Keep open. The latest checked code path still resolves a user-configured heartbeat prompt and then appends the workspace HEARTBEAT.md hint based only on whether the resulting prompt mentions HEARTBEAT.md, which conflicts with the documented verbatim prompt behavior. The issue is also paired with open, unmerged PR #40756 using closing syntax, so it should remain open until that PR or an equivalent fix lands.

## What This Changes

User-configured heartbeat prompts should be honored verbatim without appending the built-in workspace HEARTBEAT.md path hint.

## Best Possible Solution

Land PR #40756 or an equivalent narrow fix that tracks whether the base heartbeat prompt came from user configuration, skips the workspace-path hint for user-authored prompts, preserves the hint for built-in/internal prompts, and adds regression coverage for a configured prompt that mentions HEARTBEAT.md.
