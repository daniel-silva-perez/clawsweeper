## Summary

Keep open. Current main still lacks the proposed `diagnostics.queue` RPC, CLI command, and queue snapshot helper, so this PR is not obsolete; however, the branch is stale against current Gateway diagnostics and command-queue code and would regress existing stability diagnostics, diagnostics export CLI wiring, and singleton-backed queue state if merged as-is.
