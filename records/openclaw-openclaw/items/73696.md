## Summary

Keep this PR open. Current main still does not accept `agents.list[].verboseDefault` or `agents.list[].elevatedDefault` in the strict runtime config schema, and the linked issue remains a live bug. The PR is a plausible implementation candidate, but it still needs maintainer review and likely revision so schema metadata, docs, tests, and elevated-default runtime resolution all agree.

## Best Possible Solution

Keep this PR open for maintainer review and revision. The best path is to make the strict runtime schema, TypeScript config type, `ResolvedAgentConfig`/`resolveAgentConfig`, generated schema metadata, docs, and focused tests all agree for per-agent `verboseDefault` and `elevatedDefault`, then land it as the implementation for #73680.
