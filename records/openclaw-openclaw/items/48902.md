## Summary

Keep this PR open. Current main still has the central hardcoded Control UI strings the PR is trying to localize: theme mode labels, the topbar search affordance, slash-command descriptions, and overview attention titles/descriptions. Main has partial command-palette i18n for base navigation/skill items, so the PR needs a rebase and adaptation rather than cleanup closure.

## Best Possible Solution

Keep the PR open for maintainer/contributor follow-up. Rebase it onto current main, preserve the already-shipped base palette i18n, add the remaining theme/search/slash/overview-attention keys through the Control UI i18n source/generator pipeline, render locale-sensitive attention text without stale cached translations, and add focused tests for the newly localized surfaces.
