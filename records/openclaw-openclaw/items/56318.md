## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P1] Do not stop log output on backpressure:** `src/terminal/stream-writer.ts:49-50`
  - body: `stream.write()` returning `false` only means the writable buffer hit its high-water mark and should drain later, but `src/cli/logs-cli.ts` treats any false writer result as a terminal stop. With this change, high-volume `openclaw logs --follow` output can return early and truncate healthy streams.
  - confidence: 0.93
- **[P2] Read session directories fresh when resolving archives:** `src/infra/session-cost-usage.ts:344`
  - body: `resolveExistingUsageSessionFile` uses this directory listing to fall back from a missing primary JSONL to reset/deleted archives. Caching it for 5 seconds can return a stale primary path after session rotation or reset, causing usage and log loaders to treat the session as missing until the TTL expires.
  - confidence: 0.86
- **[P2] Preserve local shell command semantics:** `src/tui/tui-local-shell.ts:116-117`
  - body: Local `!` commands are documented and implemented as non-interactive shell commands after approval. Parsing them into argv with `shell: false` passes pipes, globs, redirects, and `$VAR` expansion as literal arguments, so existing operator workflows silently misbehave.
  - confidence: 0.82
