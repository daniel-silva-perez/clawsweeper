## Summary

Keep open. Current main still rejects Mattermost `post_edited` WebSocket events before they can reach the existing post, mention, and wake flow, and the linked fix PR #72513 is open and unmerged, so this issue should remain open until that PR or an equivalent focused Mattermost plugin fix lands.

## What This Changes

The issue requests that Mattermost edited-post WebSocket events be evaluated for newly added @mentions so edits can wake the tagged agent.

## Best Possible Solution
