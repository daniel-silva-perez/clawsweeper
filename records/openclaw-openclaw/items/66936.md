## Summary

Keep open. Current main fixes the unresolved SecretRef crash path and the plain JSON `agents list` fast path, but the CLI hang class is still exposed by plugin preload on human `agents list`, bare `openclaw agents`, and `agents add`; related PR #74220 is open for part of that surface, so this should not be closed before that work lands and the remaining add path is handled or split.

## What This Changes

The report asks for `openclaw agents list` to tolerate unresolved Telegram SecretRef tokens and for `openclaw agents` subcommands to exit cleanly after completion.

## Best Possible Solution

Land or adapt #74220 for bare `openclaw agents` and human `agents list`, then handle `agents add` with either a narrow startup-policy change or a generic CLI plugin lifecycle teardown. Closure should wait for built CLI smoke proof that list/add/bind/delete exit without an external timeout when the previously problematic plugin is installed.
