## What This Changes

The PR adds an AsyncLocalStorage-backed re-entrancy guard around internal hook dispatch and tests recursive, sequential, cross-session, and cross-action trigger behavior.

## Best Possible Solution

## Solution Assessment

Yes, pending maintainer acceptance. AsyncLocalStorage is the narrowest maintainable approach shown here because it blocks inherited same-chain recursion without dropping independent concurrent deliveries, addressing the earlier global Map regression feedback.
