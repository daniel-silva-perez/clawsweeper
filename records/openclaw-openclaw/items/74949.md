## What This Changes

The issue reports that bundled plugin runtime-dependency staging can write an incomplete install package manifest, causing npm v11 crashes and npm v10 destructive prune loops during Gateway plugin initialization.

## Best Possible Solution

Review the linked fix PR against current main, validate the npm v10 prune and npm v11 crash paths, then either land the regression test/fix or close the PR as superseded and close this issue only after that maintainer decision is recorded.
