## Summary

Keep open. Current main has the Teams JWT CommonJS/ESM unwrap fix and regression coverage, but the reported problem is still release-facing: the only local release tag is v2026.4.26, and that tag still contains the broken dynamic-import destructuring. The bundled @openclaw/msteams package metadata also still reports 2026.4.25 on current main, so maintainers should verify a shipped artifact before closing.

## Best Possible Solution

Keep this issue open until maintainers cut and verify a release or patch release that includes the current Teams JWT loader fix in the built OpenClaw package and the published/bundled @openclaw/msteams artifact. Before closing, confirm the shipped artifact contains the default-export unwrap, and consider adding warning-level logging for unexpected JWT validator exceptions.
