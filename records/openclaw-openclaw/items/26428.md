Implement a fish-specific completion install/detect path in `src/cli/completion-runtime.ts`: write or copy the cached fish script to `~/.config/fish/completions/<binName>.fish`, detect that path as installed, remove legacy OpenClaw source blocks from `config.fish` safely, and update doctor/setup/update messaging, docs, and focused tests. Review closed PR #27135 as prior art but rebase the behavior onto current main.
