## Summary

Keep open. Current main still emits only a flat bundled-channel load warning, and this PR is a useful active observability fix, but the proposed branch appears to special-case a lightweight entry-load path that current main deliberately keeps off runtime-deps staging, so the actual bundled runtime-deps lock-timeout path still needs maintainer/author follow-up.

## What This Changes

The PR adds file-local bundled runtime-deps lock-timeout detection and metadata parsing in `src/channels/plugins/bundled.ts`, plus new bundled-channel loader tests for lock-timeout versus generic import-error warnings.

## Best Possible Solution

Add the structured lock-timeout classification at the runtime-deps staging/load error path that current main actually uses, keep the full timeout detail visible for pretty/compact logs, and retain focused tests that exercise the real throw path rather than only a mocked entry-loader exception.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.88

Full review comments:

- **[P2] Classify the timeout where runtime deps are staged:** `src/channels/plugins/bundled.ts:261-262`
  - body: This branch is only reached while loading lightweight bundled channel entries, and current main passes `installRuntimeDeps: false` on that path. The timeout from `formatRuntimeDepsLockTimeoutMessage` is thrown by `prepareBundledPluginRuntimeRoot` during bundled runtime-deps staging in the plugin loader, so production lock timeouts will still be recorded without the new `failureReason` metadata. Please move or share this classification on the actual staging/load-error path, or add coverage that proves the real timeout source reaches this branch.
  - confidence: 0.86
- **[P3] Parse the lock path using the waited marker:** `src/channels/plugins/bundled.ts:298-299`
  - body: `indexOf(" (")` stops at the first space-parenthesis sequence after the prefix, so a valid path such as `/tmp/OpenClaw (prod)/...` would emit a truncated `lockDir`. Since this PR adds `lockDir` for structured logging, parse up to the stable ` (waited=` marker or otherwise avoid returning a misleading partial path.
  - confidence: 0.84
