## Summary

Keep this PR open, but do not merge it as-is. It targets a real documented Telegram SOCKS proxy media-download bug, but the patch still leaves the reported media path blocked by the SSRF explicit-proxy guard and advertises SOCKS variants that undici 8.1.0 does not support.

## What This Changes

The PR adds SOCKS scheme detection to the shared proxy fetch helper and Telegram explicit-proxy dispatcher, imports undici `Socks5ProxyAgent`, and adds proxy-fetch unit tests for SOCKS routing and metadata reuse.

## Best Possible Solution

Rework the patch around current main by allowing only dependency-backed `socks5://` and `socks://` explicit proxies through the guarded Telegram media path, preserving existing proxy-host validation and Telegram target SSRF checks, keeping the current HTTP/1 and bounded dispatcher behavior, and removing SOCKS4/SOCKS5H claims unless real support is added and tested.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.92

Full review comments:

- **[P1] Allow SOCKS through the media guard:** `extensions/telegram/src/fetch.ts:276-281`
  - body: The reported Telegram media path reaches `fetchRemoteMedia()` with `dispatcherAttempts`, and current `assertExplicitProxyAllowed()` still rejects any explicit proxy protocol outside `http:`/`https:` before this new dispatcher branch is used. A `socks5://` `channels.telegram.proxy` therefore still fails with `Explicit proxy URL must use http or https`, so the PR does not fix the user-visible media download path.
  - confidence: 0.94
- **[P2] Restrict SOCKS schemes to undici-supported protocols:** `src/infra/net/proxy-fetch.ts:20`
  - body: `isSocksProxyUrl()` sends `socks4://`, `socks4a://`, and `socks5h://` to `Socks5ProxyAgent`, and the Telegram branch repeats the same allowlist, but undici 8.1.0 rejects anything other than `socks5://` or `socks://`. Those configs will fail at dispatcher construction despite the new mock tests, so the branch should stop claiming these schemes or add real support.
  - confidence: 0.91
