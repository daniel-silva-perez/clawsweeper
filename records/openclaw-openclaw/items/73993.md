## What This Changes

The PR changes the Telegram plugin to persist fetched/accepted polling offsets separately from a completed handler watermark, restart from the completed watermark with recovery logging, export the expanded offset snapshot API, add regression tests, and add a changelog entry for the Telegram silent-drop fix.

## Best Possible Solution

## Solution Assessment
