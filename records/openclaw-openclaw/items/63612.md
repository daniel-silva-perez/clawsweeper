## Summary

Keep open. Current main still sends pre-prompt compaction token estimation through the pinned upstream estimator without a local malformed-message guard, and the issue is already paired with open PR #63636 for the intended fix.

## What This Changes

A long-lived main session can become unrecoverable when malformed replay history reaches compaction token estimation and unchecked `.length` reads throw before provider dispatch.

## Best Possible Solution

Land or supersede PR #63636 with a shared guarded estimator, or move history normalization ahead of every compaction/pre-prompt estimation call, with regression tests for malformed assistant, toolResult/custom, bashExecution, and summary messages.
