## Summary

Keep this PR open. Current main still does not implement the PR's central Feishu changes: ops tools/skills, event_id/create_time dedupe, recall-abort buffer handling, direct fast-abort dispatch, semantic streaming/card changes, typing emoji change, or the non-dev daemon working-directory fix. The PR also has security and correctness review concerns that need maintainer triage before it can be considered mergeable.

## Best Possible Solution

Keep this PR open for maintainer triage, but ask for a refreshed split: one focused Feishu streaming/card delivery fix, one event_id/create_time dedupe fix with bounded and account-safe semantics, one recall/fast-abort fix, one daemon working-directory fix, and a separate Feishu ops tools/skills proposal that either lands with account-scoped tests and secret-safety cleanup or moves to ClawHub/plugin packaging.
