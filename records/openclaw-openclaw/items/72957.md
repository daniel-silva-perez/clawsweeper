## What This Changes

The PR adds Control UI chat scroll state, wheel-intent handling, wiring, and regression tests so upward manual scroll during streaming releases auto-follow until the user returns to the bottom.

## Best Possible Solution

Keep this PR open as the active implementation candidate for Control UI chat scroll interruption.
