## Summary

Keep open. Current main still has no `cron edit --model` allowlist warning, so the PR is not obsolete, but the proposed CLI-side warning can disagree with the gateway/runtime behavior it is trying to predict. The branch needs revision before merge because it reads local config for a gateway-targeted command and reimplements only part of runtime model resolution.

## What This Changes

The PR adds a non-blocking `cron edit --model` warning in `src/cli/cron-cli/register.cron-edit.ts` by loading local config/model catalog, building an allowed model set, and printing to stderr when the parsed model key is absent.

## Best Possible Solution

Put the warning or validation in the gateway/runtime context, or expose a gateway-side validation RPC, and have it reuse `resolveCronModelSelection`/`resolveAllowedModelRef` semantics with focused coverage for disallowed refs, aliases, shorthand refs, configured defaults, auth-profile suffixes, and remote-gateway usage.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P2] Validate against the target gateway config:** `src/cli/cron-cli/register.cron-edit.ts:218-219`
  - body: `cron edit` can target a remote gateway via `--url` and `--token`, but this warning reads `loadConfig()` in the local CLI process. If local and remote `agents.defaults.models` differ, the command can warn for a model that will run or stay quiet for one the gateway will reject, so the diagnostic does not reliably predict execution behavior.
  - confidence: 0.93
- **[P2] Reuse runtime model resolution before warning:** `src/cli/cron-cli/register.cron-edit.ts:220-229`
  - body: The warning builds the allowlist with hard-coded defaults and compares a `parseModelRef(model, DEFAULT_PROVIDER)` key, while cron execution resolves configured defaults, aliases, bare refs, and auth-profile suffixes through `resolveAllowedModelRef`. Valid cron model inputs can therefore get a false warning, and the check can disagree with the runtime error path it is trying to preview.
  - confidence: 0.94
