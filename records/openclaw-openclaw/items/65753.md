## Summary

Keep this PR open. Current main still uses the older backtick-only boolean fence handling in the docs link audit, and the in-tree tests do not include the PR’s delimiter parser or tilde/length coverage. The PR remains a narrow, relevant implementation candidate rather than something already implemented or superseded.
