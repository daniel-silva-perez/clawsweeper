## Summary

Keep this PR open. Current main still does not implement the per-agent `tools.sessions.visibility` or `tools.agentToAgent` overrides: the schema/type surface only supports those settings globally, and all session tool call sites still resolve visibility and A2A policy from the global config. The paired feature request #59149 is also open and this PR is its implementation candidate, so it should be reviewed or reworked rather than cleanup-closed.

## Best Possible Solution

Keep this PR open and review or rework it as the implementation candidate for #59149. A clean version should add the per-agent schema/types, thread requester agent tools through all session access call sites, define and test target-only per-agent A2A allow semantics, update the docs/config metadata, and remove any unrelated files before maintainer review.
