## Summary

Keep open. Current main still has no native `openclaw doctor --dry-run` mode: the doctor command does not register or pass a dry-run flag, the typed doctor options cannot carry one, and repair flows can still write config, install bundled plugin runtime dependencies, rewrite cron state, and repair gateway services without a shared dry-run guard. The related global agent dry-run issue and the open `--check-config` PR cover different scopes and do not supersede this focused doctor safety request.

## What This Changes

Request to add `openclaw doctor --dry-run` so doctor validation can preview fix recommendations without writing config or performing repair side effects.

## Best Possible Solution

Add a first-class doctor dry-run contract: register and type `--dry-run`, thread it through the prompter/config/health contribution flows, render the repair plan without prompts or writes, suppress non-config repair side effects, return documented status codes, and update the public doctor docs while keeping #41418 and #47302 separate.

## Repair Work Prompt

Implement `openclaw doctor --dry-run` as a native doctor safety mode. Add the CLI flag in `src/cli/program/register.maintenance.ts`, add a typed `dryRun` option in `src/commands/doctor.types.ts`, and thread it through the doctor config/finalize/health contribution paths so dry-run performs validation and renders the repair plan but never prompts for confirmation, writes config, installs bundled plugin runtime dependencies, saves cron state, builds UI assets, installs/restarts gateway services, or performs other repair side effects. Define and test the requested non-zero result when repairable issues would be found. Keep #41418 global agent dry-run and #47302 `--check-config` out of scope. Update `docs/cli/doctor.md`; add a user-facing changelog entry with `Thanks @Riya-chandra` if the current changelog policy has an active feature section for CLI additions. Do not change dependency pins, package manager behavior, or broaden doctor repair behavior outside the dry-run guard.
