## Summary

Keep this PR open. Current main does not implement the Bedrock Opus 4.7 temperature strip: the Amazon Bedrock provider returns the Anthropic Bedrock stream unchanged for recognized Claude models and only wraps later paths for region/cache-point handling. The linked bug report is open, the earlier same-author PR was auto-closed for queue-cap reasons rather than merged, and this PR is the live implementation candidate. Security pass did not show an obvious supply-chain concern in the provided file list; it changes provider runtime code, a colocated test, and the changelog, not workflows, dependencies, package resolution, scripts, or secret handling.

## Best Possible Solution

Review this PR as the current implementation candidate for #73663, with focus on whether the temperature strip should also use resolved backing-model information for opaque application inference profiles. If the patch passes maintainer review and tests, land it rather than closing it as cleanup.
