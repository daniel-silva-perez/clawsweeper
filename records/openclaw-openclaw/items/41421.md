## Summary

Keep this PR open. Current main still does not implement `openclaw agents view-system-prompt`, and the existing `/context`, `/export-session`, and `/export-trajectory` surfaces do not provide the same no-LLM, agent-selectable CLI prompt preview. The PR remains a plausible implementation candidate, but it needs rebase/rework against current prompt assembly and pluginized channel/provider prompt seams before it can be merged or considered obsolete.

## Best Possible Solution

Keep this PR open as the canonical implementation candidate. The best maintainer path is to rebase it onto current main, implement the preview through shared prompt-assembly helpers or generic plugin/provider prompt seams, remove direct bundled-channel assumptions, and add focused parity tests for model, channel, workspace, sandbox, provider contribution, heartbeat, tool, and metadata behavior.
