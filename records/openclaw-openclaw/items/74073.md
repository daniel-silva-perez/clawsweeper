## Summary

Keep this issue open. Current main still has the reported race shape in the subagent announce/output path, and there is an open, unmerged PR that explicitly targets this issue, so the right next step is PR review rather than closing the report early.

## What This Changes

Prevent subagent completion announcements from using stale raw tool-result JSON when a post-compaction assistant reply arrives after `agent.wait` settles.

## Best Possible Solution

Land a bounded post-wait refresh in the subagent announce/output layer that prefers delayed assistant or silent-token output over raw tool text across both child-findings and no-child-findings paths, with focused regression coverage for raw tool JSON followed by delayed post-compaction assistant output.
