## Summary

Keep this issue open. Current main at 1b25dcf still does not implement Slack App Home support: the Slack plugin enables the App Home Messages tab, but there is no `app_home_opened` subscription, monitor handler, Slack `views.publish` path, Home-tab action/config surface, or docs/tests for publishing Home views. Open PR #73389 is the active implementation candidate, so the cleanup path is to review that PR rather than close this issue now.

## Best Possible Solution

Keep this issue open as the canonical bundled Slack App Home tracker. Review and either land or close #73389; the implementation should live in `extensions/slack`, add `app_home_opened` to setup/docs, register a monitor handler, publish a safe default Home view through Slack `views.publish`, and include focused tests. Broader agent-customized Home-tab actions/config should stay small and reviewed or be split into a follow-up.
