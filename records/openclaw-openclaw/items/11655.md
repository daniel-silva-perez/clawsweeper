## Summary

Keep this issue open. Current main still does not implement Slack App Home support: the Slack plugin enables the App Home messages tab but does not subscribe to `app_home_opened`, register an App Home monitor handler, call `views.publish`, or expose Home-tab config/action/docs. Open PR #73389 is the active maintainer-labeled implementation candidate, so the best cleanup path is to review that PR rather than close this issue now.

## Best Possible Solution

Keep this issue open as the canonical bundled Slack App Home tracker. Review and either land or close #73389; the implementation should live in `extensions/slack`, add `app_home_opened` to setup/docs, register a monitor handler, publish a safe default Home view through Slack `views.publish`, and include focused tests. Any broader agent-customized Home-tab action/config path should stay small and reviewed or be split into a follow-up.
