## Summary

Keep open. Current main does not implement `VALUE.md` as a first-class workspace context file, and the linked implementation PR was closed unmerged. The runtime, gateway file API, sandbox seeding, docs, and template/package smoke checks still enumerate the existing bootstrap files without `VALUE.md`.

## Best Possible Solution

Keep this issue open for maintainer product judgment or a new focused implementation PR. If accepted, implement `VALUE.md` in the core workspace bootstrap path: add a default filename and template, seed it for new workspaces, load and order it in Project Context with documented semantics, include it in minimal/subagent and sandbox paths as intended, expose it through the gateway file API, update template packaging smoke checks, and document it on the public workspace/system-prompt pages.
