## What This Changes

The issue requests reliable exclusion of isolated cron transcripts from Dreaming session corpus ingestion plus configurable Dreaming filters for cron jobs, session-key prefixes, agents, and source-path patterns.

## Best Possible Solution

Keep the issue open while maintainers review #72913.
