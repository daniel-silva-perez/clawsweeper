Keep this PR open. Current main still implements and tests numeric `.bak.1` through `.bak.4` config backup rotation, while this PR is a focused implementation candidate for the still-open datetime-suffix request in #39923. Related PR #40392 is also open, not merged, so there is no shipped implementation or canonical superseding item to close against.
