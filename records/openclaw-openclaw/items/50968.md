## Summary

Keep this PR open. Current main still hardcodes LiteLLM and Copilot Proxy onboarding model limits to 128k/8192 and has no LiteLLM-compatible `/v1/model/info` or `/model/info` capability probe in those plugin-owned paths. The PR is stale against the current plugin layout, but its intended fix is not implemented on main.

## Best Possible Solution

Keep the PR open as a useful bug-fix candidate, but request a rebase/port to the current plugin-owned surfaces: implement capability discovery in `extensions/litellm/onboard.ts` and `extensions/copilot-proxy/index.ts` or an appropriate plugin SDK helper, then add focused tests for discovered limits, fallback behavior, authenticated proxies, trailing `/v1`, and preserving existing limits on probe failure.
