## What This Changes

Google Chat Space/group messages should create and process group sessions instead of returning HTTP 200 without session creation while DMs continue to work.

Keep this open while maintainers review #58993 or a replacement that normalizes Google Chat Space metadata in the Google Chat plugin, adds focused tests for legacy `type`, modern `spaceType`, and `singleUserBotDm`, and verifies group sessions are created under the documented policy.
