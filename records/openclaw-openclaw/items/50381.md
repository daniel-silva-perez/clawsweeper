Keep this PR open. Current main still implements `openclaw logs` as a gateway-log-only command, so the PR is not obsolete or implemented on main. It remains a focused implementation candidate, but it needs contributor/maintainer follow-up on the existing review comments and on the privacy/scope contract for exposing cache-trace-backed LLM diagnostics through `logs.tail`.
