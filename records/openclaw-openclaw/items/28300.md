## Summary

Keep open. Current main implements a narrower Control UI theming surface: Claw, Knot, Dash, and one browser-local tweakcn import slot. It does not implement the requested six-preset gallery, hue-based generator, gradient controls, grouped editor, or per-variable override/reset studio, and the related #49788 preset request shows this remains a product/design decision rather than conservative cleanup.

## What This Changes

Request adding six preset Control UI themes plus a Custom Theme Studio with hue-based generation, gradient controls, per-variable overrides, browser-local persistence, and localized UI copy.

## Best Possible Solution

Define the intended Control UI theming model first. If approved, implement it consistently across the theme contract, CSS token families, browser-local persistence, Appearance and Quick Settings UI, generated/i18n copy, public docs, and focused regression tests while coordinating the narrower preset discussion in #49788.
