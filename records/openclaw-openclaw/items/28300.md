## Summary

Keep open. Current main has a narrower Control UI theme model: Claw/Knot/Dash plus one browser-local tweakcn import slot. It does not implement the requested Ocean/Spring/Sunset/Forest/Purple/Mocha presets or an in-app Custom Theme Studio with hue generation, gradient controls, grouped color editing, and per-variable override/reset UI.

## What This Changes

Add six preset Control UI themes plus an in-app Custom Theme Studio for generated and manually overridden theme variables.

## Best Possible Solution

Keep this open for Control UI product/design handling. Maintainers should decide whether OpenClaw wants built-in preset families, a local Custom Theme Studio, or a narrower expansion of the current tweakcn import slot, then implement the chosen direction coherently across ThemeName/resolveTheme, CSS tokens, browser-local persistence, Appearance and quick settings UI, i18n, docs, and focused UI/theme tests while coordinating with #49788.
