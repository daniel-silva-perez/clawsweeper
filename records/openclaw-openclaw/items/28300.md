## Summary

Keep open. Current main only ships the narrower Control UI theme family model plus one browser-local tweakcn import slot; it does not provide the requested preset gallery, hue-based generator, gradient controls, grouped color editor, or per-variable override/reset studio. The request is valid but broad enough to need maintainer product/design direction before an automated repair PR would be appropriate.

## What This Changes

Request to add six curated Control UI preset themes and an in-app Custom Theme Studio with generated colors, mode controls, gradients, grouped overrides, persistence, and i18n.

## Best Possible Solution

Maintainers should choose a coherent Control UI theming direction, then implement it across the closed theme contract, CSS token families, browser-local persistence, Appearance and quick settings UI, generated/i18n copy, public docs, and focused tests while coordinating any preset work with #49788.
