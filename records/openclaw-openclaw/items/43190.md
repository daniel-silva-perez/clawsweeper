## Summary

Keep this PR open. Current main still lacks the proposed Teams channel archive plugin, the typed `channel_deleted` plugin hook, and Teams `conversationUpdate/channelDeleted` dispatch. ## Best Possible Solution

Keep this PR open for maintainer review. Ask for a current rebase and split the work into a generic typed deleted-channel lifecycle hook plus Teams `channelDeleted` dispatch first, then decide whether durable Teams channel archiving should be bundled or published as an external ClawHub/npm plugin using the resulting APIs. Require focused tests and a security review before merge.
