## Summary

Keep this PR open. Current main still does not include the proposed `msteams-channel-archive` plugin, the typed `channel_deleted` plugin hook, or Teams `conversationUpdate/channelDeleted` dispatch. Some adjacent Teams media and metadata fixes have landed, but they do not replace the PR's central archive/deleted-channel lifecycle work.

## Best Possible Solution

Keep the PR open for maintainer review. Ask for a current rebase and a narrower split: first land a generic, typed deleted-channel lifecycle hook plus Teams `channelDeleted` dispatch and documented message metadata needed by plugins; then decide whether durable Teams channel archiving should be bundled after security review or published as an external ClawHub/npm plugin using those APIs.
