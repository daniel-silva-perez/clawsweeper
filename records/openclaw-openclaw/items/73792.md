Keep this PR open. Current main still lacks the transport-close eviction that this PR is trying to add, and the PR is a focused implementation candidate for a real stale cached MCP runtime path. It should not be closed as implemented or superseded; the remaining work is to address the grounded review concern around intentional close during partial startup failure and then run the targeted changed validation.
