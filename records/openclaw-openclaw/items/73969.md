## Summary

Keep this PR open. Current main still launches POSIX exec shells with plain `-c` through the shared exec shell helper, while this PR narrowly adds startup-file suppression for zsh, bash, and fish plus regression coverage. The source PR #40200 was closed unmerged and explicitly redirected to this replacement, so this remains the active implementation candidate rather than cleanup noise.
