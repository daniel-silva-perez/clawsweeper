## What This Changes

Add a configurable `agents.defaults.bootstrapTier` option so main, subagent, and cron runs can choose `minimal`, `standard`, or `full` bootstrap context loading.

## Best Possible Solution

Review and either land or deliberately replace open PR #22439 with an additive, documented bootstrap-tier contract that preserves current defaults, resolves effective tier from session type plus config override, wires tier selection through bootstrap filtering and hook-loaded extras, updates schema/types/help/docs, and adds focused coverage for main, subagent, cron, lightweight, and extra-file behavior.
