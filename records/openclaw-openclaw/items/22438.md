## Summary

Keep open. Current main still lacks the requested schema-backed `agents.defaults.bootstrapTier` contract and tier resolver; the active implementation candidate is open PR #22439 with closing syntax, so this issue should stay open until maintainers land, replace, or close that PR.

## What This Changes

Add a config-driven `agents.defaults.bootstrapTier` policy (`minimal | standard | full`) to control which bootstrap files main, subagent, and cron sessions load.

## Best Possible Solution

Land or replace #22439 with an additive, documented bootstrap-tier contract that preserves current defaults, wires tier resolution through the bootstrap resolver and extra-files hook, updates schema/types/help/docs, and covers main, subagent, cron, lightweight, and hook-extra behavior in focused tests.
