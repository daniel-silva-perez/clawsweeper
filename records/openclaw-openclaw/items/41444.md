## Summary

Keep this issue open. Current main still gates the Anthropic OAuth beta wrapper on extra-param betas, so model-level `anthropic-beta` alone does not trigger the `oauth-2025-04-20` preservation path. The same-author implementation PR #41461 is also still open with closing syntax, so the issue should wait for that PR to land, be replaced, or be closed.

## Best Possible Solution

Keep this issue open and review, land, or replace PR #41461. The fix should make OAuth beta preservation independent of `params.context1m`, merge or protect `oauth-2025-04-20` when model/provider headers include `anthropic-beta`, strip `context-1m-*` for OAuth/subscription tokens per current docs, and add a regression test for OAuth plus model-level `anthropic-beta` with no `params.context1m`.
