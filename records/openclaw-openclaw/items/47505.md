Keep open. The PR is still useful because current main only covers part of the accessibility surface, especially slash-command menu wiring, while raw config editor labeling/height limits, skip navigation, generated config form control semantics, keyboard-visible chat group footer controls, chat role headings, and sessions sort announcements remain missing.
