## Summary

This PR should stay open. Current main still does not implement the dedicated real-time Agent Activity dashboard proposed here: there is no Activity tab/route, no activity controller/view, and no dedicated activity event stream. #60810 was already closed as superseded by this PR, while #12831 and #54577 remain open broader/narrower activity-observer trackers, so closing this PR now would remove a plausible implementation thread rather than clean up obsolete work.

## Best Possible Solution

Keep this PR open as a possible implementation thread, but require maintainer design review before merge. The best path is to split or rework it into a narrow additive gateway activity contract plus a Control UI Activity panel, align remaining scope with #12831 and #54577, address metadata privacy/localStorage/XSS concerns, regenerate i18n properly, and then close this PR only after a reviewed implementation lands or a clearer canonical replacement is opened.
