## Summary

Keep this PR open. Current main still uses traversal-wide WeakSet tracking in diagnostic payload redaction, so the shared-reference false positive described by the PR is still present. The PR remains a focused implementation candidate with regression coverage and no unrelated security-sensitive file changes in the provided diff.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land the active-recursion-path cycle detection with WeakMap memoization in src/agents/payload-redaction.ts, keep focused regression coverage for shared sibling references and true cycles in the cache-trace/sanitizer tests, and separately decide whether cache-trace digest fingerprinting should align with emitted diagnostic payload semantics.
