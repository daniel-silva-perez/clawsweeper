## Summary

Keep this PR open. Current main still uses traversal-wide `WeakSet` tracking in diagnostic payload redaction, so shared acyclic sibling references are still misclassified as `[Circular]`. The PR targets the right sanitizer and its latest patch adds the requested `WeakMap` memoization plus regression coverage, so the best maintainer cleanup path is review and landing rather than closure.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land active-recursion-path cycle detection with `WeakMap` memoization in `src/agents/payload-redaction.ts`, keep focused regression coverage for shared sibling references and true cycles using the current cache-trace fixtures, and separately decide whether cache-trace digest fingerprinting should use the same shared-reference semantics.
