Keep open: current main still lacks the requested operator Dreaming filters, and this PR contains relevant work for #72611, but the branch is dirty against main and its SDK helper implementation is still on the obsolete `src/memory-host-sdk` source path instead of the packaged memory-host-sdk source used by current runtime exports.

## Best Possible Solution
