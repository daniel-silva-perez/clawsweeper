## Summary

Keep this issue open. Current main still has adjacent Plugin SDK and context-engine lifecycle hooks, but not the requested workspace/internal `HOOK.md` events for `subagent:ended`, `agent:turnComplete`, or a tool-call threshold. The thread also has fresh user discussion and an external memory-layer example, so this needs maintainer API judgment rather than cleanup.

## Best Possible Solution

Keep this open for maintainer API judgment. The likely path is a narrow, documented internal-hook bridge for `subagent:ended` with tests, plus an explicit decision on `agent:turnComplete` and tool-count/threshold semantics. If maintainers prefer lifecycle automation to remain Plugin SDK-only, close later with an explicit product decision and docs pointing users to Plugin hooks and context-engine lifecycle APIs.
