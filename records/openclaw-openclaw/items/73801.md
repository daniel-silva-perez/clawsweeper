Current main has generic Active Memory timeout mitigations, including a hard watchdog, partial timeout recovery, and a per-agent/model circuit breaker, but it does not prove the documented Cerebras gpt-oss-120b Active Memory path aborts cleanly or preserves gateway responsiveness.
