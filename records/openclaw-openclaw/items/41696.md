Keep this PR open, but it needs changes before merge. The reported Discord allowlist bug is still present on current main, while this branch patches the removed `src/discord` resolver instead of the active Discord plugin resolver under `extensions/discord`.

## What This Changes

The PR changes the old Discord resolver and its tests so bare numeric allowlist entries prefer matching guild IDs before falling back to channel lookup while preserving explicit channel and guild prefixes.

## Best Possible Solution
