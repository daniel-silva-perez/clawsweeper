Keep this PR open. Current main still does not expose `email` or `enterprise_email` on Feishu live directory peers, and live peer queries still match only `open_id` and `name`. The PR remains a focused Feishu plugin implementation candidate, with a small test/rebase follow-up and an intentional metadata-exposure review point before merge.
