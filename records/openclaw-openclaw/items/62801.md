## Summary

Keep open. Current main does not implement this PR's intended native plugin runtime API: `runtime.subagent.spawnDetached(...)` and the accompanying SDK types, gateway bridge, registry late-binding wrapper, tests, and docs are absent. Main does have an internal native `spawnSubagentDirect` path used by `sessions_spawn`, but it is not exposed as a plugin runtime surface, so the PR is still a plausible core/plugin-API change rather than already shipped or ClawHub-only work.

## Best Possible Solution

Keep this PR open for maintainer review rather than closing it as implemented or out of scope. The concrete path is to rebase it, review the proposed generic `PluginRuntime.subagent.spawnDetached` contract, verify security policy parity with current `sessions_spawn`/`spawnSubagentDirect`, keep the SDK/docs/tests aligned, and then either land it or explicitly decline the API design.
