## Summary

Keep this PR open. Current main still implements the Feishu automatic mention cascade that the PR is trying to remove: inbound `mentionTargets` are captured, passed into the Feishu reply dispatcher, and converted into outbound Feishu mention markup. The PR is open, unmerged, recently updated, and remains the active implementation candidate for the linked Feishu bug rather than something already shipped on main.

## Best Possible Solution

Keep this PR open for Feishu plugin maintainer review. The best path is to land or revise a focused Feishu plugin fix that stops unconditional inbound mention propagation for fallback, error, ack, plain-text, card, and streaming-card replies; keeps mentioned users as sanitized agent context only; and explicitly preserves or documents intentional @mention behavior with regression coverage.
