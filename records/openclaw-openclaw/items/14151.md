## Summary

Keep open. Current main still does not expose or thread a default-off option to force-prepend an opening <think> tag for local reasoning models. Existing code handles model-emitted reasoning tags, reasoning visibility, malformed tags, and final-tag enforcement, but the requested configurable injection path is still absent. The related PR #14323 is closed unmerged and remains useful context, not a landed fix.

## Best Possible Solution

Implement a default-off, typed forcePrependThinkTag-style option with clear global and per-model precedence. Resolve it before the embedded run, thread it through runner params into subscribeEmbeddedPiSession, inject exactly once before reasoning parsing for streaming and non-streaming assistant text, guard native opening tags, and update schema/docs/tests. Use #14323 as design context, but replace it with a current-main implementation.
