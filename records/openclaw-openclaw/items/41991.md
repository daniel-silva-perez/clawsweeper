## Summary

Keep this PR open. Current main still collapses Gemini CLI OAuth client-config extraction failures into the same generic setup error, while this PR is a narrow diagnostics and regression-test update against the active bundled Google plugin. Open #54289 could supersede it only if maintainers decide to remove or gate automatic extraction.

## Best Possible Solution

Keep this PR open for maintainer review. If Gemini CLI OAuth client-config extraction remains supported, land a narrow diagnostics fix in extensions/google/oauth.credentials.ts with resolveOAuthClientConfig() regression coverage for missing binary, missing credential source, parse failures, and read/extraction exceptions. If #54289 removes or gates extraction, close this PR as superseded by that security/product decision.
