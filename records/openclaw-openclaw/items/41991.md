## Summary

Keep this PR open. Current main still collapses Gemini CLI OAuth extraction failures into the same generic setup error, while the PR remains a focused diagnostics/test update for the active bundled Google plugin. The related security/product issue #54289 may eventually supersede this if maintainers remove or gate automatic extraction, but it has not done so yet.

## Best Possible Solution

Keep this PR open for maintainer review. If Gemini CLI OAuth credential extraction remains supported, land or adapt a narrow diagnostics fix in extensions/google/oauth.credentials.ts with resolveOAuthClientConfig() coverage for missing binary, missing credential source, parse failures, and read/extraction exceptions. If #54289 changes the security/product direction by removing or gating extraction, close or rework this PR as superseded by that decision.
