## Summary

Keep this PR open. Current main still does not pass agent-scoped `mediaLocalRoots` through the Feishu reply attachment path, and this PR is a focused implementation candidate with a targeted regression test and useful author reproduction context.

## Best Possible Solution

Keep this PR open as a valid implementation candidate. The best maintainer path is to choose one canonical Feishu reply-media fix, preserve this PR's regression test or equivalent coverage, and land a rebased patch that forwards `getAgentScopedMediaLocalRoots(cfg, agentId)` into `sendMediaFeishu` for reply attachments.
