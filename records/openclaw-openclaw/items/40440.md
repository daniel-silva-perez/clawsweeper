## Summary

Keep open. Current main still records Telegram no-mention group history as text-only and later replay builds media context only from current-turn and reply media; the two related fix PRs were closed unmerged, so no current implementation covers the reported user problem.

## What This Changes

Telegram group messages with media that are skipped by mention gating should preserve usable media references in pending history so a later @mention can replay them into the agent context.

## Best Possible Solution

Add an additive media-bearing pending-history contract, have the Telegram plugin attach bounded retained media references for mention-gated skipped group messages, merge those references into later MediaPaths/MediaTypes, and cover the image-without-mention followed by @mention replay flow with focused Telegram and shared-history tests.
