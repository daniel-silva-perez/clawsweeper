## Summary

Current main still matches the reported Telegram group-history media loss: skipped mention-gated media messages can be downloaded into `allMedia`, but the pending-history record and later context replay remain text-only. The two linked fix PRs are closed unmerged, and no open PR explicitly targets this issue, so there is no implemented or canonical replacement to close against.

## What This Changes

The issue requests preserving Telegram group-history media references when a media message is stored before the bot is later mentioned.

## Best Possible Solution

Add an additive media-bearing history contract, have Telegram attach bounded media references when storing skipped mention-gated group messages, merge those history media refs into later `MediaPaths`/`MediaTypes`, and cover the image-without-mention followed by @mention replay path with focused Telegram and reply-history tests.
