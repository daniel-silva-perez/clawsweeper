## Summary

Keep open: current main still drops media-bearing Telegram no-mention group messages to text-only pending history, and later replay only feeds current/reply media into `MediaPaths`. The two fix PRs linked from the discussion are closed unmerged, so there is no implemented or canonical replacement yet.

## What This Changes

Preserve downloaded Telegram media references from mention-skipped group history entries so a later @mention can inspect recent images instead of only seeing `<media:image>` placeholders.

## Best Possible Solution

Add an additive media-bearing history contract, have Telegram attach bounded media references when storing skipped mention-gated group messages, merge those references into later `MediaPaths`/`MediaTypes`, and cover the image-without-mention followed by @mention replay path with focused Telegram and shared history tests.

## Repair Work Prompt

Fix the Telegram group-history media loss reported in this issue. When a Telegram group message with media is skipped because mention gating requires an @mention, preserve bounded media references in pending history so a later @mentioned turn can include those prior images/files in `MediaPaths`/`MediaTypes` and the agent can inspect them. Use the closed source PRs #40470 and #40490 as context, but implement against current `extensions/telegram/src` and shared auto-reply code. Keep the contract additive (`HistoryEntry`/`MsgContext.InboundHistory` optional media fields), avoid owner-specific core behavior, and do not introduce unbounded long-lived media retention. Prefer reusing the existing media store or a small documented bounded cache/touch policy over ad hoc persistent files. Add focused tests for: no-mention media history recording, later @mention replay into context media, prompt metadata/history media count if exposed, and existing current-turn media behavior. Add a single user-facing changelog fix entry with appropriate `Thanks @...` attribution, crediting prior report/source PR authors if their ideas are used. Do not change unrelated Telegram mention-gating, access policy, or media download behavior.
