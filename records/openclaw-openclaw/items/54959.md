## Summary

Keep this PR open. Current main does not include the requested OpenShell-in-sandbox deployment guide, and the existing OpenShell doc covers the different host Gateway-managed sandbox backend. The PR adds useful operator docs, but it still conflicts with current proxy lifecycle and Slack proxy support, and it includes security-sensitive deployment guidance that needs maintainer review before merge.

## What This Changes

The PR adds a new OpenShell sandbox deployment guide, registers it in the Platforms docs navigation, and adds OpenShell-related zh-CN glossary entries.

## Best Possible Solution

Publish a maintainer-reviewed OpenShell deployment guide, either as a dedicated Platforms page or folded into the canonical OpenShell docs, that uses the supported proxy.enabled / OPENCLAW_PROXY_URL lifecycle, reflects current Slack/WebSocket proxy behavior, preserves strict SSRF guard boundaries, and scopes advanced supervisor replacement guidance with clear trust and rollback warnings. The glossary additions should be deduplicated before merge.

## Security Review

Status: needs_attention

Summary: The diff is docs-only, but it publishes security-sensitive operator guidance for process-wide proxy hooks and supervisor binary replacement that needs maintainer review.

Concerns:

- **[medium] Custom preload and module monkey-patch guidance:** `docs/platforms/openshell.md:328`
  - body: The guide instructs operators to use NODE_OPTIONS to preload JavaScript that installs a global proxy dispatcher and monkey-patches Module._load for ws. That is process-wide code execution and egress-routing guidance, and it conflicts with OpenClaw's supported proxy lifecycle that clears bypass env vars while active.
  - confidence: 0.88

  - body: The advanced section documents replacing the OpenShell supervisor binary inside the cluster container with docker cp and kill -9. This may be useful for upstream testing, but it normalizes swapping a trusted runtime binary outside a release process without enough trust, source, and rollback framing.
  - confidence: 0.8
