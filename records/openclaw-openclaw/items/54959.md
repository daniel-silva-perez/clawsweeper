Keep this PR open. Current main still lacks the requested OpenShell Platforms deployment page, and the existing OpenShell docs cover host Gateway sandbox delegation rather than operating OpenClaw inside an OpenShell sandbox. ## What This Changes

The PR adds `docs/platforms/openshell.md`, registers `platforms/openshell` in the Platforms nav, and updates the zh-CN docs glossary with OpenShell-related labels.

Keep the contribution under maintainer docs/security review, then either merge a dedicated `platforms/openshell` guide after reconciling it with `docs/gateway/openshell.md`, `docs/security/network-proxy.md`, current fetch-guard behavior, and #58034, or fold the durable operator guidance into the canonical OpenShell docs.
