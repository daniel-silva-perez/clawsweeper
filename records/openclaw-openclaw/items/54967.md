## Summary

Keep open. Current main still resolves Feishu inbound media only from the triggering message, while mention-gated group messages buffered before the bot mention are replayed as text history and are not converted into MediaPaths. The PR is still a focused Feishu channel implementation candidate, with actionable review follow-ups rather than evidence that main already implements it.

## Best Possible Solution

Keep this PR open as a Feishu channel implementation candidate. The best path is to rebase it, preserve raw Feishu body content where pending history resolution needs media keys, resolve a bounded number of pending pre-mention media items before building the inbound media payload, address the existing review comments, and add focused Feishu tests for image-before-mention group history.
