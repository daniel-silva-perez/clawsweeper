## What This Changes

The issue requests a versioned agent truthfulness mechanism with per-claim confidence tiers, source attribution, known-vs-inferred wording, and autonomous-mode reporting of unverified claims.

## Best Possible Solution

Define a versioned truthfulness contract that specifies which agent outputs carry confidence/source metadata, how it crosses agent runtime, task, protocol, transcript, and UI boundaries, and which parts are enforceable code versus advisory prompt guidance.
