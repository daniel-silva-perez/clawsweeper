## Summary

Keep open for maintainer/security review. Current main does not contain the guardrails plugin, and this PR is not a simple optional-integration cleanup: it is tied to security-labeled #72741 and adds a bundled trusted-code plugin with API-key-bearing HTTP providers, dynamic local-code import, runtime packaging, a lockfile change, and open review findings.

## What This Changes

This PR adds a bundled `extensions/guardrails` plugin that registers a `before_dispatch` allow/block check with blacklist, HTTP-provider, and dynamic import connectors, plus package metadata, runtime asset packaging, lockfile updates, and tests.
