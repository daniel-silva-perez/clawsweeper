## Summary

Keep open. Current main still drops manifest-declared plugin skills from loaded runtime records, so the underlying bug is real, but this PR branch is stale and cannot merge as-is because it introduces syntax errors and edits an obsolete registry type surface.

## What This Changes

The PR adds `skills` to plugin records, forwards manifest skills through loader-created records, defaults non-manifest records to an empty array, and adds loader regression coverage.

## Best Possible Solution
