## Summary

Keep this PR open. Current main still does not implement Kimi-specific OpenAI-compatible tool-schema compaction: there is no Kimi cleaner, the shared normalizer only performs Gemini and compat-declared keyword stripping, and NVIDIA/Moonshot do not register a Kimi schema-normalization hook. The supplied discussion also shows the PR had useful review fixes, while #55942 still tracks the NVIDIA NIM Kimi regression as unresolved or insufficiently fixed.

## Best Possible Solution

Keep this PR open as an implementation candidate, but refresh it against the current provider-owned schema-normalization path.
