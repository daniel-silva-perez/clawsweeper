## Summary

Keep open. Current main still has the daemon/foreground quick-start ambiguity and lacks the Node service-runtime docs requested by #72265, while this PR directly addresses that gap and has no blocking correctness or security finding from this read-only review.

This PR updates `README.md`, `docs/install/node.md`, and `docs/gateway/troubleshooting.md` to split daemon and foreground gateway startup guidance, update the README logging link, and add shell-vs-service Node runtime checks with redaction guidance.
