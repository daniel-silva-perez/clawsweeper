## Summary

Keep open. This is a draft docs-only split that intentionally depends on the still-open implementation stack, and current main only has the lower-level host-hook foundation. The added page is useful review material, but as a standalone merge against current main it would publish SDK recipes for APIs that are not present yet.

## What This Changes

This draft PR adds a host-hook examples and recipes documentation page, links it from the plugin docs navigation, and adds zh-CN glossary entries for the new page labels.

## Best Possible Solution

Keep the docs split, but land or rebase it only after the workflow and generic SDK seam follow-ups are merged; until then, either leave the page unpublished from stable nav or trim the examples to APIs already present on current main.

Not applicable as a runtime reproduction; this is a docs PR. The static verification path is high-confidence: compare the added recipe examples to the current OpenClawPluginApi and docs navigation on main.
