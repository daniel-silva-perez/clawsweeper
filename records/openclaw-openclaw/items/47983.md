Current main has WebChat attachment upload support and the non-video file attachment path from #72299 is in the v2026.4.26 changelog, but the exact reported symptom is the paperclip button doing nothing. The current button still triggers the hidden file input through a global `document.querySelector(...).click()` call, and the tests cover file selection/change handling rather than the button-to-picker trigger. This should stay open for a narrow Control UI fix or a current Chrome/Windows retest.
