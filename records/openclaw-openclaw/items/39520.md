## Summary

Keep this PR open. Current main still does not implement the requested Feishu-owned `channels.feishu.localRoots` or account-scoped `localRoots` media upload configuration. Main only forwards caller-provided `mediaLocalRoots` arrays in some send paths, while the PR adds the missing Feishu config field, merged-account resolution, `"any"` handling, and targeted media tests.

## Best Possible Solution

Keep the PR open for maintainer review or land an equivalent Feishu plugin-owned patch: add `localRoots: "any" | string[]` to shared top-level/account Feishu config, resolve it from the merged account config in `sendMediaFeishu`, honor explicit empty arrays, inject raw `readFile` only for explicit `"any"`, document the security implications, and test top-level config, account overrides, `"any"`, empty arrays, and no `readFile` for array roots.
