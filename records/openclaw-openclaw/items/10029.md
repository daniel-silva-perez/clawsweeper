## Summary

Keep open. Current main still renders WebChat/Control UI code blocks as escaped plain code with existing code-block chrome, and the existing highlighter code is limited to transcript export rather than the live WebChat renderer.

## What This Changes

Request adding syntax highlighting for WebChat Markdown code blocks, including language detection and theme-compatible styling.

## Best Possible Solution

Add syntax highlighting in the Control UI markdown rendering layer or a small helper used by it, with theme-aware styles and regression coverage that proves highlighting coexists with copy buttons, language labels, JSON collapse, sanitization, cache behavior, and oversized-message fallback.
