## Summary

Keep this PR open. Current main still has the model-usage documentation TODO the PR targets, and the underlying Python script already supports the portable `--input` path. The PR is not implemented on main, but it should be revised before merge to address the review feedback around macOS-only skill eligibility and the ambiguous “install path above” wording.
