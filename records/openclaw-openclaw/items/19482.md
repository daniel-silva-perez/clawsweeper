The PR remains a focused bug-fix candidate, but it needs a rebase because current main uses `src/agents/workspace.test.ts` and no longer has the PR's `src/agents/workspace.e2e.test.ts` path.

## Best Possible Solution

Rebase it onto current main, move/refit the regression coverage into `src/agents/workspace.test.ts`, set up the negative fixture with all starter files except `HEARTBEAT.md`, guard `writeFileIfMissing(heartbeatPath, heartbeatTemplate)` behind `isBrandNewWorkspace`, then land after targeted workspace and heartbeat validation.
