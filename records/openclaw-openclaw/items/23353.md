## Summary

Keep this issue open. Current main still does not implement the broad Anthropic native server-tools request: there is no Anthropic serverTools/PTC/allowed_callers config surface, no Anthropic server-tool injection in the payload path, no explicit server-tool result block handling, and no bundled Anthropic web-search/web-fetch provider contract. The related web-search-only work remains narrower than this issue, while the earlier server-tools/PTC PRs were closed unmerged.

## Best Possible Solution

Keep this issue open as the broad Anthropic native server-tools tracker. Let #49949/#50692 cover the narrower native web_search provider if it passes review, then separately design web_fetch, code_execution/PTC, allowed_callers, server content-block preservation/rendering, docs, tests, and security defaults for the broader request.
