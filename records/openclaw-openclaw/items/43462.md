The PR lowercases memory-directory prefix checks in old `src/memory` helpers and adds regression tests for mixed-case memory paths and temporal decay.

Repair or replace the PR against the active memory-host SDK and `memory-core` modules: normalize memory paths once, compare the default `memory/` directory prefix case-insensitively, lowercase before dated filename parsing, preserve canonical `MEMORY.md` and `DREAMS.md` behavior, and add focused mixed-case regression tests.
