Keep open. ## What This Changes

The issue requests fixing a v2026.4.29 gateway-backed TUI startup hang where the bundled provider plugin loader repeatedly reloads provider plugins until the TUI spins at 100% CPU.

Any code fix should stay in the TUI gateway startup, provider hook cache, model catalog, or bundled runtime-deps convergence paths rather than globally disabling bundled plugins.
