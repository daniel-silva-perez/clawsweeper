## Summary

Keep open. Current main still lacks a Mattermost draft-preview opt-out: the schema is strict and has no `streaming` field, the runtime unconditionally creates the draft stream, and the shared/docs surface points to `streaming.mode: "off"` as the canonical opt-out. This PR remains a live implementation candidate for open bug #73211, not obsolete cleanup.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to implement the Mattermost preview-streaming opt-out through schema, types, account resolution, runtime selection, tests, and docs, preferably by wiring the existing `streaming.mode: "off"` contract unless maintainers intentionally want a separate `draftPreview` flag. After the implementation lands, #73211 can be closed by the merged change.
