Keep open. Current main fixes the Control UI refresh-button slice and has heartbeat display filtering, but the delivery mirror still turns outbound media into filename-only transcript text, and the Docker/WSL2 reconnect plus Telegram typing-gap symptoms are not proven fixed.

The issue reports an umbrella regression where active runs can stall Control UI reconnects, interrupt Telegram typing, show heartbeat poll noise in transcripts, and mirror outbound Telegram media as filenames instead of previews.

## Best Possible Solution

Keep this issue open until maintainers split or finish the remaining work: retain the shipped refresh guard and heartbeat filtering, fix delivery mirroring so outbound media is stored as previewable structured/managed media, and continue the Docker/WSL2 gateway plus Telegram responsiveness profiling through #73428 or a narrower canonical follow-up.
