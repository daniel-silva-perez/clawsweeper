## Summary

Keep open. Current main still drops numeric-string costs and still accepts booleans and non-finite floats in the model-usage cost paths; the linked implementation PR #37877 is closed and unmerged in the provided GitHub context.

Add focused unit tests in `skills/model-usage/scripts/test_model_usage.py` for numeric-string aggregation/current/latest-day behavior and ignored bool/non-finite costs.
