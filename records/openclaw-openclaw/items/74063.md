## Summary

Keep open. Current main still does not provide an official all-platform desktop app like the issue requests. OpenClaw has a macOS companion app, mobile node apps, a browser Control UI/PWA, and a cross-platform headless node host, but the platform docs explicitly say Windows and Linux companion apps are planned/not available yet.

## Best Possible Solution

Keep this open for maintainer product planning. The best path is to decide whether OpenClaw wants native Windows/Linux companion apps, a packaged cross-platform desktop shell, or stronger PWA packaging, then split that decision into narrower platform-specific design and implementation issues that fit the Gateway, Control UI, node host, and macOS app boundaries.
