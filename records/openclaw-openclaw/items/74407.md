## What This Changes

This PR adds a `Skill credentials` note in `setupSkills`, precomputes credential-gated missing skills, and adds a unit test for mixed local Whisper plus API-backed Whisper/ElevenLabs skill onboarding.

## Best Possible Solution

Land one narrow onboarding UX fix for #74382 that keeps dependency installation separate from env-secret setup in the prompt flow, preserves regression coverage for the mixed Whisper/Whisper API/SAG case, and then close the linked bug after merge.

## Reproduction Assessment

Yes. The focused reproduction is a `setupSkills` run where `openai-whisper` is missing only a local binary while `openai-whisper-api` and `sag` are missing env credentials; current main has no transition note before those credential prompts.

## Solution Assessment

Yes. Framing the credential phase inside `setupSkills` using existing `primaryEnv` and `missing.env` metadata is the narrowest maintainable fix, and it does not change installer behavior, skill eligibility semantics, or secret storage.
