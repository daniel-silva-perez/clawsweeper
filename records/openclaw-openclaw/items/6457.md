Keep this PR open as useful source material, but it is not mergeable as-is and current main does not fully implement the intended Telegram forum-topic behavior. The PR branch updates the legacy Telegram native command handler and tests to register and clear bot commands in both default and `all_group_chats` scopes, and to preserve Telegram topic thread params for auth/diagnostic and plugin command-not-found replies.
