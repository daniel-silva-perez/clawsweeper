## Summary

Keep this PR open. Current main has partially absorbed the auth/threading side via `resolveTelegramThreadSpec`, but the central group-scope command registration/cleanup is still missing, and the plugin-command `Command not found.` fallback still sends without topic params. The PR is stale because it targets the removed `src/telegram` layout, so the useful outcome is a rebase or replacement in `extensions/telegram` rather than cleanup closure.

## Best Possible Solution
