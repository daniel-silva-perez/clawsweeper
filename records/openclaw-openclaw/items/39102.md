## Summary

Keep this PR open. Current main still implements only the global `tools.agentToAgent` gate for cross-agent session tools; the proposed per-requester `agents.list[].tools.agentToAgent.allow` config surface, policy evaluation, docs/help, and tests are not present. The PR remains a plausible implementation candidate, but it needs maintainer review and a rebase onto the current `src/plugin-sdk/session-visibility.ts` policy location.

## Best Possible Solution

Keep this PR open for maintainer review or author refresh. The best path is to rebase onto current main, implement the per-agent requester override in `src/plugin-sdk/session-visibility.ts`, add strict schema/type/generated-help/docs support for `agents.list[].tools.agentToAgent.allow`, preserve global participation plus outbound-only requester semantics, prune deleted agent IDs immutably, and validate with focused A2A/session tests plus the changed gate.
