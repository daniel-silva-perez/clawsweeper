## Summary

Keep this PR open. Current main still has the Windows registry command extraction bug in the moved browser plugin file, so the PR remains functionally useful but needs to be rebased and covered with regression tests.

## Best Possible Solution

Rebase the PR onto current main, apply the regex fix in extensions/browser/src/browser/chrome.executables.ts, and add targeted Vitest coverage for Windows registry command extraction through the resolver or a narrow test seam. The security review does not show supply-chain or permission concerns.
