## Summary

Keep open. Current main still does not make manual `/compact` discoverable or cancelable through the active-run abort path, and the PR is a relevant contributor implementation, but the proposed patch is not ready to merge because it misses harness-owned compaction and bundles a launchd/profile behavior change that would break existing custom label overrides.

## What This Changes

The PR exports embedded active-run helpers, wires queued embedded compaction into `ACTIVE_EMBEDDED_RUNS` with an abort controller and drain guard, and also changes CLI profile handling/tests for `OPENCLAW_LAUNCHD_LABEL`.

## Best Possible Solution

Land a focused compact cancellation fix that registers an abortable active-run handle at the lifecycle point where compaction actually starts, covers both queued PI/context-engine compaction and harness-owned compaction, forwards cancellation into the running operation, cascades same-session reply abort only through an approved seam, and adds targeted cancellation tests. Split the launchd/profile label behavior into a separate review for #65643.
