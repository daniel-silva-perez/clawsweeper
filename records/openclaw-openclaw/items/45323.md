## Summary

Keep open. Current main still has slash-command autocomplete and an existing `agents.list` data source, but it does not implement Control UI `@agent` autocomplete, mention chips in the composer, validation against known agents, or mention-chip rendering in chat history.

## Best Possible Solution

Keep this open for Control UI product/UI follow-up. The best path is likely to reuse `agents.list` for the directory, add a focused @mention dropdown in the chat composer, and have maintainers decide whether the first implementation should stay plain-text insertion or include a tested rich-text/contenteditable chip model with matching history rendering and accessibility coverage.
