## Summary

Keep this PR open. Current main still lacks the dedicated Telegram same-group, multi-bot, multi-agent setup example that the PR is trying to add; the existing docs cover Telegram account routing and Telegram group access controls separately. The PR remains useful, but its sample should be revised before merge to scope the intended shared Telegram group under each participating account instead of using open group policy without account-level groups.

## Best Possible Solution

Then merge as a focused docs improvement after the docs check passes; no changelog entry should be needed for docs-only guidance.
