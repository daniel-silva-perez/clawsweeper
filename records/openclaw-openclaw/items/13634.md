## Summary

Keep open. Current main already covers workspace-file injection through shared bootstrap Project Context, but the requested Slack account/global `systemPrompt` surface is still missing from typed config, strict schema, runtime prompt composition, and Slack docs.

The issue requests Slack workspace context parity by adding a global Slack `systemPrompt` and automatic workspace-file injection for Slack sessions.
