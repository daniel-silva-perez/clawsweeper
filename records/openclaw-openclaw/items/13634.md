## Summary

Keep open. Current main already handles the workspace-file half through shared Project Context/bootstrap injection, but the requested Slack account/global `systemPrompt` is still absent from Slack config types, strict schema, runtime prompt composition, and docs. ## Best Possible Solution

Add a documented Slack account/global prompt field that inherits through Slack account config, composes below shared Project Context in a stable order, and applies consistently to DMs, MPIM/group DMs, public/private channels, threads, and native slash commands.
