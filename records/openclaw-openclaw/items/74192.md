## Summary

Keep open. The report is new, concrete, and still maps to active OpenAI Codex provider/transport behavior on current main. Current main has leaner raw model-run paths than v2026.4.26, but it still routes openai-codex/gpt-5.5 through openai-codex-responses with 400k/272k context metadata and forwards the model maxTokens value as max_output_tokens. The investigation should capture request payload shape, max_output_tokens, first-byte latency, total stream duration, prompt/cache/session setup, and whether Pi 0.70.5 changes the behavior compared with v2026.4.26.
