## Summary

Keep this PR open. Current main still has the non-Apple command-palette shortcut problem, so the PR contains useful remaining behavior, but the current head is stale against main and would drop recently added Control UI shell routing/affordance behavior while leaving the current locale pipeline partially regenerated.

The PR localizes Control UI shell, theme, update-banner, status, and mobile chat settings labels through UI i18n maps, adds zh-CN strings/generated locale metadata, and introduces a platform-aware command-palette shortcut helper with tests.

## Best Possible Solution
