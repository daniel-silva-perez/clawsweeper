## Summary

Keep this PR open. Current main has already absorbed much of the Control UI shell localization under the newer `chat.*` and `common.*` keys, but it still hard-codes the command-palette shortcut as `⌘K`, so the PR's platform-aware `Ctrl+K` behavior for non-Apple platforms is not implemented. The branch is also far behind current main and overlaps the newer generated-locale/raw-copy pipeline, so it needs maintainer review and a rebase rather than cleanup closure.

## What This Changes

The PR localizes Control UI shell/topbar labels, update/status/theme affordances, and related locale metadata, and adds a user-agent-based command-palette shortcut helper with focused UI tests.

## Best Possible Solution

Rebase the PR onto current main, preserve the current Control UI i18n structure unless maintainers deliberately choose a namespace migration, add the platform-aware shortcut to both the title and visible kbd, regenerate all current locale outputs with the current pipeline, and rerun the focused Control UI/i18n checks.
