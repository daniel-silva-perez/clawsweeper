## Summary

Keep this PR open. The platform-aware shortcut work is still useful because current main still renders a fixed Apple shortcut, but the PR head is stale against newer Control UI changes and would regress current breadcrumb/base-path navigation, the sidebar trigger affordance, and the current generated-locale set.

## What This Changes

The branch moves Control UI shell/topbar/update/status/theme labels through i18n keys, adds zh-CN shell translations and generated locale metadata, and introduces a platform-aware command-palette shortcut helper with focused tests.

## Best Possible Solution

Rebase the branch onto current main, keep the platform-aware command-palette shortcut, preserve the current dashboard-header `basePath`/`@navigate` wiring and `sidebar-menu-trigger` class, regenerate every current locale bundle and metadata file through the i18n pipeline, translate the zh-CN dismiss title, and rerun the focused Control UI/i18n checks.
