Keep open. Current main still has the reported Telegram group activation gap: captionless photo messages are converted to media context, but the mention decision is computed from empty text/caption and the shared mention helper still blocks configured catch-all patterns from matching empty input before the group turn is skipped.

## What This Changes

The issue asks for captionless Telegram group photos to trigger image understanding consistently with DMs when group policy and configured mention patterns allow the bot to respond.

Update shared mention matching or Telegram inbound activation so configured catch-all mention patterns can authorize captionless media-only Telegram group turns, while preserving group allowlists, sender authorization, native bot mentions, reply-to-bot implicit mentions, control-command gating, and media attachment delivery.
