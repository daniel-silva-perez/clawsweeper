## Summary

Keep open. Current main still has the reported Telegram group path: captionless photos can become media-bearing inbound payloads, but the group mention calculation receives empty Telegram text and returns false before catch-all regex patterns can be evaluated, so the message is skipped before image understanding runs.

## What This Changes

The issue asks for Telegram captionless group photos to trigger image understanding consistently in direct messages and allowed group chats without requiring a caption workaround.

## Best Possible Solution

Update shared mention matching or Telegram activation so captionless image messages with catch-all group mention patterns pass the mention gate while preserving sender authorization, explicit native mention behavior, specific regex behavior, audio preflight semantics, and media attachment delivery.
