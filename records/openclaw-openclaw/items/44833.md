## Summary

Keep open. Current main still has the reported Telegram group activation gap: captionless photos are represented as media, but group mention matching is computed from empty Telegram text/caption, and the shared mention helper still prevents configured catch-all patterns from evaluating against empty input before the group turn is skipped.

## What This Changes

The issue requests that captionless Telegram group photo messages trigger image understanding like DMs when the group’s normal authorization and mention policy allow it.

## Best Possible Solution

Update shared mention matching or Telegram inbound activation so intentionally configured catch-all mention patterns can authorize captionless media-only Telegram group turns, while preserving specific pattern behavior and the existing group safety gates.
