## Summary

Keep this PR open. Current main still has one bug-report `logs` textarea labeled for logs, screenshots, and evidence while also using `render: shell`; no separate screenshots/evidence textarea or upload field exists. The PR is a focused implementation candidate, and its only changed file is the issue-template YAML.

## Best Possible Solution
