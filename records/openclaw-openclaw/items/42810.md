## Summary

Keep this PR open. Current main still does not implement an ACP control-plane audit logging system, and this open PR remains an implementation candidate rather than obsolete work. The PR also still needs maintainer review and security/reliability hardening before it would be safe to merge.

## Best Possible Solution

Keep this PR open for maintainer follow-up. The best path is a rebased, narrower ACP-owned audit implementation with a real best-effort durable sink under the normal OpenClaw state/log directory, restrictive file permissions, bounded failure behavior, correct query/retention semantics, lifecycle coverage for init/close/cancel/run/runtime-control events, focused tests, and unrelated config/schema changes split out.
