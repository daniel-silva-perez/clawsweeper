## Summary

Keep this PR open. Current main still does not implement the PR’s intended additive `agent.wait` terminal metadata: the wait snapshot, cached wait response, lifecycle/dedupe race response, and internal wait wrapper expose status/timing/error plus stop/liveness/yield fields, but no optional `meta` with session id, usage, last-call usage, provider/model, or cost. The PR is a plausible implementation candidate for the closed feature request #49494, but it needs rebase and maintainer review because current main has since added wait fields that the PR does not preserve, and the proposed cost calculation/direct `agent` response exposure need deliberate tests/docs/security review.

## Best Possible Solution

Keep this PR open as an implementation candidate. The best path is to refresh it against current main, preserve existing `stopReason`/`livenessState`/`yielded` wait fields, add an optional `AgentWaitUsageMeta` response shape from `EmbeddedPiAgentMeta`, keep cost estimation non-fatal, return metadata consistently through cached and lifecycle-race `agent.wait` paths, decide whether direct `agent` responses and `waitForRun` wrappers should expose the same metadata, and add focused regression tests plus protocol docs.

## Risks / Open Questions

- The PR branch is stale against current main: current `agent.wait` now forwards `stopReason`, `livenessState`, and `yielded`, and the PR diff does not preserve those fields in its response objects.
- The proposed `buildUsageMetaFromRunResult` performs config/cost resolution in the success path; optional metadata should be best-effort so a config/pricing failure cannot convert a completed agent run into an error response.
- The PR adds `meta` to the direct `agent` terminal response as well as `agent.wait`; that may be fine, but it needs to be intentional and documented/test-covered.
- Issue #68066 remains open for OpenRouter billed-cost correctness, so any `costUsd` field should be clearly treated as an estimate unless authoritative provider billing is also resolved.
