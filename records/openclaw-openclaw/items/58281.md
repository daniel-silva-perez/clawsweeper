## Summary

Keep this PR open. Current main still has the tilde-only shared path resolver, so the PR's intended fix for `${VAR}`/`$VAR` references in user path resolution is not implemented on main. The PR is a focused implementation candidate for the linked bug #53628, but it still needs maintainer review because an existing review comment flags a behavioral risk around missing `$HOME` collapsing to a root-relative path.

## Best Possible Solution

Keep this PR open as a live implementation candidate for the linked path-resolution bug. Maintainers should review the `$HOME` fallback concern, adjust the helper semantics if needed, and land focused `home-dir` regression tests for `${VAR}`, `$VAR`, and missing-variable behavior.
