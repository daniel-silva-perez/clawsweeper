## Summary

Keep this PR open. Current main still serves Control UI index.html without rewriting favicon/apple-touch-icon hrefs for a configured basePath, so the user-visible bug remains, but the PR was authored before the current CSP-aware index serving path and needs a careful rebase before merge.

## What This Changes

The PR adds a Control UI index HTML helper to prepend the configured basePath to favicon/apple-touch-icon hrefs and adds gateway tests for the rewrite and no-op cases.
