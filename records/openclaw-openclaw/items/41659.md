## Summary

This PR adds a gateway HTML-serving helper that rewrites favicon and apple-touch-icon hrefs under `gateway.controlUi.basePath`, threads the basePath through direct index and SPA fallback serving, and adds gateway misc tests for rewrite and no-op behavior.

## Best Possible Solution
