## Summary

Keep open. Current main only covers part of this PR’s intended sub-agent docs cleanup, and the remaining docs drift is still real. The PR is still a plausible implementation candidate for #47558, but it needs a refresh because current runtime defaults and tool-policy behavior have moved since the PR body/patch.

## Best Possible Solution

Keep this PR open, but require a rebase/update against current main or designate #56157 as the canonical refreshed patch. The correct docs update should align docs/tools/subagents.md and docs/gateway/config-tools.md with current source: document announceTimeoutMs with the current 120000 ms default, keep streamTo ACP-only, separate global and Discord thread-binding keys, make depth/announce/auto-archive/cascade guidance depth-N through maxSpawnDepth 5, and match the deny-list wording to src/agents/pi-tools.policy.ts. After that lands, close #47558 and any superseded duplicate PR.
