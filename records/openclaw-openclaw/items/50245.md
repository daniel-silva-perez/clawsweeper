## Summary

Keep open. Current main still carries `durationMs` in the health snapshot/JSON contract, but the human-readable `openclaw health` renderer does not print a gateway probe-duration line. The issue is also paired with open PR #50250 using closing syntax, so the maintainer path is to review or land that PR, or an equivalent narrow patch, rather than close this issue early.

## What This Changes

## Best Possible Solution

Land PR #50250 or an equivalent narrow change that logs `summary.durationMs` in the non-JSON `openclaw health` output, preserves the JSON response unchanged, and keeps focused regression coverage for the new line.
