## Summary

Current main still skips grouped skill layouts like `skills/<group>/<skill>/SKILL.md`. This PR remains a focused implementation candidate for the open regression #56915, with recent maintainer activity and a narrow file set that matches the stated fix.

## Best Possible Solution

Keep this PR open for maintainer review and land it, or an equivalent patch, as the canonical fix for #56915. The implementation should add bounded one-level grouped discovery in `src/agents/skills/workspace.ts`, keep direct skill behavior unchanged, preserve containment and scan limits, and ship the regression tests plus docs/changelog update.
