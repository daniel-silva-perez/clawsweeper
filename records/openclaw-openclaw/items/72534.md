## Summary

Current main still skips grouped skill layouts such as `skills/<group>/<skill>/SKILL.md`, so this open PR is not obsolete. The PR remains an active, focused implementation candidate for the linked open regression, with tests/docs/changelog in its diff and no unrelated workflow, dependency, lockfile, install, release, or package metadata changes in the reviewed file list.

## Best Possible Solution

Keep this PR open for maintainer review and land it, or an equivalent patch, as the canonical fix for grouped skill discovery. The implementation should add bounded one-level discovery in `src/agents/skills/workspace.ts`, keep direct skill behavior unchanged, preserve containment and scan limits, and ship the included regression tests plus docs/changelog update. Related watcher work such as #51733 is adjacent, but it does not replace this loader fix.
