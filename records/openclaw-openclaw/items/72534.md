## Summary

Keep this PR open. Current main still performs shallow discovery for configured skill roots: it checks only direct child directories for `SKILL.md` and skips grouping folders like `skills/coze/koze-retrieval/SKILL.md`. The PR is a focused implementation candidate for the still-open #56915 bug and has recent maintainer repair/merge-ready context rather than being obsolete.

## Best Possible Solution

Review and land this PR, or an equivalent patch, as the canonical fix for #56915. The implementation should add bounded one-level grouped discovery in `src/agents/skills/workspace.ts`, preserve direct skill behavior, keep realpath/symlink and size/candidate limits intact, and retain regression coverage plus docs/changelog updates.
