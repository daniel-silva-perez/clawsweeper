## Summary

Keep this PR open. Current main still does not implement the PR's intended Discord system-message audit: the Discord mapper lacks ChannelNameChange, ChannelIconChange, ChannelFollowAdd, call, discovery, invite reminder, subscription, premium upsell, stage raise-hand, and app-subscription cases, and preflight only drops/queues messages when that mapper returns text. The live PR is open and unmerged, and the same-author narrower #49308 is also open rather than a merged superseder.

## Best Possible Solution

Keep this PR open for maintainer review as the broader Discord native system-message audit, or explicitly choose #49308 as the narrower canonical fix. Before landing, rebase onto current main, update the preflight mock to the current system-event-runtime import, add the required changelog attribution, and preserve regression coverage for both mapper coverage and preflight drop/enqueue behavior.
