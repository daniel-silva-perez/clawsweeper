## Summary

Keep this PR open for maintainer/author changes. The linked regression appears real and the retry-disable half overlaps a narrower open PR, but this branch is not mergeable as-is because the new thinking-repetition guard misses the Qwen transport named in the report and can inject a synthetic user turn from unrelated repeated thinking blocks.

## What This Changes

The PR disables pi-coding-agent SDK retry in the embedded settings manager and adds inline repeated-thinking hint injection to the transport message transform.

## Best Possible Solution

Land a targeted, tested fix for the replay loop: verify and disable the inner pi-coding-agent retry if appropriate, then put any reasoning-loop mitigation behind a correct transport/runtime seam that covers Qwen's `openai-completions` path and requires consecutive assistant tool-call turns before steering the model.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Run the detector for Qwen transports:** `src/agents/transport-message-transform.ts:125`
  - body: Qwen uses the `openai-completions` API, which is not in `SYNTHETIC_TOOL_RESULT_APIS`, so `transformTransportMessages` returns at the existing `!allowSyntheticToolResults` branch before this new block. The claimed Qwen thinking-loop fix therefore never executes for the linked regression; move the detection before that return or explicitly cover the intended API family.
  - confidence: 0.92
- **[P2] Require repeated tool-call turns before adding the hint:** `src/agents/transport-message-transform.ts:134-148`
  - body: The new scan collects all historical thinking blocks and compares the last three strings; it does not require consecutive assistant turns or any tool calls. Three repeated reasoning blocks in ordinary history would append a synthetic user message and steer the next model call even when no tool loop exists, so the detector should track consecutive assistant turns that include tool calls before injecting a hint.
  - confidence: 0.88
