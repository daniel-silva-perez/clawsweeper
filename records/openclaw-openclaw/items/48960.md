## Summary

Keep this PR open. Current main still lacks the detailed skill-loading diagnostics this PR targets: missing skill roots return empty, malformed or incomplete SKILL.md files are dropped without a parse/missing-field warning, and the requested discovery-completion/loading-summary metrics are absent. The PR is still a plausible implementation candidate, but it should be refreshed before merge because the discussion flags inaccurate metrics, brittle logger assertions, path-detail exposure, and possible repeated-warning log noise.

## Best Possible Solution

Keep the PR open and ask the contributor to refresh it against current main. The best path is to add targeted diagnostics in `src/agents/skills/local-loader.ts` and `src/agents/skills/workspace.ts`, sanitize or compact path/error details, aggregate or cap repeated parse warnings, report accurate per-source and deduplication metrics, and add focused tests that assert the actual logger string output.
