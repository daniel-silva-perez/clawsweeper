## Summary

Keep this open. Current main still renders the Control UI cron surface as summary data, filters, a flat jobs list, run history, and an edit form, while the requested timeline/day/week/calendar/heatmap/drag-to-reschedule experience is not present. The issue requests a Control UI calendar/timeline visualization for cron jobs with day/week modes, quick stats, job detail popups, drag-to-reschedule, and cron validation.

## Best Possible Solution
