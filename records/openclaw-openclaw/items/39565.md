## Summary

Keep this PR open. Current main still has the pre-PR TUI sanitizer behavior: dotted IDs without underscores and punctuation-wrapped dotted entity IDs can still be chunked with inserted spaces, and the linked bug #39505 remains open in the provided GitHub context.

## Best Possible Solution

Keep this PR open, or land a rebased/cherry-picked equivalent. The best path is the narrow `src/tui/tui-formatters.ts` fix: classify the punctuation-stripped candidate, include markdown `*` in edge stripping, preserve dotted identifiers and digit-bearing hyphenated IDs, keep punctuation-only and prose-like long tokens chunkable, and retain the focused regression tests in `src/tui/tui-formatters.test.ts`.
