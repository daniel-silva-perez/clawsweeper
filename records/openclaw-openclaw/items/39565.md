## Summary

Keep this PR open. Current main still has the pre-PR TUI sanitizer behavior: it classifies URL/path/file-like checks against the raw token, preserves only underscore file-like tokens, does not strip markdown asterisks at token edges, and still chunks missed long tokens by inserting spaces. The linked bug #39505 remains open, and this PR is a focused implementation candidate with follow-up commits addressing the review-raised guardrails.

## Best Possible Solution

Keep this PR open for maintainer review or land a rebased/cherry-picked equivalent. The right implementation path is the narrow `src/tui/tui-formatters.ts` fix: classify the punctuation-stripped candidate, include markdown `*` in edge stripping, preserve dotted identifiers and digit-bearing hyphenated IDs, keep punctuation-only and prose-like long tokens chunkable, and retain focused regression tests in `src/tui/tui-formatters.test.ts`.
