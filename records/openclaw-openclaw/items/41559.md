- **[P2] Use the injected gateway caller for new send and history calls:** `src/agents/tools/sessions-send-tool.ts:297`
  - body: The ingress echo path and nested relay guard call the module-level `callGateway` instead of the `gatewayCall` selected from `opts.callGateway`. In tests, embedded use, or alternate gateway contexts, enabling these new paths can send or read history through the default gateway instead of the caller's scoped gateway context.
  - confidence: 0.86

  - body: `relay.requireDelivery` is parsed and documented as strict, but `relayTurn` catches every send failure and only logs it. With strict relay enabled, mirror delivery failures still let the A2A flow continue, so operators cannot rely on the setting to block or abort a failed relay.
  - confidence: 0.89

  - body: The PR adds public `session.agentToAgent.*` fields to the zod schema, types, help, and labels, but the generated public schema at PR head still only contains `maxPingPongTurns`.   - confidence: 0.82

## Security Review

Status: needs_attention

Summary: The diff has no new dependency or workflow supply-chain changes, but the new channel relay paths need security fixes before merge.

Concerns:

- **[medium] New delivery path bypasses injected gateway context:** `src/agents/tools/sessions-send-tool.ts:297`
  - body: The ingress echo send and nested guard history lookup use the module-level gateway caller instead of the injected caller passed into `createSessionsSendTool`. In contexts that intentionally provide a scoped or test gateway caller, the new external channel send can run through default gateway resolution instead of the caller's intended context.
  - confidence: 0.82
- **[low] Strict relay policy is parsed but not enforced:** `src/agents/tools/sessions-send-tool.a2a.ts:88`
  - body: Relay delivery failures are swallowed even when `relay.requireDelivery` is true. Because this feature mirrors inter-agent content to external channels, strict delivery policy should fail closed when configured rather than continuing as best effort.
  - confidence: 0.84
