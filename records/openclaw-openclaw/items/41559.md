## Security Review

Status: needs_attention

Summary: The diff adds no dependency or CI execution path, but it introduces external A2A channel sends with scoped-gateway and strict-delivery concerns.

Concerns:

- **[medium] New delivery path bypasses injected gateway context:** `src/agents/tools/sessions-send-tool.ts:297`
  - body: The ingress echo send and nested guard history lookup call the module-level gateway helper instead of the injected caller selected by `createSessionsSendTool`, which can bypass scoped gateway resolution while sending to external channels or reading history.
  - confidence: 0.84

  - body: Relay send failures are swallowed even when `relay.requireDelivery` is true. Because this feature mirrors inter-agent content to external channels, strict delivery mode should fail closed when configured.
  - confidence: 0.85
