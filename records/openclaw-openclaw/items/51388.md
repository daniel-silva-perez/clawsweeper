## Summary

Keep this PR open. Current main still does not implement the PR’s intended fix: subagent `alsoAllow` is accepted in config, but it is not carried through `SandboxToolPolicy` or collected into the plugin tool allowlist, so optional plugin tools allowed only via `tools.subagents.tools.alsoAllow` can still be omitted from subagent sessions.

## Best Possible Solution

Keep this PR open for maintainer review, or supersede it with an equivalent maintainer patch. The best fix is to preserve `alsoAllow` through subagent policy construction, include it in `collectExplicitAllowlist()` for both native and gateway tool resolution, preserve it during plugin-group expansion, and add focused tests for an optional plugin tool allowed only by `tools.subagents.tools.alsoAllow`.
