## Summary

Keep this PR open. Current main still hardcodes the exec approvals JSON and socket defaults under `~/.openclaw`, while `OPENCLAW_STATE_DIR` is still the documented mutable state root used by hosted deployments. The PR remains an open implementation candidate after maintainer repair, not cleanup-close material.

## Best Possible Solution

Keep this PR open for maintainer review and landing, or explicitly supersede it with #62938 if maintainers choose that branch. The shipped fix should make exec approvals JSON/socket defaults honor an explicit `OPENCLAW_STATE_DIR`, preserve the legacy `~/.openclaw` fallback when unset, update effective-policy reporting/docs, and include focused regression coverage.
