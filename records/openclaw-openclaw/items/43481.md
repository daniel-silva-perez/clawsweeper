The issue requests provider-level fallback trust controls so users can prevent images and attachments from crossing provider boundaries during model fallback unless explicitly allowed.

Review and either land PR #57959 with any needed revisions, or replace it with a centralized fallback-boundary implementation that passes primary and target provider context, covers explicit and prompt-detected media consistently, adds schema/docs for the trust control, and includes same-provider plus cross-provider tests.
