## What This Changes

The issue requests investigation or a fix for Linux Gateway startup showing about 22GB VmSize/VSZ and high anonymous mapping counts while RSS remains normal on stable OpenClaw releases.

## Best Possible Solution

Keep the issue open and collect a focused current-main or latest-release reproduction with ps RSS/VSZ over time, /proc/<pid>/status, map counts, pmap or smaps summaries, minimal-config results, enabled plugins/providers, Node version, and launch method.
