## Summary

Current main implements the sandbox-image fix: the default Docker image path no longer silently tags plain Debian as `openclaw-sandbox:bookworm-slim`, the bundled sandbox Dockerfile includes `python3`, regression coverage exists, and docs now describe the required build path. The open older PR #57789 is superseded by merged PR #73362.

## Best Possible Solution

Close this issue as implemented on current main, keep the merged #73362 behavior and regression coverage, and separately close or supersede the older open PR #57789.
