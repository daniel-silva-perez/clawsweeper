## Summary

Keep this PR open. Current main still has the ASCII-only Discord outbound mention parser, so the Unicode/CJK handle rewrite is not implemented or obsolete. The Discord code in the PR looks focused and the author addressed the earlier email/URL/wrapped-URL review feedback, but the branch still carries unrelated Yuanbao install metadata changes and is far behind current main, so it needs maintainer rebase/scoping before merge.

## What This Changes

The PR expands Discord outbound mention rewriting to Unicode/CJK handles with email, URL-token, reserved mention, already-formatted mention, and code-span regressions, while the branch also includes Yuanbao catalog install metadata updates.

## Best Possible Solution

Rebase and scope the branch so it only changes the Discord mention parser and focused tests, then either land the narrow Unicode/CJK rewrite or explicitly consolidate it into the broader #59818 work if maintainers want the cross-account bot-to-bot behavior at the same time.
