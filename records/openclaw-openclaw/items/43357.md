## Summary

Keep open. Current main still has the ASCII-only Discord mention parser, so the PR addresses a real gap, but the current patch is not merge-ready because the CJK URL guard still misses Markdown link destinations and can rewrite inside link URLs. Maintainers should also coordinate this narrow parser fix with the broader open Discord CJK/multi-account work in #59818.

## What This Changes

The PR expands Discord outbound mention rewriting to Unicode/CJK handles, adds mention-safety regression tests, and carries Yuanbao catalog pin/contract updates.

## Best Possible Solution

Land a Discord-only parser and regression-test update after Markdown link URL destinations are guarded, then explicitly decide whether #59818 should supply the broader cross-account cache behavior. The Yuanbao catalog pin should stay as current-main work rather than remain bundled into this Discord PR.

## Reproduction Assessment

Yes. Current main can be checked by seeding the Discord directory cache with `张三` and calling `rewriteDiscordKnownMentions('你好@张三，收到请回复。', { accountId: 'default' })`; the source regex and a focused probe show no match. A separate PR-head regex probe reproduces the remaining Markdown-link URL corruption risk.

## Solution Assessment

No, not as-is. The narrow parser direction is maintainable, but the PR should skip CJK matches inside Markdown link URL destinations and be rebased/scoped against current main before it is a safe merge candidate.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Skip CJK matches inside Markdown link URLs:** `extensions/discord/src/mentions.ts:104-108`
  - body: The URL-context check only strips leading wrapper characters from the whitespace token. For `[资料](https://x.com/你好@张三)`, the token becomes `资料](https://...)`, so it fails `URL_SCHEME_TOKEN_PATTERN`; the CJK pass then rewrites the link destination and corrupts the Markdown URL. Detect URL schemes inside Markdown link destinations or skip those tokens before applying the CJK pass.
  - confidence: 0.9
