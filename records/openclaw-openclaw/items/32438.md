## Summary

Keep open. Current main still persists Gateway file logs as a single JSONL stream and only formats those lines when the CLI or Control UI reads them; no derived plain-text companion file is written beside `logging.file`. The earlier same-author implementation PR is closed unmerged, the earlier duplicate is closed, and the merged date-boundary log discovery PR is adjacent but does not satisfy the requested sidecar file.
