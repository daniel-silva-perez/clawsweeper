## Summary

This PR is still a relevant implementation candidate. Current main does not implement the central Slack A2A parser fix, and the restart-sentinel routing path still lacks the PR’s parsed-target-before-base fallback. The paired same-author issue #49747 is also still open, so the PR should not be closed as obsolete.

## Best Possible Solution

Keep this PR open as an implementation candidate. Revise the parser to reuse or mirror parseRawSessionConversationRef(), add regression coverage for agent-prefixed Slack thread keys under webchat/provider-fallback contexts, and align restart-sentinel fallback so parsed thread targets win before base-session route fill-ins. Then review it together with the paired open issue #49747.
