Keep this PR open. Keep this PR open as an implementation candidate. Revise the parser to reuse or mirror parseRawSessionConversationRef(), add regression coverage for agent-prefixed Slack thread keys under webchat/provider-fallback contexts, and update restart-sentinel fallback so parsed thread targets win before base-session route fill-ins while preserving useful partial hints. Review it together with #49747.
