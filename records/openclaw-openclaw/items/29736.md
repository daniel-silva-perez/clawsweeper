## Summary

Keep open. Current main still routes exec approvals storage and effective-policy default host reporting through home-backed defaults, while the focused repair PR #74002 remains open and unmerged.

## What This Changes

Exec approvals should honor the active OpenClaw state root/profile instead of reading and writing the approvals file and socket under the default home-backed path.

## Best Possible Solution

Land #74002 or a maintainer-approved equivalent that derives the approvals JSON path, socket path, and effective-policy default host path from the active state-dir/profile contract, preserves the legacy fallback when no state override is active, and updates focused runtime/CLI tests plus the public approvals docs.
