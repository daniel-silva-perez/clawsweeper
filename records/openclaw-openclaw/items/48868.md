## Summary

Current main at e476523 still does not implement this PR’s intended TypeScript Gateway behavior. The logger and temporary-directory resolver do not honor `OPENCLAW_LOG_DIR` or `OPENCLAW_TMP_DIR`, and the PR remains a plausible focused infra fix rather than cleanup-closure material.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to either land a corrected version with focused tests for both env vars, preserved temp-root trust checks, aligned log path helpers, and docs updates, or request those changes from the contributor before merge.
