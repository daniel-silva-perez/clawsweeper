Keep open. Current main still supports provider-based imports from other agent systems and backup create/verify, but it does not provide same-product cross-platform `migrate export`/archive import or `backup restore`; the issue is also linked to an open fork implementation PR that should be reviewed, replaced, landed, or closed before this tracker is resolved.
