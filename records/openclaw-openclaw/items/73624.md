## Summary

Keep open. Current main still has the reported prompt-composition bug: native subagent spawn places the delegated task in `extraSystemPrompt`, but the embedded, compaction, and CLI prompt paths still short-circuit to `systemPromptOverride` before the builders that append that extra prompt. The linked fixing PR #73637 is open/unmerged and uses closing syntax, so this issue should remain open until that PR or an equivalent fix lands.

## Best Possible Solution

Review and land PR #73637, or an equivalent source change, so `systemPromptOverride` is preserved while trimmed runtime `extraSystemPrompt` is appended for subagent/task contexts in embedded run, compaction, and CLI prompt preparation. Add regression coverage proving a subagent spawned under an agent with `systemPromptOverride` receives both the override and the generated `## Your Role` task block.
