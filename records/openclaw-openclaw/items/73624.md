## Summary

Keep open. Current main still has the reported prompt-composition failure: native subagent spawn passes the delegated `Your Role` task through `extraSystemPrompt`, but the embedded, compaction, and CLI prompt paths still short-circuit to `systemPromptOverride` before reaching the builders that append that extra prompt. The linked PR #73637 remains the implementation candidate, so this issue should stay open until that PR or an equivalent fix lands.

## Best Possible Solution

Review and land PR #73637, or an equivalent source change, so `systemPromptOverride` is preserved while runtime `extraSystemPrompt` is appended in embedded attempt, compaction, and CLI prompt assembly. Add regression coverage proving a subagent spawned under an agent with `systemPromptOverride` receives both the override text and the generated `Your Role` task block.
