## Summary

Keep this PR open. Current main still lacks the legacy `flow_runs` table rebuild, so the reported `owner_session_key` NOT NULL failure remains plausible for upgraded SQLite databases. The PR is still a focused implementation candidate with a regression test; the related open bug #65563 and parallel open PR #65767 confirm the same remaining work but do not prove this PR is obsolete or safely closable without maintainer choice of a canonical fix.

## Best Possible Solution

Keep this PR open. Review it alongside #65563 and #65767, then land this PR, #65767, or an equivalent maintainer patch that rebuilds legacy `flow_runs` tables, preserves existing owner data in `owner_key`, removes the blocking `owner_session_key` constraint, and adds focused regression coverage.
