## Summary

Current main implements the runtime-deps mirror fix via merged PR #74213: the core mirror allowlist now includes the reported root-package imports, the runtime-deps collection path folds those root package deps into plugin install/retention specs, and a regression guard now fails when a value-imported root dependency is neither mirrored nor extension-owned. The fix is on main only; no release tag currently contains the proving commit.

## What This Changes

The issue requests that the packaged runtime-deps mirror include all root-package bare imports used by built dist code, including nine missing packages from the 2026.4.26 release.

## Best Possible Solution

Close this issue as implemented on main by PR #74213, keep the new drift guard in place, and let the next release carry the runtime-deps mirror fix to npm users.
