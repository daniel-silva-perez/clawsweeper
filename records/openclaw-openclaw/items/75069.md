## Summary

Keep open. ## What This Changes

The issue requests that bundled plugin runtime dependency staging and mirror preparation stop running repeatedly and synchronously on pi-agent and Gateway hot paths.

## Best Possible Solution

Keep this issue open until PR #75183 or an equivalent staged fix lands, then close only after packaged/global-install profiling shows bounded Gateway readiness and first-agent event-loop lag for the bundled runtime-deps/mirror paths.

## Reproduction Assessment
