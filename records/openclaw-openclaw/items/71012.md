## What This Changes

The PR splits `extensions/memory-core/src/dreaming-phases.ts` into a slimmer runner plus new `dreaming-session-corpus.ts` and `dreaming-shared-types.ts` modules for session ingestion and shared helpers.

## Best Possible Solution

A salvageable version should rebase onto current main, preserve the public compatibility shim and test hooks, keep current Dream Diary model override behavior, and then prove the split with focused memory-core typecheck/test coverage for daily ingestion, session ingestion, light/REM phases, and narrative dispatch parity.

## Solution Assessment
