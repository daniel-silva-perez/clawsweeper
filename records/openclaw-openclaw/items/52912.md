## Summary

Keep this PR open, but do not merge it as-is. Current main still has the relevant single-agent auth-profile writes, while the PR broadens OAuth credential propagation against the documented portability policy and only fixes the plugin auth-choice path, leaving the models auth and non-interactive Anthropic token paths from the linked regression uncovered.

## What This Changes

The PR routes plugin auth-choice profile persistence through a shared helper, syncs Anthropic token and OAuth credentials to sibling agent directories, adds Anthropic token sibling-sync coverage, and adds DeepSeek to the provider contract registry.

## Best Possible Solution

Land a narrower fix that centralizes Anthropic token profile persistence for the intended setup-token and manual-token contexts, covers plugin auth-choice plus models auth setup-token/paste-token and non-interactive setup-token where applicable, preserves OAuth copying as explicit opt-in, and separates unrelated registry cleanup into its own change.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.91

Full review comments:

- **[P1] Restrict OAuth auth-profile sync to intended contexts:** `src/plugins/provider-auth-helpers.ts:222-223`
  - body: `shouldSyncSiblingAgentsForCredential` returns true for every OAuth credential, so plugin OAuth logins would copy refresh material into all sibling agent stores. That conflicts with the documented non-portable OAuth default and can overwrite credentials for agents that should remain isolated; gate sibling sync to the Anthropic token case or an explicit provider/context opt-in.
  - confidence: 0.92
- **[P2] Cover the models auth token paths:** `src/plugins/provider-auth-choice.ts:133-135`
  - body: The linked #51911 repro uses `models auth setup-token` and `paste-token`, but this patch only changes the plugin auth-choice loop. Current models auth and Anthropic non-interactive token writers still persist to one agent store, so the user-facing multi-agent failure can remain after this patch.
  - confidence: 0.86
