## Summary

Keep open. Current main still lacks the OpenRouter catalog/default-thinking hooks from this PR, so the reported dynamic-model reasoning gap is not already implemented, but the proposed patch has blocking correctness issues around reasoning visibility and fresh-cache catalog loading.

## What This Changes

This PR adds OpenRouter catalog augmentation and default-thinking fallback from cached OpenRouter model capabilities, exports an all-cached OpenRouter capability accessor through the provider-stream SDK surface, and reformats one daemon lifecycle test mock.

## Best Possible Solution

Revise or rebase the PR so dynamic OpenRouter capability metadata is available before thinking and reasoning defaults are resolved, native OpenRouter reasoning blocks remain visible by default when intended, and any SDK/cache surface is narrow, documented, and covered by cached plus fresh-cache dynamic-model tests.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P1] Preserve reasoning visibility when thinking is inferred:** `extensions/openrouter/index.ts:175-178`
  - body: The new OpenRouter default-thinking path returns `"low"` for reasoning-capable dynamic models. Current reply defaults only call `resolveDefaultReasoningLevel()` while `thinkingActive` is false, and the subscriber only extracts native thinking when `reasoningMode` is `on` or `stream`, so this can still hide the reasoning blocks the PR is meant to surface.
  - confidence: 0.83
- **[P2] Await the OpenRouter catalog before reading it:** `src/agents/pi-embedded-runner/openrouter-model-capabilities.ts:310-311`
  - body: `getAllCachedOpenRouterModels()` calls the nonblocking cache ensure path, which can return an empty map on a fresh machine while the OpenRouter API fetch continues in the background. Since default resolution happens before the later `prepareDynamicModel()` warm-up, first use of an uncached dynamic reasoning model can still resolve to the old off defaults.
  - confidence: 0.78
