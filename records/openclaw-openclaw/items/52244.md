## Summary

Keep open. Current main does not contain the PR hooks/export, and the proposed change targets a real dynamic OpenRouter reasoning-default gap, but the patch still has blocking behavior issues: inferred thinking prevents default reasoning visibility, and the cache accessor can return empty on first use before the async OpenRouter catalog warm-up runs.

## What This Changes

This PR adds OpenRouter catalog augmentation and default-thinking fallback from cached OpenRouter model capabilities, exports an all-cached OpenRouter capability accessor through the provider-stream SDK surface, and reformats one daemon lifecycle test mock.

## Best Possible Solution

Revise or rebase the PR so dynamic OpenRouter capability metadata is available before thinking and reasoning defaults are resolved, automatic reasoning visibility is preserved for native OpenRouter reasoning blocks, and any SDK/cache surface is narrow, documented, and covered by both cached and fresh-cache dynamic-model tests.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.82

Full review comments:

- **[P1] Keep reasoning visible when thinking is inferred:** `extensions/openrouter/index.ts:175-178`
  - body: Returning `"low"` for dynamic OpenRouter reasoning models makes `thinkingActive` true before model reasoning defaults run. Since the embedded subscriber only extracts native thinking when `reasoningMode` is `on` or `stream`, the reasoning blocks this PR is meant to surface can still be dropped by default.
  - confidence: 0.83
- **[P2] Await fresh OpenRouter catalog before augmentation:** `src/agents/pi-embedded-runner/openrouter-model-capabilities.ts:310-311`
  - body: `getAllCachedOpenRouterModels()` only calls the nonblocking cache ensure path, so a fresh machine with no disk cache can return an empty map while the API fetch runs in the background. Default resolution happens before the later dynamic-model warm-up, leaving first use of an uncached dynamic reasoning model at the old defaults.
  - confidence: 0.78
