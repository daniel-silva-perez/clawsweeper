## Summary

Keep this issue open. Current main still does not ship Matrix or Cyberpunk as first-class Control UI theme families, and the only newer theming work is one browser-local tweakcn import slot. The linked Matrix PR was closed unmerged, and the Cyberpunk work is only described as fork work in the issue comments.

## Best Possible Solution

Leave this issue open for maintainer product/design judgment. If accepted as core UI work, implement Matrix and Cyberpunk across the theme model, early HTML bootstrapping, base CSS, full and quick Appearance selectors, docs, and reduced-motion/accessibility coverage. If maintainers prefer external distribution, first add and document a supported Control UI theme-pack/plugin API; the current tweakcn slot and generic plugin UI descriptor are too narrow for the requested animated theme families.
