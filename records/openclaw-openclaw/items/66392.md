## Summary

Keep this issue open. Current main still does not ship Matrix or Cyberpunk Control UI theme families. It has Claw, Knot, Dash, and one browser-local tweakcn custom token slot; the newer plugin Control UI descriptor seam is metadata for session/tool/run/settings surfaces, not a theme-pack registration API. The linked Matrix PR is closed unmerged, and the Cyberpunk work is only described as fork work in the issue comments.

## Best Possible Solution

Leave this issue open for maintainer product/design judgment. If accepted as core UI work, implement Matrix and Cyberpunk as first-class Control UI theme families across the theme model, early HTML bootstrapping, base CSS, full and quick Appearance selectors, docs, and accessibility/reduced-motion coverage. If maintainers prefer external distribution, first add and document a supported Control UI theme-pack/plugin API; the current tweakcn token import and generic UI descriptor seam are too narrow for the requested animated theme families.
