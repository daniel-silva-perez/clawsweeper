## Summary

Keep this PR open. Current main has mostly absorbed the local one-shot stateless behavior through a different lean completion path, but it has not absorbed the profile-scoped transcript fallback fix, so the PR is not obsolete and still needs maintainer review/rebase.

## What This Changes

This draft PR changes local `infer model run --local` to use an explicit ephemeral session by default, changes fresh transcript fallback paths to use the profile-local sessions directory derived from the store path, adds focused tests, and records canary baseline docs.

## Best Possible Solution

Keep the current lean local model-run implementation, then land or cherry-pick a narrow transcript fallback change that resolves fresh fallback files inside the sessions directory derived from `storePath`, with focused tests for profile-local stores and thread/topic paths. Treat the canary baseline docs as optional maintainer-reviewed context rather than required runtime behavior.
