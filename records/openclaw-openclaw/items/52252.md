## Summary

Keep this PR open. Current main still does not rewrite Volta tools/image Node paths to a stable Volta shim, and the PR remains a plausible implementation candidate for the linked LaunchAgent fragility. The unresolved review concern about custom VOLTA_HOME and Windows Volta layouts is still material.

## Best Possible Solution

Keep this PR open and update it, or land a replacement, to implement Volta tools/image path rewriting in src/infra/stable-node-path.ts with regression coverage for default ~/.volta, custom VOLTA_HOME roots, POSIX and Windows separators, .exe handling, and inaccessible-shim fallback. Then use it as the implementation path for #52184.
