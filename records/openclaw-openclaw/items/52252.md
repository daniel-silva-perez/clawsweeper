## Summary

Keep this PR open. Current main still only rewrites Homebrew Cellar Node paths, so the Volta LaunchAgent fragility described by #52184 remains unresolved; however, the submitted PR still needs the existing review concern addressed for custom VOLTA_HOME and Windows Volta layouts.

## Best Possible Solution

Keep this PR open and update it, or land a replacement, in `src/infra/stable-node-path.ts` with regression coverage for default `~/.volta`, custom `VOLTA_HOME`, Windows Volta layouts, `.exe` handling, and inaccessible-shim fallback; then use that implementation to resolve #52184.
