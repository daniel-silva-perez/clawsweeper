## Summary

Keep this PR open. Current main still lacks the central avatar override race fix: delayed chat-avatar refresh results can clear or replace the browser-local assistant data-image override, and the render resolver still prefers a fetched blob candidate before the local identity avatar.

## Best Possible Solution

Review and land this focused UI fix, or an equivalent patch, so browser-local assistant data-image overrides are authoritative across chat refresh, render resolution, and clear-override reload paths. The implementation should keep the existing same-origin/data-image/blob safety rules and retain focused tests in `ui/src/ui/app-chat.test.ts`, `ui/src/ui/controllers/assistant-identity.test.ts`, and `ui/src/ui/views/agents-utils.test.ts`.
