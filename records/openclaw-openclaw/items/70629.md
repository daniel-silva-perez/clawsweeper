## Summary

Keep this PR open. Current main still sends `thinkingBudget: 128` for Gemini 2.5 Pro and the generic `2.5-flash` branch, which includes Flash-Lite, so the PR is not obsolete; however, the proposed change appears broader than the current published Google contract because only Flash-Lite is documented with a 512 minimum while Pro and Flash still document lower minima. 

## What This Changes

The PR raises the Google Gemini 2.5 Pro and Flash/Flash-Lite `minimal` thinking budgets from 128 to 512 and adds a transport test matrix for Gemini 2.5 reasoning budget values.

## Best Possible Solution

Land a targeted Google plugin fix that aligns each Gemini 2.5 family with the verified provider contract: special-case Flash-Lite to a 512 minimal budget, keep Pro and non-Lite Flash at their documented minima unless live API proof shows the docs are stale, and keep focused transport tests for the affected budget tiers.

## Solution Assessment

Unclear. Raising Flash-Lite to 512 matches the current published dependency contract, but raising Pro and non-Lite Flash to 512 is not clearly the narrowest maintainable fix without live evidence that Google's public docs are stale.
