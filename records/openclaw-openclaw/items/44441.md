## Summary

Keep this PR open. Current main still lacks the optional macOS LaunchAgent watchdog runbook and the updating-guide cross-reference. The PR is focused and docs-only, and the author addressed the original launchd PATH footgun, but source-backed review concerns remain around the health predicate and profile/config/state environment propagation. The broader watchdog PR #46502 is still open, unmerged, and contains no docs changes, so it does not make this PR implemented on main.

## Best Possible Solution

Keep this PR open for maintainer docs review. The best path is to revise the runbook so the watchdog predicate either clearly says it is reachability-only or actually fails for the intended unhealthy states, propagate OPENCLAW_PROFILE/OPENCLAW_STATE_DIR/OPENCLAW_CONFIG_PATH where relevant, and keep the guidance explicitly optional and operator-managed. If maintainers prefer productized watchdog behavior, coordinate with #46502 rather than closing this PR as already implemented.
