## What This Changes

The issue requests fixing Windows exec/read tool argument corruption where Qwen/DashScope-style `</arg_value>>` XML suffixes are appended to command strings and file paths.

## Best Possible Solution

Keep this issue open and treat #48792 as the active implementation candidate.
