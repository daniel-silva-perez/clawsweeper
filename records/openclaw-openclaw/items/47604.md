Keep this PR open. Current main still has no Wear OS companion, so the contribution is not obsolete, but the patch is not merge-ready because the new Wear app enables backup/data transfer for files that include the device-auth private key.

## What This Changes

The PR adds a Wear OS companion app, shared Android gateway helpers, phone-side Wear proxying, direct watch gateway access, tile/complication services, background reply notifications, docs, and Android/Wear tests.

## Best Possible Solution

Keep the PR open for deliberate Android/Wear product and security review.
