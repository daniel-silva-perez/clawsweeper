## Summary

Keep this PR open. Current main still returns the host-built skills snapshot prompt before considering sandbox-local entries, and both attempt and compaction runs still skip live skill loading when the snapshot has resolved skills. The live PR branch remains a focused implementation candidate for open bug #43383 and now includes repairs for the earlier review concerns around empty sandbox entries, snapshot skill filters, remote eligibility, and env override alignment.

## Best Possible Solution

Keep this PR open and review or land the repaired branch, or replace it with an equivalent focused agents fix. The final implementation should force-load sandbox-local, config-eligible skill entries for non-rw attempt and compaction runs, rebuild prompts from prompt-visible sandbox entries instead of host snapshots, avoid empty-entry fallback to host prompts, preserve snapshot filters and remote eligibility, align env overrides with the prompt-visible entries, and keep the focused regression coverage. After that lands, #43383 can close as fixed.
