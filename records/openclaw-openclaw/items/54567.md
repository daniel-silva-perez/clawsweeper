## Summary

Keep open. Current main still runs the non-interactive daemon service install before enabling systemd linger, so this PR’s intended fix is not implemented on main. The PR remains a focused implementation candidate for the headless Ubuntu path described in #54517.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land the focused reorder, preferably with a narrow mocked sequence test for the non-interactive daemon install path or explicit live headless Ubuntu verification if maintainers decide source-level ordering is enough.
