## Summary

Keep this PR open. Current main still lacks the requested distinct Agents overview control for editing the global default primary model: the overview selector remains selected-agent scoped and the write path still updates `agents.list[].model`, not `agents.defaults.model`. The earlier ClawSweeper keep-open comment still matches current main at `17811480dae20f65b526454d75e971ae8424ec10`.

## Best Possible Solution

Keep this PR open for maintainer review, rebase, or a fresh port. The concrete path is to add a distinct global default primary-model selector in `ui/src/ui/views/agents-panels-overview.ts`, thread an `onDefaultModelChange` callback through `ui/src/ui/views/agents.ts` into `ui/src/ui/app-render.ts`, update `agents.defaults.model` without touching per-agent `agents.list[].model` overrides, preserve fallback lists in canonical `fallbacks` form, and keep focused tests for default-vs-override model state resolution.
