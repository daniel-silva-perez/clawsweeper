## Summary

Keep open. The requested global default-model selector is still not implemented on current main, and the PR contains useful source work but currently targets the older Agents overview shape rather than the split overview component that now renders the visible UI.

## What This Changes

The PR adds a Control UI Agents overview selector for the global active primary model, threads an `onDefaultModelChange` callback to update `agents.defaults.model`, preserves fallback lists through a shared helper, and adds focused UI utility tests.

## Best Possible Solution

Port the global default primary-model selector into `ui/src/ui/views/agents-panels-overview.ts`, thread a dedicated default-model callback through `renderAgents` into `ui/src/ui/app-render.ts`, write `agents.defaults.model` while preserving canonical `fallbacks`, and keep focused tests for default-vs-agent override resolution.
