## Summary

Keep this issue open. Current main still has the reported steer-mode failure shape: steer injection is gated on streaming state, Discord inbound messages are serialized by session before they reach the reply pipeline, and an open maintainer PR is still the active implementation path rather than a merged fix.

## Best Possible Solution

Keep this issue open until a focused fix lands. Review and land #73365 if it fully covers the lifecycle gate, or extend/supersede it with the missing Discord inbound steer bypass and regression coverage before closing this issue as implemented.
