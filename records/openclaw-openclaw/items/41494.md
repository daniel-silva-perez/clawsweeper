## Summary

Current main still does not fully solve the reported Gemini plain-text reasoning leak. It handles provider-marked Google thought parts and XML-like reasoning tags, but unmarked or localized reasoning preambles such as `Thinking Process:`, `Analysis:`, `Tankegang:`, `Analyse:`, or `Begrunnelse:` are not classified before delivery. PR #60551 is still open and appears to be the right implementation candidate or reference point.

## Best Possible Solution

Keep this issue open as the Gemini plain-text reasoning leakage tracker. Review and land PR #60551 or a replacement that adds a conservative shared sanitizer for untagged reasoning/planning preambles, with regression tests for the reported Gemini examples, localized headers, tagged fragments, code/prose pass-through, normal `/reasoning` visibility, and Telegram/message-tool delivery paths.

- No live Gemini plus Telegram roundtrip was run in this read-only review, but the current source still lacks the plain-text classifier needed for the examples in the report.
