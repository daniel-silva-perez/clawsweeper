## Summary

Keep open. Current main has adjacent generic tool-loop detection, but it is opt-in, exact-argument based, diagnostic-only for warnings, and does not implement the requested read-specific same-turn reread guard.

## What This Changes

The issue requests a warning-only, progress-aware same-turn reread detector for the first-class `read` tool to reduce repeated file-read token burn.

## Best Possible Solution

Add a read-aware core guard that normalizes per-run read file identities, tracks repeated same-file and small-file-set rereads, suppresses after real progress, returns a model-visible soft nudge, emits diagnostics, and documents/tests the warning-only behavior without changing exec policy by default.

## Repair Work Prompt

Implement a warning-only same-turn reread guard for the first-class `read` tool. Current main only has opt-in exact-params generic loop detection; do not enable global loop detection by default and do not try to solve the exec-runtime proposal except to avoid conflicting with #63352. Add per-run/turn read accounting in the core agent tool path, normalize read file identity from canonical `path` while collapsing offset/limit windows to the same file, detect repeated same-file reads and small 2-4 file-set reread loops, and suppress/reset after real progress such as `write`, `edit`, `apply_patch`, likely test/build `exec`, or materially broadened file evidence. The intervention must stay warning-only and model-visible, preferably as a synthetic/read-result nudge or equivalent returned content, and also emit a diagnostic event. Add focused tests for exact same-file rereads, A-B-A-B small file-set loops, legitimate offset continuation/expanded context, reset after edit/write/test, default config behavior, and diagnostic emission. Update `docs/tools/loop-detection.md` and `docs/gateway/config-tools.md`; if a user-facing behavior change lands, add a single-line active changelog entry crediting `Thanks @junoclaws`. Do not change provider/model defaults, bundled plugin behavior, or unrelated loop detectors.
