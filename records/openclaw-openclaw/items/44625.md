## What This Changes

## Best Possible Solution

Implement the fix inside the Slack plugin: discover scopes for modern granular bot tokens through `auth.test` response metadata or accessible Web API scope headers, preserve legacy workspace-app fallbacks, and add focused coverage for scope extraction and `channels capabilities` diagnostics.
