## What This Changes

This PR updates heartbeat empty-file detection to ignore a trailing docs-only `## Related` section with local documentation links and adds helper- and runner-level regression coverage for the current `HEARTBEAT.md` template shape.

## Best Possible Solution

Land the narrow heartbeat preflight fix after standard review and CI so scaffold-only heartbeat files seeded from the current docs template skip LLM calls again, while keeping real tasks under `Related` actionable. A broader docs-template/runtime-template split can be considered separately, but it is not required to solve this regression.

## Solution Assessment

Yes. The proposed fix is narrowly scoped to a trailing docs-only `Related` section with root-relative documentation links, and the added tests cover both the current scaffold and a non-doc task under `Related`; a loader-level template cleanup would be broader and less targeted for this regression.
