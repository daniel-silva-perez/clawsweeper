## What This Changes

This PR updates heartbeat empty-content detection to ignore a trailing docs-only `## Related` section of root-relative documentation links and adds helper- and runner-level regression tests for the current `HEARTBEAT.md` template shape.

## Best Possible Solution

Land the narrow heartbeat preflight fix after standard maintainer review and CI so scaffold-only heartbeat files seeded from the current docs template skip LLM calls again while real heartbeat tasks remain actionable. A broader docs-template/runtime-template split can be considered separately, but it is not required for this regression.
