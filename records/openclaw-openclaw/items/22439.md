The PR adds `agents.defaults.bootstrapTier`, tier resolution and filtering for workspace bootstrap files, root/hook provenance tagging, `bootstrap-extra-files` hook wiring, generated config metadata/help/types, and focused tests.

## Best Possible Solution

No. Tiered bootstrap control may be a reasonable feature, but this patch is not yet the best implementation because it conflates unset legacy behavior with explicit `standard` and leaves explicit `minimal` source-blind.
