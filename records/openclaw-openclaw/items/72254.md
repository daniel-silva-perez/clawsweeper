## Summary

Keep open. Current main still only logs failed sandbox skill copies and leaves any per-skill destination cleanup undone; this PR is a narrow, plausible fix, with remaining maintainer review focused on regression coverage, cleanup diagnostics, and coordination with the broader related sync rewrite.

## What This Changes

The PR adds best-effort removal of the resolved per-skill sandbox destination after `syncSkillsToWorkspace` fails to copy that skill.

## Best Possible Solution

Land the scoped cleanup in `syncSkillsToWorkspace` after maintainer review, ideally adding a focused regression test that simulates copy failure after destination creation while keeping deletion limited to the resolved target skill directory.
