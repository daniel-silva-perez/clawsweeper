## Summary

Keep this PR open. Current main still does not implement the requested `openclaw infer model run --thinking <level>` surface: the command has no option, the local model-run path forwards only `maxTokens`, and the gateway model-run request omits `thinking`. The PR remains a focused implementation candidate rather than cleanup.

## Best Possible Solution

Keep this PR open as the active implementation candidate. Review and land it, or an equivalent change, after deriving CLI validation from the shared thinking-level source, verifying the `pi-ai` simple-completion option contract, and keeping coverage for local forwarding, gateway forwarding, and invalid input rejection.
