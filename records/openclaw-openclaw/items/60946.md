## Summary

Current main still only partially satisfies the request: edit mismatch errors append bounded current file contents, but the requested similar-match, multiple-match, line-numbered retry guidance, apply_patch diagnostics, and exec command-not-found suggestions remain missing.

## What This Changes

The issue requests richer tool failure messages with contextual troubleshooting hints, similar text or ambiguity details, and concrete fix suggestions for edit, apply_patch, and exec failures.

## Best Possible Solution

Add a shared, bounded diagnostic helper for tool-result errors, then use it in edit and apply_patch mismatch paths to report safe line-numbered nearby matches or ambiguity locations with concise retry guidance; enhance exec command-not-found output only with conservative suggestions based on available local evidence.
