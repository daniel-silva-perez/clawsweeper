## Summary

Keep open. Current main still lacks the proposed gateway status event, but this PR should not merge as-is because the promised completion metadata is added only to a synthetic terminal lifecycle branch that normal embedded runs skip after receiving the real lifecycle end/error event.

## What This Changes

The branch adds an `agent.run.status` gateway event, registers it for gateway broadcasts, and attempts to include start/completion status, timing, model, usage, tool-call count, and exit reason from agent lifecycle events.

## Best Possible Solution

Revise the PR so the real embedded and ACP lifecycle events that reach the gateway carry consistent `startedAt`, model, usage, tool-count, and exit-reason data, then add focused gateway/agent tests for started, completed, failed, and interrupted status payloads.

## Reproduction Assessment

Yes. A high-confidence static reproduction path follows a normal embedded run: `handleAgentEnd` emits lifecycle `end`/`error`, `agentCommandInternal` marks `lifecycleEnded = true`, and the PR’s enriched synthetic terminal event branch is skipped.

## Solution Assessment

No. Adding the gateway broadcast is a reasonable direction, but enriching only the fallback synthetic terminal lifecycle event is not the narrowest maintainable fix because it misses the common terminal lifecycle path.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P2] Populate metadata on the actual terminal lifecycle event:** `src/agents/agent-command.ts:996-1019`
  - body: Normal embedded runs emit lifecycle `end`/`error` from `handleAgentEnd`, which makes `agentCommandInternal` set `lifecycleEnded = true` and skip this new synthetic event branch. The gateway will still broadcast `agent.run.status`, but common completion payloads will miss the promised model, usage, tool-call count, and often duration. Move the enrichment onto the lifecycle event that actually reaches the gateway, or preserve the run metadata before suppressing the synthetic end.
  - confidence: 0.9
