## Summary

Keep this PR open. Current main at 891c7d9f1c1f5e0f52935fadfeacf695b0ef1eb8 still has no symlink fallback for fixed workspace bootstrap files, so an external symlinked AGENTS.md/SOUL.md/USER.md/MEMORY.md is rejected by the boundary reader and rendered as missing. The maintainer replacement #73401 is open and closely matches the intended fix, but it is not merged, so current main does not yet implement this PR’s user-visible behavior.

## Best Possible Solution

Keep this PR open while maintainers review or land #73401, or explicitly choose and harden this branch. The best implementation belongs in src/agents/workspace.ts, scoped to fixed top-level bootstrap files only, using safe fd-based target validation, preserving hardlink/non-regular/dangling/oversized/extra-bootstrap rejections, and retaining regression tests for accepted and rejected symlink cases.
