## Summary

Keep this PR open. Current main at 6d323ee73640afdf02ed9fa3aded13da23359b98 still routes fixed workspace bootstrap files through the guarded boundary reader with no symlink fallback, so external symlinked AGENTS.md/SOUL.md/USER.md/MEMORY.md files are still reported missing. The maintainer replacement #73401 is relevant but open and unmerged in the provided context, and its own body says not to close this security-routed PR as part of that replacement path.

## Best Possible Solution

Keep this PR open while maintainers either land #73401 or explicitly choose and harden this branch. The best implementation belongs in src/agents/workspace.ts, scoped to fixed top-level bootstrap files only, with fd-pinned target validation and regression tests for accepted external bootstrap symlinks plus rejected hardlink, non-regular, dangling, oversized, and extra-bootstrap traversal cases.
