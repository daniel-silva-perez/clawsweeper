## Summary

Keep this PR open. Current main still lacks the targeted symlink fallback for fixed workspace bootstrap files, so an AGENTS.md/SOUL.md/USER.md/MEMORY.md symlink that resolves outside the workspace is still rejected by the boundary layer and rendered as missing. The open maintainer replacement #73401 may become the canonical landing path, but it is not merged on current main, and this contributor PR remains useful until a replacement ships or maintainers explicitly close it.

## Best Possible Solution

Keep this PR open while maintainers review or land #73401, or explicitly choose this branch. The best implementation should stay in `src/agents/workspace.ts`, allow only fixed top-level bootstrap symlinks when accepted, preserve hardlink, non-regular, dangling, oversized, fd identity, and target-identity protections, keep extra/glob bootstrap paths constrained to the workspace, and retain regression tests for accepted and rejected cases.
