## Summary

Keep open. The PR still contains a concrete path-normalization regression in the bundled facade loader, and the latest head has pending CI, so it is not ready for merge even though the main Skill Workshop hardening direction is valid.

## What This Changes

The branch hardens Skill Workshop path containment, write-boundary revalidation, and prompt-budget checks, adds plugin-SDK exports/docs/tests, and carries related QA/plugin-loader/test stabilization changes.

## Best Possible Solution

Land the Skill Workshop hardening only after the bundled override comparison uses the same user-path and realpath normalization as bundled-dir resolution, CI is green for the latest head, and maintainers are comfortable with the combined branch scope or split the support changes into a smaller follow-up.

## Reproduction Assessment

Yes for the PR finding: set OPENCLAW_BUNDLED_PLUGINS_DIR to a value such as `~/bundled`; bundled-dir expands it with resolveUserPath, but the new facade-loader comparison uses path.resolve on the raw string and can misclassify a module loaded from that override.

## Solution Assessment

Not yet. The overall hardening approach fits the existing root-bound write and SDK seam direction, but the proposed loader change should normalize the override with the same resolver before deciding whether runtime dependency staging is safe to skip.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.83

Full review comments:

- **[P2] Normalize bundled overrides before comparing paths:** `src/plugin-sdk/facade-loader.ts:149-156`
  - body: `OPENCLAW_BUNDLED_PLUGINS_DIR` is resolved through `resolveUserPath` when discovering bundled plugins, so values like `~/bundled` can point at the actual override root. This new check compares against `path.resolve` of the raw env string, so a module loaded from that override can be treated as outside it and `installRuntimeDeps` gets disabled. Use the same user-path/realpath normalization before skipping runtime dependency staging.
  - confidence: 0.87
