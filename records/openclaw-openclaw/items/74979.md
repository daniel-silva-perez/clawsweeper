## Summary

Keep open. Current main still has the original `abortable(activeSession.prompt())` race, and this PR is useful but incomplete because cancellation can still arrive after the new guard and before `AgentSession.prompt()` creates `Agent.activeRun`.

## What This Changes

The PR adds embedded Pi runner abort guards, Agent cleanup calls, and regression coverage for abort-before-prompt zombie Agent loops.

## Best Possible Solution

Make the actual Pi prompt-start boundary abort-aware: the run abort signal should be checked after dependency preflight and immediately before `Agent.prompt()` can create `activeRun`, with regression coverage for already-aborted, abort-before-activeRun, and abort-after-activeRun cases.

## Reproduction Assessment

Yes. #74859 gives a deterministic pre-aborted-signal reproduction, and the remaining PR gap can be reproduced by delaying real `AgentSession.prompt()` preflight, aborting before it calls `Agent.prompt()`, and observing that the later prompt start is not canceled.

## Solution Assessment

No. The current PR is a partial fix for the already-aborted and post-activeRun cases, but the narrow maintainable fix needs cancellation at the dependency prompt-start boundary rather than only before entering `AgentSession.prompt()`.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.87

Full review comments:

- **[P2] Make prompt start abort-aware:** `src/agents/pi-embedded-runner/run/attempt.ts:2721-2728`
  - body: The new guard only checks before entering `AgentSession.prompt()`. In the pinned Pi dependency, `prompt()` awaits command/input/template/auth/compaction/before-start work before it calls `this.agent.prompt(messages)`, while `agent.abort()` is a no-op until `activeRun` exists. If cancellation arrives in that preflight window, `abortable()` rejects and the prompt promise can later start an uncanceled Agent loop. Please make the actual prompt-start boundary observe the run abort signal and add coverage for abort-before-activeRun.
  - confidence: 0.87
