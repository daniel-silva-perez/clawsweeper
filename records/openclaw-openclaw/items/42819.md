Keep open. Current main still installs Playwright Chromium under the hidden `/home/node/.cache/ms-playwright` path, still documents that path for Docker persistence, and does not expose a stable Dockerfile-created `chrome` or `chromium` command on PATH. ## Best Possible Solution
