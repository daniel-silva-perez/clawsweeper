## Summary

Keep open. Current main still does not implement this PR’s intended last-chance blank visible prompt guard in the embedded runner. The existing helper still treats whitespace-only prompts as submit-worthy when replay history exists, and the runner only skips when prompt, history, and images are all empty. The PR remains a focused implementation candidate for #74137 and has no visible supply-chain or security-sensitive file changes.

Keep this PR open for normal maintainer review and targeted validation. If the additive guard is accepted, land it as the focused pre-provider fix for #74137; after merge, the linked issue can close as implemented.
