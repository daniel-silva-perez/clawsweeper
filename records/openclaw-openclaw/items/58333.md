## Summary

Current main does not implement this PR’s intended Control UI chevron fix. The broad descendant selectors are still present in `ui/src/styles/config.css`, the nested-object DOM still makes descendant matching relevant, and the regression tests proposed by the PR are absent. This PR remains a live implementation candidate, though its test review comments should be addressed before merge.

## Best Possible Solution

Keep this PR open, or supersede it with an equivalent target-repo fix. The best path is to scope the two chevron rotation selectors to the direct summary/header chevron, keep focused browser regression coverage for nested config objects and the pending-changes panel, address the existing review comments in the tests, and then merge after the relevant Control UI checks pass.
