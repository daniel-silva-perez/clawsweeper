## Summary

Keep this PR open for maintainer follow-up. Current main still does not implement the intended rw-mode skill sync, and the submitted PR is not safe to close as obsolete or merge as-is: the sync remains under the sandbox-workspace branch for the normal rw path, the PR has a broken resolver import per the provided diff/review context, and it bundles an unrelated locale change that is already present on main.

## Best Possible Solution

Keep the underlying rw-mode sandbox skills bug active, but revise or replace this PR with a focused change in `src/agents/sandbox/context.ts`: prepare the sandbox skill mirror for rw sessions, ensure prompt locations resolve to readable sandbox-workspace copies, preserve existing skill eligibility/filtering behavior, and add a targeted rw-mode regression test. Leave the already-implemented locale dropdown change out of the sandbox fix.
