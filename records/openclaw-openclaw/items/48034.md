## Summary

Keep this PR open. Current main still does not implement the intended rw-mode skill sync, and the submitted diff is not safe to merge as-is: the sync path remains unreachable for normal rw sandbox workspaces, the PR imports the bundled-skill resolver from the wrong module, and the locale dropdown change is unrelated and already covered on main.

## Best Possible Solution

Revise or replace this PR with a focused sandbox fix in `src/agents/sandbox/context.ts`: prepare a sandbox-readable skill mirror for rw sessions, make prompt skill locations resolve to readable sandbox-workspace copies, preserve current skill eligibility/filtering behavior and path guards, and add a targeted rw-mode regression test. Leave the already-implemented locale dropdown change out of the sandbox PR.
