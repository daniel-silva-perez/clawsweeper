## Summary

Keep this PR open. Current main still has the QQBot slash-command auth, pre-dispatch registration, clear-storage path, and log file-delivery gaps this PR targets. The PR is a plausible fix candidate, but its current head still needs maintainer follow-up on the reviewed /bot-version group regression, dm private-chat semantics, and focused regression coverage.

## Best Possible Solution

Keep this PR open as the broader QQBot fix candidate. Update it to remove the unintended /bot-version private-chat restriction unless maintainers explicitly want it, decide and test whether dm counts as private chat for c2cOnly, add focused tests for auth resolution, registry matching, private/group gating, clear-storage directory resolution, and file-delivery options, then review it as the canonical fix and coordinate the narrower #71158 afterward.
