## Summary

Keep open. ## What This Changes

This PR localizes Feishu bot, wiki, bitable, media, send-result, and docx error messages to Chinese, adds contextual IDs/URLs/tokens to failures, and updates related Feishu tests.

## Best Possible Solution

A rebased Feishu-owned localization change should preserve the current single-dispatch [System: ...] permission-notice contract and the docx workspace-only localRoots enforcement, then apply approved Chinese error copy with focused regression tests for copy, context exposure, and sandbox behavior.
