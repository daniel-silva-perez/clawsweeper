## Summary

Current main still lacks the PR's per-run browser cleanup dispatch guard, and both embedded subagent completion paths still converge on the same unguarded cleanup wrapper. The linked bug remains live, so this PR is an implementation candidate rather than obsolete.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land this narrow fix or an equivalent change that adds a per-run browser cleanup dispatch guard, resets it during rearm, and includes the parallel completion regression test for #68668.
