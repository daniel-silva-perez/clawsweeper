## Summary

Keep this PR open. Current main still does not relay ACP parent-stream events whose stream is neither `assistant` nor `lifecycle`, so the branch is not implemented on main; however, the PR needs maintainer review/update because it lacks a regression test, does not count relayed `system_event` text as progress for stall detection, and the current Codex event search did not prove the exact `stream: "system_event"` producer contract.

The PR adds a `system_event` branch in `src/agents/acp-spawn-parent-stream.ts` that relays non-empty `data.text` to the parent session with a fallback ACP context key.
