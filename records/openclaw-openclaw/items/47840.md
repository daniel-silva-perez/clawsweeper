Current main still has a canonical provider-id normalizer, but the Control UI Usage path continues to aggregate, filter, suggest, export, query-match, and display raw provider strings, so supported aliases like `z-ai`/`z.ai` and `aws-bedrock` can still appear or group separately from their canonical providers.
