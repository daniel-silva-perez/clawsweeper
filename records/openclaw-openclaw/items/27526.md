## Summary

Keep open. Current main has useful foundations for flat session listing/search, per-session labels/overrides, parent/child session metadata, and sub-agent orchestration, but it still does not implement the requested WebChat conversation directory, folders/tags/archive/persistent pins/recommendations, side-by-side multi-model collaboration, AI roles, consensus/meeting workflows, or project committee/task-tracking surface. The issue also has maintainer signal keeping it as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep this issue open as the canonical product tracker. The best path is to split it into explicit milestones: add session organization metadata and protocol/storage support for folders, tags, archive state, persistent pins, richer search, summaries, and recommendations; build the WebChat/Control UI directory navigation; then separately design the multi-model collaboration UX with roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, and task extraction/project tracking. Current session list/search, labels, checkpoints, parent/child sessions, and sub-agents should be treated as foundations, not completion.
