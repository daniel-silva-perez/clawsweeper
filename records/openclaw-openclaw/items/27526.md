## Summary

Keep open. This is a broad WebChat conversation-management and multi-AI collaboration product tracker that current main still does not implement, and maintainer discussion already marked it as canonical after duplicate #27603 was consolidated here. Current main has useful flat session foundations, WebChat history transport, labels/search/session overrides, dashboard session APIs, browser-local message pins, and sub-agent primitives, but it still lacks project/topic folder trees, multi-tag/archive/persistent-pin/recommendation metadata, side-by-side multi-model collaboration, role/consensus/meeting workflows, committee templates, task extraction, and project tracking.

## Best Possible Solution

Keep this issue open as the canonical product tracker. Split it into explicit milestones: session organization metadata and protocol/storage support for folders, tags, archive state, persistent pins, richer search, summaries, and recommendations; WebChat/Control UI directory navigation; and a separate multi-model collaboration design covering roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, task extraction, and project tracking. Treat #50404 and #73725 as narrower slices, not replacements for this broader tracker.
