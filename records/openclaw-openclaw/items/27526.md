## Summary

Keep open. Current main has useful foundations for WebChat transport, flat session listing/search, labels, per-session overrides, parent/child session metadata, and sub-agent orchestration, but it does not implement the requested conversation directory, folders/tags/archive/persistent pins/recommendations, side-by-side multi-model collaboration, AI roles, consensus/meeting workflows, or project committee/task tracking. The discussion also has maintainer signal keeping this as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep this issue open as the canonical product tracker, then split implementation into explicit milestones: session organization metadata and protocol/storage support for folders, tags, archive state, persistent pins, richer search, summaries, and recommendations; WebChat/Control UI directory navigation; and a separate multi-model collaboration design for roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, task extraction, and project tracking.
