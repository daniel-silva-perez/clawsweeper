## Summary

Keep open. Current main has useful foundations: WebChat transport, flat session list/search, labels, session overrides, dashboard session creation, browser-local message pins, and sub-agent orchestration. It still does not implement the requested project/topic conversation directory, folders/tags/archive/session pins/recommendations, side-by-side multi-model collaboration, AI roles, consensus/meeting workflows, or project committee/task tracking. Maintainer discussion also explicitly kept this as the canonical tracker after #27603 was consolidated here.

## Best Possible Solution

Keep this issue open as the canonical product tracker. Split it into explicit milestones: session organization metadata and protocol/storage support for folders, tags, archive state, persistent session pins, richer search, summaries, and recommendations; WebChat/Control UI directory navigation; and a separate multi-model collaboration design for roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, task extraction, and project tracking. Review narrower related work such as #50404 and #73725 as slices rather than closing this broader tracker.
