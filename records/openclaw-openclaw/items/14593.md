## Summary

Keep open. Current main still does not provide an official Docker/Linuxbrew path or Linux-container fallback for brew-only skill installers, and live API checks show open implementation candidates that explicitly target this issue.

## Best Possible Solution

Keep this issue open while maintainers choose and validate one focused path: land or reject #42303, #55940, and #57553, or implement an official opt-in Linuxbrew Docker setup, Linux-native installer metadata/fallbacks for affected bundled skills, or container-aware onboarding filtering. The chosen fix should include Docker skill-dependency regression coverage and Docker/skills docs updates.
