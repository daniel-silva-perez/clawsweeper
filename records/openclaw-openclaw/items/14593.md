## Summary

Keep open. Current main still lacks an official Docker/Linuxbrew path, an apt/native fallback for brew-only skill installers, or container-aware onboarding filtering. The reporter's openai-whisper example remains brew-only, and live PR checks show active implementation candidates still open, including PRs that explicitly target this issue.

## Best Possible Solution

Keep this issue open while maintainers choose a focused path: review and land or close #42303, #55940, and #57553, or implement an official opt-in Linuxbrew Docker setup, native Linux installer metadata/fallbacks for affected bundled skills, or container-aware onboarding filtering. The final path should include Docker skill-dependency regression coverage and updates to https://docs.openclaw.ai/install/docker and the skills docs.
