## Summary

Keep open. Current main and the latest release still contain the reported false-positive Raw-mode fallback path, and the linked implementation PRs remain open/unmerged; the fix needs maintainer review because it changes the secret-redaction safety gate.

## What This Changes

The issue asks to re-enable Control UI Raw mode by fixing false-positive redaction round-trip failures caused by undefined safe-bin keys and runtime-vs-source config comparison drift.

## Best Possible Solution

Ship a focused fix that removes empty safe-bin fields, compares the raw redaction round trip against a JSON-serializable authored/source config shape, preserves fallback for ambiguous redaction collisions, and adds regression tests for the reported configs.
