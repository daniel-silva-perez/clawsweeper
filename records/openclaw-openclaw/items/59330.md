The issue asks to re-enable Control UI Raw config mode by fixing false-positive raw redaction round-trip failures caused by materialized config differences, undefined safe-bin keys, and related raw redaction edge cases.

## Best Possible Solution

Ship a focused fix that omits empty safe-bin fields, compares the raw redaction round trip against a JSON-serializable authored/source config shape, preserves fallback for ambiguous redaction collisions, and adds regression coverage for the reported configs.
