## Summary

Keep open. Current main still has the reported Raw-mode false-positive path, and the open fix PRs that reference this issue are unmerged, so there is no conservative implemented or duplicate close path yet.

## What This Changes

The issue requests restoring Control UI Raw config editing by fixing safe-bin undefined-key normalization and the raw redaction round-trip comparison against materialized runtime config.

## Best Possible Solution

Land #59336 or a maintainer-owned replacement that omits empty safe-bin fields, compares the redacted raw round trip against a JSON-serializable authored/source config shape, fixes short sensitive-value substring corruption, and adds focused regression coverage for all three cases.
