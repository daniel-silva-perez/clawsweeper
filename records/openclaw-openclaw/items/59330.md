## Summary

Keep open. Current main still has the reported false-positive Raw-mode fallback path, and the two open implementation PRs that reference this issue are unmerged, so there is no conservative close path yet.

## What This Changes

Restore Control UI Raw mode availability by fixing config snapshot raw-redaction round-trip checks around empty safe-bin fields, materialized runtime config, and sensitive-value redaction edge cases.

## Best Possible Solution

Land #59336, #1824, or a maintainer-owned replacement that omits empty safe-bin fields, compares the raw redaction round trip against a JSON-serializable authored/source config shape, preserves safe fallback for ambiguous redaction collisions, and adds focused regression tests for the reported cases.
