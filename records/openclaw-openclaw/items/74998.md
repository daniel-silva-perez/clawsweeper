The PR adds a Google Gemini CLI subscription auth method with install/login prompts, model allowlist migration, docs, tests, changelog, and a provider-auth SDK re-export.

Keep the subscription onboarding path, but have it write canonical `google/*` model defaults/allowlists while selecting `agents.defaults.agentRuntime.id: "google-gemini-cli"`; then align docs/tests/changelog and run the Google extension plus changed-gate validation.
