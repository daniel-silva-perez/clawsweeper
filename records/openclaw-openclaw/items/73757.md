## Summary

Keep this PR open. Current main still has the exact docs/runtime mismatch the PR is trying to fix: the three docs pages say per-call `host=node` is allowed from `tools.exec.host=auto`, while the current runtime and regression test reject that override when a sandbox runtime is active. The PR is a narrow docs-only candidate for #61009, with no security-sensitive file changes in the provided diff.

## Best Possible Solution

Review and land this narrow docs-only PR, or apply equivalent wording to `docs/tools/exec.md`, `docs/tools/exec-approvals.md`, and `docs/nodes/index.md`, so the public docs match the existing sandboxed exec routing behavior and can close #61009 after merge.
