## Summary

Keep this PR open. Current main still contains the docs/runtime mismatch the PR targets: three docs pages say per-call host=node is allowed from tools.exec.host=auto, while runtime and tests reject node and gateway overrides from auto when a sandbox runtime is active. The PR is narrow docs-only work tied to open bug #61009, and the provided diff does not introduce security-sensitive changes.

## Best Possible Solution

Review and land this narrow docs-only PR, or apply equivalent wording to docs/tools/exec.md, docs/tools/exec-approvals.md, and docs/nodes/index.md so https://docs.openclaw.ai/tools/exec, https://docs.openclaw.ai/tools/exec-approvals, and https://docs.openclaw.ai/nodes match the existing runtime/test contract. After merge, #61009 can close as implemented.
