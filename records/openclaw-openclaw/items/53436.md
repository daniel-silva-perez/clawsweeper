## What This Changes

This PR adds GitHub Actions workflows for AI PR review, Codex autofix, coverage reporting, dead-code issue tracking, and bundle-size comments, modifies CodeQL/actionlint configuration, and adds an `agent-dna-auditor` package bundle.
