Keep this PR open. Current main still has the reported doctor/status cleanup-hint behavior and strict non-env SecretRef resolution path, and the PR is paired with still-open #52510. ## What This Changes

The PR changes doctor cleanup hints to render from detected extra gateway services, wraps `doctorCommand` in an env-scoped lenient exec SecretRef mode, updates SecretRef helper signatures/API baselines, and adjusts `maybeScanExtraGatewayServices()` to pass non-legacy extras into cleanup-hint rendering.
