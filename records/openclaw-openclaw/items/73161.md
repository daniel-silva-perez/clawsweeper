- **[P1] Preserve the real subagent runtime initializer:** `scripts/runtime-postbuild.mjs:14-20`
  - body: The synthetic `subagent-registry.runtime.js` wrapper replaces the real runtime entry during postbuild, but its `ensureContextEnginesInitialized` only flips a local boolean.   - confidence: 0.78

  - confidence: 0.9
