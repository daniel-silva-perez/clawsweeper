## Summary

Keep this issue open. Current main still renders injected bootstrap/project context as flat path/content text without inline `ctx:provenance`, `source`, `injectedAt`/`injected_at`, or volatility metadata, and the same author has open PR #54830 as the explicit Phase 1 implementation candidate with closing syntax for this issue.

## Best Possible Solution

Keep this issue open and review PR #54830 as the immediate Phase 1 candidate. If maintainers accept the direction, land a focused provenance-tagging change covering the context type, bootstrap assembly, prompt rendering, and tests; track memory freshness, stronger prompt rules, and orchestration-level auto-refresh as narrower follow-ups. If maintainers reject inline tags, close or redirect PR #54830 with the preferred design before resolving this tracker.
