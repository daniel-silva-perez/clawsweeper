## Summary

Keep this PR open. Current main still does not implement the proposed loop-aware compaction guard hook: there is no `agents.defaults.compaction.guard` config contract, no `guardEnabled` runtime field, no transcript-tail detector/scorer, and no conditional guard augmentation in `session_before_compact`. The PR is also a draft in an active same-author stack for #48238, with open prerequisite and follow-up PRs rather than obsolete work.

## Best Possible Solution

Keep this PR open. If product direction changed, close or redirect the whole stack with an explicit maintainer decision rather than sweeping this PR alone.
