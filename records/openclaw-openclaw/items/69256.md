## Summary

Keep open. Current main still lacks the PR's intended active-descendant guard for isolated cron direct-delivery cleanup, and the PR remains an active maintainer-touched implementation candidate with targeted tests and docs/changelog updates in its branch.

## Best Possible Solution

Keep this PR open and review or land this branch, or an equivalent patch.
