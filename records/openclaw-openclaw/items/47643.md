## Summary

Keep open. Current main has materially better Telegram token diagnostics, config reload handling, polling stall visibility, and Windows/network mitigations than the reported v2026.3.13 build, but the report still combines Telegram 401/auth, Windows config visibility, cross-surface routing, and sub-agent loop symptoms without current-release logs proving the whole observable problem is fixed.
