## Summary

Current main still does not implement MathJax, KaTeX, or another LaTeX math rendering layer in the Control UI chat markdown path. The related KaTeX PR #67103 is still an open implementation candidate, while prior PR #61510 was closed unmerged, so this issue should remain open until a focused implementation lands on main.

## Best Possible Solution

Keep this as the canonical Control UI math-rendering request. Review and land #67103 or an equivalent implementation in the shared Control UI markdown renderer, with tests for inline math, display math, code spans/fences, invalid formulas, sanitizer behavior, and ordinary dollar-sign text. Close only after the implementation is merged to main.
