## Summary

Current main still lacks MathJax, KaTeX, or another LaTeX math-rendering layer in the built-in Control UI markdown/chat renderer, and the related KaTeX PR #67103 remains open rather than shipped. This should stay open as the canonical Control UI math-rendering request.

## What This Changes

Add MathJax/LaTeX rendering support to the Control UI so inline and display formulas in chat messages render as equations instead of raw source text.

## Best Possible Solution

Keep this as the canonical Control UI math-rendering request and close it only after #67103 or an equivalent focused implementation lands on main with tests for inline math, display math, code spans/fences, invalid formulas, sanitizer behavior, and ordinary dollar-sign text.
