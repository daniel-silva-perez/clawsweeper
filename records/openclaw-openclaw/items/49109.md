## Summary

Keep open. Current main still lacks both central changes from this PR: cron helper calls are not quiet or skipped for JSON output, and the CLI gateway wrapper has no quiet/probe mode or bounded retry for transient transport failures. The related #54475 is only an open, narrower gateway candidate and does not cover the cron JSON/quiet helper behavior. This PR also remains too broad to merge as-is because its diff includes unrelated runtime, plugin registry, SSRF, build, generated-baseline, and channel-test churn.

## Best Possible Solution

Keep this PR open for maintainer follow-up, but do not merge it as-is. Review or land #54475, or an equivalent focused gateway fix, for the transient pre-hello close path. Then apply the cron helper quieting as a small separate change against current main with tests for JSON-mode skip, quiet/probe helper mode, and `expectFinal: false`. Split or drop the unrelated SSRF, plugin registry, build entrypoint, generated-baseline, runtime/API, and channel-test churn before this branch is considered mergeable.
