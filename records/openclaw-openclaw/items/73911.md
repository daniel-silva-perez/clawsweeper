Keep this PR open. Current main still drops explicit hidden `[[tts:text]]hello[[/tts:text]]` because the short-text guard is applied after directive parsing, while the PR head is a narrow runtime/test fix for the linked #73758 regression and includes the empty-markdown edge-case follow-up.
