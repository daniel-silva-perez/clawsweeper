Keep open. Current main has useful adjacent gateway status, restart-health, diagnostics, cron-alert, and rescue-bot surfaces, but it still lacks the requested external gateway-down watchdog, visible escalation path, startup-error artifact, dead-man monitoring, and first-class sibling health monitoring.
