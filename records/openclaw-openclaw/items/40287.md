## Summary

Keep open. Current main at e583db63c602d8d21b4ebdcd119061571a2e40a9 still lacks Slack `file_shared` event registration and setup docs, while this PR targets the removed `src/slack` tree. The work remains a plausible implementation candidate, but it needs to be ported into `extensions/slack` with setup/docs, tests, and a safer bounded history lookup.

## Best Possible Solution

Keep this PR open as an implementation candidate, but require a rebase or replacement port into `extensions/slack`: add and register a `file_shared` handler, update setup/docs manifest bot events, hydrate the matching message with a timestamp-anchored bounded `conversations.history` lookup, dedupe Slack retries and normal `message`/`file_share` delivery, handle multi-file uploads, preserve current media/security checks, and add focused Slack plugin tests.
