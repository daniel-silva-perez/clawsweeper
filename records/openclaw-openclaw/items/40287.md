## Summary

Keep this PR open. Current main at ef58307f843eed1a7832e9c23c519591f1b7cdbd still lacks a Slack `file_shared` event handler and setup/docs subscription entry, while this PR targets the removed `src/slack` layout. The work is still a plausible implementation candidate, but it needs to be ported into `extensions/slack` and tightened before merge.

## Best Possible Solution

Keep this PR open as an implementation candidate, but require a rebase or replacement port into `extensions/slack`: add and register a `file_shared` handler, update Slack setup/docs manifest bot events, hydrate the matching message with a timestamp-anchored bounded `conversations.history` lookup, dedupe Slack retries and normal `message`/`file_share` delivery, handle multi-file uploads, preserve current media/security checks, and add focused Slack plugin tests.
