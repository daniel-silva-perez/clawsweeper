## What This Changes

The PR adds trailing-dimension norm accumulation and `cosineSimilarity` regression tests to the old `src/memory/internal.ts` memory helper.

## Best Possible Solution

Yes. The active helper on current main returns `1` for `cosineSimilarity([1, 1], [1])` because it only accumulates norms over the shared prefix; zero-padding semantics should return `1 / Math.sqrt(2)`.
