## Summary

Keep open. The reported cosine-similarity bug is real on current main, but this PR changes the old `src/memory/internal.ts` path, which no longer exists. The branch updates the old `src/memory/internal.ts` cosine helper to include trailing vector dimensions in norm calculation and adds regression tests under `src/memory/internal.test.ts`.
