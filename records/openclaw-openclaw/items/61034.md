## Summary

Keep open. Current main still implements reasoning visibility and thinking-effort controls, but I found no config, session field, UI selector, docs entry, or runtime path for choosing the language of visible reasoning output; the earlier ClawSweeper keep-open comment remains consistent with the current tree.

## What This Changes

Request a configuration option, such as `agents.defaults.reasoningLanguage`, to choose the language used for visible reasoning output without changing the final response language.

## Best Possible Solution
