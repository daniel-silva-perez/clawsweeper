## Summary

Keep this PR open. Current main still does not implement the terminal-field preference the PR targets: the JSONL parser only preserves Claude result events plus generic item.text fallback, while the OpenAI Codex CLI backend still routes successful JSONL stdout through that parser. The PR remains a focused implementation candidate for the closed #59918 bug report rather than obsolete work.

## Best Possible Solution

Keep this PR open for normal maintainer review and landing. The best path is a narrow parser fix that prefers task_complete.last_agent_message, falls back to phase:"final_answer" assistant content, deliberately handles commentary-only fallback semantics, preserves Claude stream-json behavior, and keeps focused regression coverage in src/agents/cli-output.test.ts.
