The PR updates targeted provider hook resolution to include manifest model-catalog providers and aliases in owner lookup, avoids caching transient in-flight provider-hook misses, adds provider-runtime/provider ownership regression tests, and records the provider fix in the changelog.

## Best Possible Solution
