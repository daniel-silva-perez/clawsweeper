## Summary

Keep this issue open. Current main still accepts internal `throttleMs`/`minInitialChars` parameters in the Telegram draft stream, but does not expose `streamThrottleMs` or `minInitialChars` in Telegram config types, strict schema, docs/UI hints, or dispatcher wiring. The active path is the still-open PR #38255, not a merged fix.

## Best Possible Solution

Keep this issue open until PR #38255, or an equivalent maintainer-approved change, lands. The fix should add documented top-level and per-account Telegram preview cadence settings, validate them in the strict config schema, update generated metadata/docs/UI hints, merge account overrides through the existing Telegram config path, pass resolved values into `createTelegramDraftStream`, and cover defaults, validation, fallback behavior, and account overrides with focused tests.
