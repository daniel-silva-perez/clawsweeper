## Summary

Current main still lacks the requested cross-agent team/relationship view. The Control UI has useful per-agent panels, core workspace-file editing, tools/skills controls, and a flat Sessions table with some subagent metadata, but there is no graph/orchestration tab that joins agent config, subagent delegation, shared-workspace/TASK artifacts, and live session status. The related open umbrella #52803 overlaps, but this issue is more specific and should stay open for maintainer product direction rather than be closed as implemented or duplicate.

## Best Possible Solution

Keep this open for maintainer product review. The best path is to decide whether this should be tracked as the concrete graph/config-editing slice under #52803, then define a read-only MVP first: a Control UI relationship view backed by safe Gateway methods for agent config, session/subagent metadata, and any approved shared-workspace/TASK artifacts. Follow-up editing/chat slices should be separate PRs with explicit API and security review.
