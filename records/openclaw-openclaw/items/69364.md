## Summary

Current main still does not implement the requested Control UI cross-agent relationship/debugging view. The existing Control UI has single-agent panels, allowlisted workspace-file editing, tools/skills controls, and a flat Sessions table with useful subagent metadata, but no team graph or relationship tab that joins allowAgents/subagent delegation, per-agent config, shared-workspace/TASK artifacts, and live status. The request overlaps open umbrella #52803, but it is a narrower, concrete graph/config-debugging proposal and should stay open for maintainer product/API direction.

## Best Possible Solution

Keep this open for maintainer product review. The best path is to define a read-only MVP first: a Control UI relationship view backed by safe Gateway methods for agent config, subagent/session parent-child metadata, allowAgents relationships, and explicitly approved shared-workspace/TASK artifacts. Editing, tools/skills mutation, and per-agent chat should be split into follow-up issues or PRs with API and security review, while #52803 can remain the broader orchestration umbrella.
