## Summary

Keep open. Current main still treats `background: false` the same as omitting `background`: only `background === true` is special-cased, and otherwise the default yield timer is computed. The PR is a focused core exec-tool behavior change, not obsolete cleanup.

## Best Possible Solution

Keep the PR open and review it as a real core exec-tool API change. The best path is to port the behavior to current main by adding a `background === false` branch that sets `yieldWindow` to `null`, updating `src/agents/bash-tools.schemas.ts`, `src/agents/bash-tools.descriptions.ts`, and `docs/tools/exec.md`, and keeping the targeted regression test in `src/agents/bash-tools.test.ts`.
