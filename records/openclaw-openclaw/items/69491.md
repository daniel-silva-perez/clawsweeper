## Summary

Keep this PR open. Current main does not implement the Bedrock Converse fine-grained tool-streaming beta for Anthropic Claude models, so the PR is not obsolete. The patch still needs revision before merge because it overwrites existing `anthropic_beta` values, reuses a heuristic model predicate for an Anthropic-only payload field, and must merge with pi-ai’s existing `additionalModelRequestFields` behavior.

## Best Possible Solution

Keep the PR open and request a focused revision: add a Bedrock-owned payload patch that merges and dedupes default beta flags with any existing `additionalModelRequestFields.anthropic_beta`, preserves other `additionalModelRequestFields` such as pi-ai reasoning fields, limits injection to confirmed Anthropic Claude Bedrock targets, and coordinates with #39734 so future custom betas compose with the defaults.
