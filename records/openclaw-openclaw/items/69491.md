## Summary

Keep this PR open. Current main still does not forward default Anthropic beta flags through the Amazon Bedrock Converse stream route, so the PR targets a real provider gap. The proposed patch is not ready as-is because it overwrites existing `additionalModelRequestFields.anthropic_beta`, can collide with pi-ai's own Bedrock reasoning fields, and uses a broad Claude-name heuristic for an Anthropic-only request field.

## Best Possible Solution

Keep the PR open and request a focused revision: add a Bedrock-owned payload patch that merges and dedupes default Anthropic beta flags with any existing `additionalModelRequestFields.anthropic_beta`, preserves other `additionalModelRequestFields`, limits injection to confirmed Anthropic Claude Bedrock targets, and adds regression tests for guardrail/no-guardrail plus existing beta-field composition. Coordinate the data shape with #39734 so custom Bedrock betas can compose later.
