Keep this PR open. Current main still lacks the PR's central proactive behavior: there is no beforeAttempt hook in runWithModelFallback, no cron deadlineAtMs plumbing into isolated cron execution, and no test proving near-expired cron jobs skip a fallback before starting it. The PR adds an optional model-fallback beforeAttempt stop hook, threads a cron deadline into isolated agent execution, and adds tests so cron jobs can skip starting fallback model attempts when the remaining cron timeout budget is too low.
