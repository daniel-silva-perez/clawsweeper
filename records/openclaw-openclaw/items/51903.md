## Summary

Keep open. Current main still matches the reported schema/implementation mismatch: the flattened `nodes` tool schema permits omitted `node`, but the `describe` branch requires `node` before default-node resolution can run. Open same-author PR #51926 uses closing syntax and remains the active implementation path, so this issue should not be closed before that PR is reviewed, landed, or replaced.
