## Summary

Keep open. Current main still exposes channel agent-tool factories as cfg-only and drops sender identity before calling them, while adjacent core/plugin tool wiring already receives requesterSenderId and senderIsOwner. ## What This Changes

This PR adds optional requesterSenderId and senderIsOwner parameters to ChannelAgentToolFactory/listChannelAgentTools and forwards createOpenClawCodingTools sender context into channel-owned agent-tool factories.
