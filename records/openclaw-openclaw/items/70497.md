## Summary

Keep this PR open. Current main and v2026.4.26 still do not strip JSON Schema `if`/`then`/`else` in the Gemini schema cleaner, while Gemini provider paths still call that cleaner. The PR is a focused continuation of #69964 for #69768, and the provided Greptile/prtags context supports it as a small code-plus-regression-test fix rather than obsolete work.

Keep this PR open and review or rebase it against current main.
