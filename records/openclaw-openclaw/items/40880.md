## Summary

Keep open. Current main still has the hardcoded 5 MB media store and sandbox staging cap, and the top-level media config schema/type surface still does not expose media.maxBytes; PR #40950 remains the open implementation path that explicitly closes this issue, so the issue should stay open until that PR is resolved.
