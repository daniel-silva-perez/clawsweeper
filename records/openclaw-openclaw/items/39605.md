## Summary

Current main still has the reported bug for Discord and Slack: both native/slash command routing paths pass provider-startup config into route resolution, while Telegram has already moved to fresh runtime config. ## What This Changes

Native slash commands in Discord, Telegram, and Slack should respect `session.dmScope` when routing direct-message sessions.
