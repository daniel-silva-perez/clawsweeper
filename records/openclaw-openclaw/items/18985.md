## Summary

Keep open. Current main still rejects the reported MSYS2/MINGW `install.sh` path before install work begins, the installer still gives bash/zsh-only PATH persistence and warning guidance, and the linked Microsoft tracker #74163 is open tracking work rather than an implementation.

## What This Changes

Request support for running OpenClaw from Windows 11 MSYS2/MINGW Unix-like environments, including fish shell PATH setup guidance.

## Best Possible Solution

Keep this open until maintainers either explicitly support MSYS2/MINGW in `scripts/install.sh` with tested OS detection, Windows/MSYS-safe install decisions, fish-aware PATH persistence or guidance, docs, and smoke coverage; or document that Windows Unix-like environments should use WSL2 or `install.ps1` instead.
