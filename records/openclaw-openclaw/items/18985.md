Keep this open until maintainers either explicitly support MSYS2/MINGW in `scripts/install.sh` with tested OS detection, Windows/MSYS-safe install decisions, fish-aware PATH persistence or guidance, docs, and smoke coverage; or document that Windows Unix-like environments should use WSL2 or `install.ps1` instead.
