## Summary

Keep this PR open. Current main still lacks the PR's intended Mattermost message-tool fix: `handleAction` does not receive or forward `mediaLocalRoots`, while the core dispatch path does provide them and `sendMessageMattermost` can already consume them. The related #67796 work does not supersede this PR; it covers `filePath`/`path` media fallback and explicitly treats this PR as the `mediaLocalRoots` piece.

## Best Possible Solution

Keep the PR open and review or land a current-base version of the narrow fix: destructure `mediaLocalRoots` in `extensions/mattermost/src/channel.ts` `handleAction`, pass it to `sendMessageMattermost`, and keep the targeted regression test for the send action. Review #67796 separately for the complementary `filePath`/`path` fallback work.
