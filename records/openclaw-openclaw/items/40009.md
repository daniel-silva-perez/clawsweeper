## Summary

Keep open. Current main still renders the Control UI Raw config surface as a native textarea, with textarea-specific styling and textarea-based tests. The requested folding/highlighting/validation editor is not implemented, and the direct CodeMirror candidate PR is still open rather than shipped.

## Best Possible Solution

Keep this issue open. Review or consolidate #56401 and the adjacent #40732 work into one tested Control UI Raw config editor that provides JSON/JSON5 syntax highlighting, folding, validation/error display, full-document formatting, editor state coverage, and preserves raw availability, sensitive redaction/reveal, reset-to-saved, raw diff, `onRawChange`, `config.set`, `config.apply`, and `baseHash` behavior.
