## Summary

Keep this PR open. Current main still gates the Anthropic beta-header wrapper on explicit extra-param betas in the active provider plugin path, so OAuth/default beta merging is still skipped for the reported model/provider-level header case. The PR is a plausible implementation candidate for open bug #41444, but it needs to be ported from the older embedded-runner target to the current `extensions/anthropic/stream-wrappers.ts` wrapper path before merge.

## Best Possible Solution

Keep this PR open and update it for current main: move the unconditional beta-wrapper composition into `extensions/anthropic/stream-wrappers.ts`, pass an empty beta list when no explicit extra-param betas are configured, preserve direct-Anthropic/custom-endpoint suppression for implicit betas, and add a focused regression test for an OAuth token plus model/provider-level `anthropic-beta` with no `context1m` or `anthropicBeta` extra param.
