Keep this PR open. Current main still clears the Control UI chat queue during session switches and has no per-session queue store, while this PR is the active implementation candidate for open bug #73621. The provided PR diff is narrow UI/test work and is not obsolete or superseded by main.

Review and either land or request changes on this PR as the implementation candidate for #73621.
