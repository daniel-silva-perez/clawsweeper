## Summary

Keep this PR open. Current main still clears the Control UI chat queue on session switches and has no per-session queue store, while this PR is the active implementation candidate for the open linked bug #73621.

## Best Possible Solution

Review and either land or request changes on this PR as the implementation candidate for #73621. The likely product path is a small Control UI fix that stores queued messages per session, routes session-switch surfaces through one helper, and keeps the new focused regression test, without changing gateway or disk persistence.
