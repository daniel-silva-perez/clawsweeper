## Summary

Keep this PR open. Current main still returns early for fresh persisted token totals before the token-threshold gate when transcript-byte compaction is inactive, so the central bug fix is not implemented. The PR is a focused implementation candidate for the same-author open bug #63892, and overlapping work such as #66716 is still open rather than shipped.

## Best Possible Solution

Keep this PR open until maintainers either rebase and land a minimal fix here or explicitly choose another open implementation such as #66716. The fix belongs in `src/auto-reply/reply/agent-runner-memory.ts`, with regression coverage in `src/auto-reply/reply/agent-runner-memory.test.ts` for fresh persisted totals above and below the token threshold while transcript-byte compaction is inactive, preserving the current transcript-byte guard behavior.
