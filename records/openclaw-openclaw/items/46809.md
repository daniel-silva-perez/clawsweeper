Keep open. ## What This Changes

Add a dedicated Control UI memory panel that displays and edits USER.md, MEMORY.md, and today's memory/YYYY-MM-DD.md file.

## Best Possible Solution

Expose USER.md, root MEMORY.md, and today's memory/YYYY-MM-DD.md for the selected agent through a scoped Gateway API and Control UI memory surface with strict date/path validation, UI coverage, gateway path-safety tests, docs, and a user-facing changelog entry.
