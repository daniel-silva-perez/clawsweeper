## Summary

Keep open. Current main still implements the old Anthropic 1M beta-header behavior: `context1m: true` adds `context-1m-2025-08-07`, OAuth-style `sk-ant-oat` tokens still strip that beta and warn, and docs/tests still describe 1M context as beta-gated. Open PRs #45592 and #45613 already reference this issue with closing syntax, so the issue should stay open until maintainers review and land or replace one canonical fix.

## Best Possible Solution

Keep this issue open and review one canonical implementation PR, preferably consolidating #45592 and #45613. The fix should remove `context-1m-2025-08-07` injection and the OAuth-specific 1M beta skip/warning from the active bundled Anthropic plugin stream wrappers, preserve `context1m: true` as the explicit 1M context-window sizing opt-in, and update Anthropic docs plus focused stream-wrapper/context tests.
