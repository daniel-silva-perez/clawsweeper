## Summary

Keep this PR open. Current main still has the Amazon Bedrock Claude context-window gap the PR targets: discovery has no family fallback and still lacks the Claude 3 Opus/Sonnet and Claude 3.5 Sonnet base IDs, so those models fall through to the 32K Bedrock discovery default. The PR is a narrow implementation candidate for the linked open report, and the provided diff does not show a security or supply-chain concern.

## Best Possible Solution

Keep this PR open as the focused implementation candidate for #73328. Review the exact-entry additions, family fallback precedence, changelog credit, and regression tests; if accepted, land it and run the normal amazon-bedrock changed validation path.
