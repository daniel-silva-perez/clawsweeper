- **[P1] Keep MP3 files out of the Opus upload path:** `extensions/feishu/src/media.ts:538-548`
  - body: This routes `.mp3` and `audio/mpeg` files to `fileType: "opus"` and `msgType: "audio"`, but the send path uploads the original bytes without MP3-to-Opus transcoding. Feishu can reject or play broken audio when MP3 data is declared as Opus; leave MP3 on the generic file path unless it is transcoded first.
  - confidence: 0.9
- **[P2] Clear the timeout when fetch throws:** `extensions/ollama/src/stream.ts:711-738`
  - body: The new timeout is cleared only after `fetch` resolves, so DNS/connect/TLS failures skip cleanup and leave the timer pending until it fires. Move the cleanup into a `finally` path so repeated failed Ollama requests do not accumulate live timers.
  - confidence: 0.84
