## Summary

Keep this PR open. Current main has partial delivery-mirror hardening, but it still does not implement this PR's full consumer-path goal: filtering transcript-only OpenClaw assistant artifacts from visible WebChat history and the `sessions.get` API before pagination/budgeting. The PR also has explicit maintainer signal keeping it as the canonical Track B candidate under #69208.

## Best Possible Solution

Keep this PR open as the canonical Track B candidate under #69208, or replace it with an equivalent narrow rebased patch. The best implementation is a shared transcript-only OpenClaw assistant-artifact predicate applied before pagination and byte budgeting in `chat.history`, `sessions.get`, and any equivalent session-history visible/API surfaces, while preserving raw JSONL audit entries and adding focused regression tests.
