The PR adds an early Feishu text-message guard that drops whitespace-empty text payloads before route resolution and adds a regression test asserting no agent route or Feishu reply is invoked.

Feishu should ignore whitespace-empty text events before routing or session writes, while preserving existing media, post, share_chat, merge_forward, and audio-transcription behavior.
