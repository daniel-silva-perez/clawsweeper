## What This Changes

The PR adds raw Feishu thread-message body content, resolves media for media-typed thread-history messages, appends `[media attached: ...]` hints, and adds a Feishu bot regression test for image history downloads.

## Best Possible Solution

Land a polished version of this PR: preserve raw Feishu body content for thread-history records, resolve media for media-typed history messages before formatting ThreadHistoryBody, keep the existing attachment hint contract, add the download-failure regression requested in review, and run the Feishu plugin lane before merge.
