Keep open. Current main has several adjacent mitigations after the report, but there is still no source, release, or live-profile evidence proving the full Ubuntu packaged Gateway CPU saturation, Telegram-triggered slowdown, WS/RPC timeout, stuck heartbeat, and cron timeout regression is fixed.

## What This Changes

The issue requests a fix for a regression where packaged Linux Gateway becomes CPU-saturated, WS/RPC calls time out, and agent/heartbeat/cron work stops progressing after upgrading from 2026.4.20 to 2026.4.23 or later.
