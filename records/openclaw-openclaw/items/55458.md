- **[P1] Preserve OpenAI provider selections:** `scripts/sync-runtime-config.mjs:164`
  - body: `availableProviders` skips `openai` once `openai-codex` has consumed `OPENAI_API_KEY`, so an existing `openai/...` primary model is treated as unavailable and rewritten to Codex on startup. That breaks this script's stated preserve-user-overrides contract.
  - confidence: 0.91
- **[P1] Restore the non-root runtime user:** `Dockerfile:262`
  - body: The Dockerfile removes the existing `USER node` hardening and adds an entrypoint without switching back, so the gateway and entrypoint now run as root. That broadens the impact of any gateway, plugin, or dependency compromise and should not be bundled into a status diagnostics PR.
  - confidence: 0.89
- **[P2] Fix the Docker GPG awk quoting:** `Dockerfile:238`
  - body: The awk program is single-quoted, so `\"` is passed literally to awk and causes a parse error when `OPENCLAW_INSTALL_DOCKER_CLI` is enabled. That aborts the Docker build before key fingerprint verification can complete.
  - confidence: 0.87
- **[P3] Keep empty-message probe failures visible:** `src/commands/status.scan.ts:183`
  - body: The conditional spread drops `probeError` when a probe throws `new Error("")`, so that failure is still hidden. Preserve the field with a presence check such as `probeError !== undefined` and render based on presence if needed.
  - confidence: 0.78
