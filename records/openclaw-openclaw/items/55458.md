  - body: The awk program is single-quoted, so the embedded escaped quotes are passed literally and cause an awk parse error when `OPENCLAW_INSTALL_DOCKER_CLI` is enabled. That aborts the Docker build before key fingerprint verification completes.
  - confidence: 0.87

  - body: The conditional spread drops `probeError` when the probe throws `new Error("")`, so that failure remains hidden. Preserve the field based on presence, and render based on presence, so every thrown probe failure is visible.
  - confidence: 0.78
