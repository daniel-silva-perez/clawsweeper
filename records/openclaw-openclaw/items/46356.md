## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P1] Do not default metadata misses to direct DMs:** `extensions/discord/src/monitor/message-handler.preflight.ts:311-312`
  - body: When channel lookup fails and `guild_id` is absent, this forces DM semantics. Discord group DMs also omit `guild_id`, so this can bypass group-DM policy and collapse group conversations into direct-user routing.
  - confidence: 0.86
- **[P1] Ignore message types when deriving channel type:** `extensions/discord/src/monitor/message-utils.ts:132-139`
  - body: `message.rawData.type` is a Discord message type, not a channel type. Using it as `ChannelType` can make real DMs look non-DM or apply arbitrary channel semantics when REST metadata is unavailable.
  - confidence: 0.9
- **[P2] Skip mirror registration for delivered agent runs:** `src/gateway/server-methods/agent.ts:510-518`
  - body: For agent RPC calls with `deliver: true`, normal outbound delivery already sends the final reply. Registering `webchatMirrorTarget` here makes the lifecycle handler mirror the same text again.
  - confidence: 0.84
- **[P2] Preserve full peer IDs when resolving mirror targets:** `src/gateway/server-methods/webchat-mirror.ts:94-107`
  - body: Splitting the session key on every colon truncates peer IDs such as Matrix room IDs. A key like `agent:main:matrix:channel:!room:example.org` can resolve to `channel:!room` instead of the full room ID.
  - confidence: 0.82
- **[P2] Trim command text before mirroring externally:** `src/gateway/server-methods/chat.ts:1378`
  - body: This guard checks the untrimmed message for `/`, while command parsing trims first. Inputs like ` /help` can execute as local commands and still be mirrored to the external chat.
  - confidence: 0.88
- **[P1] Preserve Matrix sync persistence on shutdown:** `extensions/matrix/src/matrix/monitor/index.ts:143`
  - body: Replacing `releaseSharedClientInstance(client, "persist")` with `stopSharedClientInstance(client)` drops `stopAndPersist()`. Graceful Matrix restarts can then look like cold starts and skip backlog replay.
  - confidence: 0.86
- **[P1] Keep Matrix lifecycle setters on the live registry:** `extensions/matrix/src/matrix/thread-bindings.ts:65-66`
  - body: This file creates a new local manager registry while the exported lifecycle setters still use `thread-bindings-shared.ts`. Session lifecycle commands can update an empty registry instead of the active Matrix bindings.
  - confidence: 0.84
