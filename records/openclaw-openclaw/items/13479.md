## Summary

Keep this open. Current main still persists and searches cron job descriptions and renders prompt/system payload text plus run summaries, but it does not render the saved job description as the requested human-readable summary/detail in the Control UI cron job card or selected-job surface.

## What This Changes

Request to show each cron job's prompt/task and saved human-readable description/summary in the Control UI cron dashboard when inspecting a job.

## Best Possible Solution

Implement a narrow Control UI cron presentation fix: render saved job.description as a visible job summary/detail on cron job cards or the selected job detail area for both systemEvent and agentTurn jobs, while preserving prompt/system payload rendering, sanitized markdown behavior, and selected-job run history filtering.
