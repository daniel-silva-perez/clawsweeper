## Summary

Keep open. Current main still has the reported mismatch: `openclaw doctor` detects missing recent session transcripts but suggests a preview cleanup command without `--fix-missing`, while `sessions cleanup --dry-run` only counts missing transcript pruning when that flag is set.

## Best Possible Solution

Land a narrow commands fix in `src/commands/doctor-state-integrity.ts` so the missing-transcript `Preview cleanup impact` command includes `--dry-run --fix-missing`, and update `src/commands/doctor-state-integrity.test.ts` to assert that exact hint. Keep the cleanup behavior flag-gated unless maintainers intentionally choose a broader semantics change.
