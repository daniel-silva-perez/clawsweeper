## Summary

Keep open. Current main still has the reported mismatch: doctor warns about missing recent session transcripts, but its preview command omits `--fix-missing`, while sessions cleanup only counts missing transcript pruning when that flag is set.

## Best Possible Solution

Land a narrow commands fix in `src/commands/doctor-state-integrity.ts` so the missing-transcript `Preview cleanup impact` command includes `--dry-run --fix-missing`, then update `src/commands/doctor-state-integrity.test.ts` to assert that exact hint. Keep `sessions cleanup` missing-transcript pruning flag-gated unless maintainers explicitly choose a broader behavior change.
