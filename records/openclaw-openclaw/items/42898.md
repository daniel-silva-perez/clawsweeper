## Summary

Keep this PR open for maintainer review. Current main still misclassifies dotted read-only gateway actions as mutating, and the latest provided PR diff appears to fix that narrow classifier bug while addressing prior review concerns about cron/canvas fail-closed behavior.

## What This Changes

The PR updates `src/agents/tool-mutation.ts` to classify dotted gateway read-only leaf actions as non-mutating and adds regression tests in `src/agents/tool-mutation.test.ts` while keeping cron/canvas dotted or `lookup` actions fail-closed.

## Best Possible Solution
