## Summary

Keep this PR open. Current main still returns raw session transcript text and writes it into session-memory files without applying the existing LLM special-token sanitizer, while this PR is a focused implementation candidate for the still-open #69943 report. The overlapping broader PRs #69961 and #71257 are also open and unmerged, so there is no shipped or canonical replacement that makes this PR obsolete.

## Best Possible Solution

Keep this PR open for maintainer review. The practical path is to either land this focused token-literal sanitizer with its regression tests as a scoped partial fix, or explicitly choose #69961/#71257 as the broader canonical session-memory artifact sanitizer and ask this author to rebase, narrow, or close against that path.
