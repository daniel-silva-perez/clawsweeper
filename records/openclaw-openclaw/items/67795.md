## Summary

Keep this PR open. Current main still uses the narrower `linux-gnu*` installer OS check, and a read-only shell check reproduces the reporter’s openSUSE-style failure for `OSTYPE=linux`. The PR is a focused compatibility fix rather than obsolete work.

## Best Possible Solution

Keep the PR open for maintainer review. The best fix is the focused `OSTYPE == linux*` detection change, ideally with a small `install-sh` regression test covering plain `linux`, `linux-gnu`, WSL, and unsupported OS behavior.
