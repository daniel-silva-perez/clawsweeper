Keep open. Current main has fixed adjacent model-provider selection behavior and replaced the no-op-looking overflow action with explicit toolbar buttons, but the main Agents view regression remains: the selected agent is still chosen through a native label-only dropdown, and the Overview panel still does not surface the selected agent's avatar, emoji, or richer identity metadata.
