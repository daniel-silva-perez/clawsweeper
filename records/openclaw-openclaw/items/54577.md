## Summary

Keep this open. Current main still lacks the requested first-class agent activity timeline: no `openclaw timeline` CLI command, no `/timeline` chat command, no TUI or Control UI timeline route, and no dedicated `activity`/`AgentActivityEvent` gateway stream or JSON-exportable cross-agent lifecycle view. Existing task records, session lifecycle events, live tool cards, and per-session trajectory export are useful building blocks, but they do not satisfy the requested product surface. Related #60814 remains an open draft implementation thread rather than merged current behavior.

## What This Changes

The issue requests an interactive agent activity timeline with lifecycle timing, idle/error visibility, filters, `/timeline`, TUI monitoring, and JSON export.

## Best Possible Solution

Keep this issue open until maintainers scope and ship a reviewed activity timeline surface. The end state should include a durable or explicitly ephemeral lifecycle event contract, CLI `openclaw timeline` filters, `/timeline`, a TUI or Control UI view, safe JSON export/redaction semantics, docs, and focused tests; #60814 and #12831 are useful coordination points but are not yet shipped.
