## Summary

Keep this issue open. Current main has adjacent task, subagent, trajectory, and live agent-event plumbing, but it still does not ship the requested built-in agent activity timeline surface: no `openclaw timeline` command, no `/timeline` chat command, no Control UI/TUI timeline route, and no durable cross-agent lifecycle timeline with the requested filters/export behavior. Related #60814 remains only an open draft candidate in the provided context, not a merged implementation of this issue.

## Best Possible Solution

Keep this open until maintainers scope and ship the built-in agent activity timeline product. A good implementation path is to build on the task registry, subagent registry, trajectory capture, and gateway agent events, adding durable lifecycle event capture, `openclaw timeline --since --agent --type --json`, `/timeline`, a Control UI or TUI timeline view, public docs, and focused tests. #60814 should be reviewed, revised, landed, or replaced rather than used as a close reason before it merges and covers this scope.
