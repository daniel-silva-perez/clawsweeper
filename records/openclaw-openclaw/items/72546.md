## Best Possible Solution

Enforce a clear memory-core cron sweep invariant: a successful sweep with eligible workspaces should leave expected phase/deep artifacts and memory.dream.completed events, while all-workspace failures before artifact writes should produce a failed cron result or an explicitly diagnosable non-success state.
