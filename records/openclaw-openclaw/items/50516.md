## Summary

Keep this PR open. Current main still has the Nextcloud Talk webhook verifier behavior the PR is meant to harden: an early return on signature-length mismatch followed by a manual XOR comparison. The PR is narrow, security-relevant, and not implemented or superseded on main, but its mismatch-branch dummy comparison should be tightened before merge.

## Best Possible Solution
