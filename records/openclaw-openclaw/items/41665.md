## Summary

Keep this PR open. Current main still has the Windows onboarding problem the PR targets: installer filtering only hides specs that carry OS metadata, and current bundled skills still contain unscoped Homebrew installers. The PR is also stale against current skill inventory and includes an xurl installer-kind change that needs deliberate maintainer review rather than cleanup closure.

## Best Possible Solution

Keep this PR open for maintainer follow-up or replacement. The best path is to refresh it against current main, audit all bundled Homebrew installer specs, add top-level or per-installer OS metadata where appropriate, preserve or add Windows-capable installers when available, explicitly review the xurl node installer change, and add a small manifest audit test for unscoped Homebrew-only installers.
