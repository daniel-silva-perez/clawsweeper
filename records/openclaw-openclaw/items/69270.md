## Summary

Current main at 0382ac5 still does not implement this PR's central compaction/reset invariant fixes. Reset hooks still omit normalized provider context, subscribe-time compaction hooks still pass only sessionKey, the shared hook-message-provider resolver is absent, and auto/run compaction accounting still ignores tokensAfter: 0. The paired same-author issue #69269 remains open and this PR is still the active implementation path, so it should stay open for maintainer review or replacement by an equivalent canonical fix.

## Best Possible Solution

Keep this PR open for maintainer review and landing, or replace it with an equivalent canonical fix. Close #69269 only after the chosen fix merges, and treat #60858 as a separate compaction-input issue.
