## Summary

Keep this PR open. Current main still has the `/bash` exec-default inheritance gap, and the PR is a focused implementation carrying forward #50392 and #41598, but the patch needs maintainer/security resolution before merge because it starts applying persisted session exec policy fields to direct chat bash execution.

## What This Changes

The PR updates chat `/bash` and `!` execution to inherit global, agent, and session `tools.exec` defaults, forwards `sessionEntry` into the bash handler, adds focused regression/routing tests, and adds a changelog entry.

## Best Possible Solution

Land a focused fix that makes `/bash` inherit the same effective exec defaults as the normal exec tool while preserving elevated gates and bang routing, and explicitly define whether session-level exec overrides are allowed for direct bash or must be owner/admin-gated or limited to stricter settings.

## Reproduction Assessment

Yes. The functional gap is reproducible by source inspection or a mocked unit test asserting the arguments passed to `createExecTool`: current main only forwards timeout/notify defaults, while the PR forwards the missing exec policy fields.

## Solution Assessment

Unclear. The inheritance approach is the narrowest functional direction, but the session override part is security-sensitive and should not merge until the maintainer/security decision on direct bash trust boundaries is explicit.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.78

Full review comments:

- **[P1] Gate chat bash session exec overrides:** `src/auto-reply/reply/bash-command.ts:65-72`
  - body: This resolver gives stored `sessionEntry.execHost`, `execSecurity`, `execAsk`, and `execNode` precedence for `/bash`. Those fields can be persisted by authorized non-internal chat directives, and `/bash` is a direct host-command route; an elevated-allowlisted chat sender could weaken the approval policy for later bang commands. Please gate these session overrides for direct bash, or limit them to stricter settings, before consuming them here.
  - confidence: 0.76
