## Summary

Keep open. Current main still emits a `response.completed` SSE terminal event for the pending-tool-call path even though the response resource is `status: "incomplete"`; the PR is pointed in the right direction, but it needs a current-main rebase plus test and docs updates before it is mergeable.

The PR changes the OpenResponses gateway SSE tool-call terminal event to `response.incomplete`, adds the matching streaming event type to the schema, updates regression coverage, and fixes OpenResponses doc-comment URLs.
