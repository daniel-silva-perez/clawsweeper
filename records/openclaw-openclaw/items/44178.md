Keep this PR open. Current main still does not implement the proposed `feishu_doc` comment actions; the shipped overlap is limited to `feishu_drive` comment listing/add/reply and Feishu comment-event handling. The PR also has an unresolved pagination review finding and adds comment update/delete network capability, so it needs maintainer product and security review rather than cleanup closure.
