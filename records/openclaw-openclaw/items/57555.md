## Summary

Keep this issue open. Current main still ships the bundled `summarize` skill with a hard `summarize` binary requirement and only a `steipete/tap/summarize` Homebrew installer. The Control UI has generic mutation error plumbing now, but `skills.update` still only writes the enabled flag and does not validate the missing binary or show the requested requirement-specific enable failure. The later comment that the Homebrew formula was deleted makes the bundled installer path look even less resolved.

## Best Possible Solution

Keep the issue open and either remove/move the bundled `summarize` skill to ClawHub, mark it platform-limited with accurate installer metadata, or add a maintained Linux/download install path plus explicit Control UI feedback when enabling a skill cannot make it eligible because required binaries are missing.
