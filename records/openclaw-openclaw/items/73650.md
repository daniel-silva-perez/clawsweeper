## Summary

Keep this PR open. Current main still does not implement the shared task snapshot: `getStatusSummary()` still calls the registry and audit helpers separately, and each helper still reconciles by calling `listTaskRecords()` independently. The PR remains a focused implementation candidate for open #73531, with only source/test/changelog files in the provided diff and no visible security-sensitive or supply-chain changes.

## Best Possible Solution

Keep this PR open for normal maintainer review. Land this shared-snapshot implementation with its regression tests, or replace it with an equivalent main-branch fix that makes `getStatusSummary()` reconcile inspectable tasks once and pass the same snapshot into the registry and audit summaries. Keep #73531 open until the fix merges.
