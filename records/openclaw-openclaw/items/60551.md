## Summary

Keep this PR open. Current main still does not implement the PR’s intended last-mile sanitizer for untagged plain-text reasoning/planning preambles before outbound delivery, and the related Gemini reasoning-leak family still has open follow-up context. The PR is not obsolete or implemented on main; it needs maintainer review of the sanitizer tradeoffs rather than cleanup closure.

## Best Possible Solution

Keep the PR open as a focused implementation candidate. Maintainers should review the latest sanitizer diff against the open Gemini reasoning-leak cases, decide whether the structural-contamination cleanup belongs in the same change or a narrower follow-up, and require regression tests for both stripped leaks and preserved legitimate multi-paragraph/code/CJK replies before landing or closing it.
