## Summary

Keep this PR open. Current main still lacks the intended `infer model run --model` canonicalization: local dispatch receives the raw override, gateway dispatch splits and forwards the raw provider/model strings, and current tests do not cover mixed-case catalog model refs. The linked #73715 bug remains open, so this PR is still an active implementation candidate rather than cleanup.

## Best Possible Solution

Keep this PR open as the active implementation candidate for #73715. Review the resolver approach and regression tests for both local and gateway transports, then land this PR or an equivalent fix so the bug can close after merge.
