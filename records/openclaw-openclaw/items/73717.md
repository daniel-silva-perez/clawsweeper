## Summary

Keep this PR open. Current main still does not implement the PR’s intended `infer model run --model` canonicalization: the local path passes the user-supplied model ref directly into simple-completion preparation, and the gateway path splits and forwards the raw provider/model strings. The linked #73715 bug remains open with concrete reproduction, so this PR is still a focused implementation candidate rather than cleanup.

## Best Possible Solution

Keep this PR open as the active implementation candidate for #73715. Review the resolver approach, the local and gateway regression tests, and the custom-ref preservation behavior; then land this PR or an equivalent fix so the linked bug can close after merge.
