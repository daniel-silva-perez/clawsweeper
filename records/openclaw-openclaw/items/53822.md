## Summary

Keep this PR open. Current main still only supports raw `sandbox.docker.binds`; it does not expose `sandbox.docker.volumes`, merge volume config, validate volume strategies, or emit Docker `--mount` args. The PR remains a relevant implementation candidate, and the provided review discussion shows security feedback on reserved target handling was actively addressed in later PR commits, so this needs normal maintainer review rather than cleanup closure.

## Best Possible Solution

Keep the PR open as the implementation candidate.
