This PR adds a config validation helper that appends min/max numeric constraint hints to mapped Zod `too_big`/`too_small` errors and adds regression coverage for `session.agentToAgent.maxPingPongTurns`.

## Best Possible Solution

## Reproduction Assessment

Yes.
