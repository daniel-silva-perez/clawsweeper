## Summary

Keep open: current `main` still matches the bug described by PR #57477. The gateway still reports skipped updates as `ok: true`, the update runner still emits `skipped` for the same dirty/no-upstream/not-git-install cases, and the Control UI still only shows a simple `Update now`/`Updating…` button with no skip-reason, force, or live-progress flow.
