## Summary

Keep open. Current `main` still implements ACP `streamTo:"parent"` relays as local `onAgentEvent(...)` listeners only, without the PR’s proposed gateway-state recovery from `chat.history` / `agent.wait`, so the reported missing child output/completion path is not implemented on `main`. The PR also has unresolved review feedback about race/duplication behavior in the proposed fallback.
