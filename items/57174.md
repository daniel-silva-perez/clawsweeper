## Summary

Current `main` does not implement per-agent compaction overrides in OpenClaw config. Compaction is still defined and documented only at `agents.defaults.compaction`, while `AgentEntrySchema` has no `compaction` field. There is a partial per-workspace workaround via embedded Pi project settings, but that is not the requested `agents.list[]`/per-agent config surface.
