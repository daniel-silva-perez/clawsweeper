The issue has concrete reproduction details and an open linked PR (#58993) proposing the relevant fix, while current main still classifies Google Chat spaces using only the deprecated/legacy `space.type` field and does not model `space.spaceType` or `space.singleUserBotDm`. This is a plausible current bug, not a conservative close candidate.
