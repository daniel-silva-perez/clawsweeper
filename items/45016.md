## Summary

Keep open. Current `main` has the requested `last-only` stripping primitive, but only as a provider/model-specific replay policy. There is still no user-facing `thinkingHistory` config or general transcript policy that strips thinking blocks across sessions by default, and current code explicitly preserves prior thinking for newer Claude models.

## Evidence
