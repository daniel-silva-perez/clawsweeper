## Summary

Keep open. Current `main` still lacks the PR’s core ACP event-bridging and stall-classification changes, so the reported misclassification is not strongly evidenced as fixed on `main`. The relay still collapses all silent ACP runs into a generic "waiting for interactive input" notice, and both ACP execution paths still ignore `status`/`tool_call` runtime events instead of forwarding them into agent event streams.
