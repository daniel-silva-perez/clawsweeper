## Summary

Keep open. Current main still has a plausible Feishu outbound/inbound session-key mismatch for bare `oc_` OpenChat targets: the Feishu outbound session-route helper defaults unprefixed targets to `direct`, while inbound message handling routes group/topic-group events as `group`. The issue is recent, concrete, author association is NONE, and the report remains actionable rather than closeable.
