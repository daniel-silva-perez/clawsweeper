## Summary

Keep open. Current main does not implement this PR's requested transport-level signed/redacted thinking preservation, but the submitted patch is too broad: preserving opaque Anthropic thinking signatures for every target model conflicts with OpenAI Responses conversion and has an unresolved P1 review finding. The useful path is a narrower provider-compatible fix, not conservative auto-close.

## Best Possible Solution

Keep the PR open for revision or replacement with a provider-compatible replay fix: preserve signed/redacted thinking only when the target replay path can legally send Anthropic/Bedrock thinking blocks, continue downgrading or dropping them for OpenAI/Google-style targets, and add regression tests for both Anthropic-compatible cross-model replay and the OpenAI Responses opaque-signature failure case.
