## Summary

Keep open. Current `main` still has the reported mismatch: the macOS node advertises `screen.record`, the docs still list it as a macOS node command, but Gateway policy continues to classify `screen.record` as dangerous and excludes it from the default macOS allowlist unless `gateway.nodes.allowCommands` explicitly enables it.

## Evidence
