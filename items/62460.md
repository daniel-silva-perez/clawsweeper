## Summary

Keep open. Current `main` still force-runs `qmd embed` during boot updates, including `searchMode: "search"`, and there is no `memory.qmd.update.embedOnBoot` or equivalent config knob. The behavior is explicitly documented in code comments and covered by tests, so this feature request is still live rather than already implemented or obsolete.

## Evidence

- **Boot path still forces a QMD update on startup:** `initialize()` calls `this.runUpdate("boot", true)` whenever `memory.qmd.update.onBoot` is enabled, so boot sync is always a forced update.
