## Summary

Keep open. Current `main` does not implement cron pre-hooks, and the submitted PR as shown in the provided diff still has a functional gap: it adds CLI/schema surface for `hooks.pre` but `createJob`/`applyJobPatch` do not persist hooks into stored cron jobs, so the advertised gating behavior would not reliably take effect.
