## Summary

Keep open. Current main still has the reported mismatch: the doctor missing-transcript warning suggests a dry-run cleanup command without `--fix-missing`, while `sessions cleanup` only counts missing transcripts when that option is present. The linked fix PRs #54896 and #54925 are useful prior work, but the provided GitHub context says both were closed unmerged, and current main still reflects the old behavior.

## Best Possible Solution

Patch `src/commands/doctor-state-integrity.ts` so the missing-transcript `Preview cleanup impact` command includes `--fix-missing`, and tighten `src/commands/doctor-state-integrity.test.ts` to assert that exact preview hint. The existing `sessions cleanup` semantics can stay unchanged.
