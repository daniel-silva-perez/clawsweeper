## Summary

Current main still has the flat-only dated-memory matcher, so nested dated memory files are not decayed. This PR is a focused implementation candidate for the still-open #67594 and should stay open for maintainer review and landing or an equivalent patch.

## Best Possible Solution

Keep this PR open. The best path is to review and land this focused memory-core fix, or apply an equivalent change that broadens `DATED_MEMORY_PATH_RE` for nested dated memory files while preserving evergreen behavior and the added regression coverage.

## Evidence
