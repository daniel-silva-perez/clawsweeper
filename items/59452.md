## Summary

Keep open. The requested `openclaw cron add --dry-run` feature is not implemented on current main: the cron add command exposes `--json` but no `--dry-run`, and it still always calls `cron.add` after building params. The linked implementation work exists in PR #59490, but that PR is still open, so the issue should not be closed as implemented on main.

## Evidence

- **Current main lacks --dry-run option:** `registerCronAddCommand` defines cron add options through `--json`, then enters the action handler; no `--dry-run` option is registered in this command on current main.
  - file: [src/cli/cron-cli/register.cron-add.ts:936](https://github.com/openclaw/openclaw/blob/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75/src/cli/cron-cli/register.cron-add.ts#L936)
  - sha: [d6ef1fcf24ee](https://github.com/openclaw/openclaw/commit/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75)
- **Current main still commits through gateway:** After constructing `params`, the cron add action calls `callGatewayFromCli("cron.add", opts, params)` and prints the response, with no dry-run short-circuit before the API call.
  - file: [src/cli/cron-cli/register.cron-add.ts:1170](https://github.com/openclaw/openclaw/blob/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75/src/cli/cron-cli/register.cron-add.ts#L1170)
  - sha: [d6ef1fcf24ee](https://github.com/openclaw/openclaw/commit/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75)
- **Linked PR is relevant but not merged:** Issue #59452 links PR #59490 (`feat(cli): add --dry-run flag to cron add`). The PR page shows it is open and proposes exactly this feature, including `--dry-run` output and tests, but it has not landed on main.
  - command: `GitHub issue/PR context for #59452 and #59490`
- **Public CLI docs do not show this as shipped:** The public CLI reference lists `cron add` as requiring `--name`, one schedule option, and one payload option, and lists common cron options; it does not document `cron add --dry-run` as an available mode.
  - file: [docs/cli.md](https://github.com/openclaw/openclaw/blob/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75/docs/cli.md)
  - command: `Public docs lookup: https://docs.openclaw.ai/cli`
