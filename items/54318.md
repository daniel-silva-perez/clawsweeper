## Summary

Current `main` still bypasses the standard outbound hook pipeline for Discord inbound replies, so this PR is not already implemented on `main`. The proposed fix remains relevant, and the review context also shows unresolved correctness concerns in the PR itself, so conservative cleanup should keep it open.
