## Summary

Keep this PR open. Current main still lacks Control UI LaTeX/KaTeX/MathJax rendering, #42840 remains the open canonical feature request, and this PR is the newer current-renderer implementation attempt rather than an obsolete duplicate. It is not mergeable as-is because review comments identify blocking regressions in the PR head.

## Best Possible Solution

Keep this PR open as a rework candidate for #42840. The best path is to update it against current main, preserve the existing markdown cache and DOMPurify link-hardening hooks, add robust math delimiter handling that avoids currency and code-span/code-fence false positives, narrowly allow KaTeX-required styling, and add focused tests for inline math, display math, invalid LaTeX, mixed markdown, currency text, and single/multi-backtick code spans. If the author does not continue, a successor PR can take over the same Control UI markdown-renderer approach.
