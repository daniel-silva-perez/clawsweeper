## Summary

Keep open. Current `main` partially implements the PR by accepting `GOOGLE_CLOUD_API_KEY` in `resolveEnvApiKey("google-vertex")`, but the rest of the requested cleanup is not on `main`: the provider env-var registries still omit `google-vertex`, so shell-env fallback and secret scrubbing do not learn this key, and the docs still say Vertex uses ADC only.
