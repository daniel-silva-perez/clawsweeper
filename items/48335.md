## Summary

Current `main` does not contain the PR’s claimed runtime behavior. The safeguard runtime still wires only existing compaction and quality-guard fields, the compaction runtime type has no `guardEnabled`, the safeguard hook applies only the existing base instructions, and the repo has no `compaction-guard` or `transcript-tail-detector` modules. This PR is not implemented on `main`, so there is not strong evidence to close it.
