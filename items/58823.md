## Summary

Keep PR #58823 open. Current main still has the old subagent model precedence in both the normal spawn path and the cron isolated-agent path, and tests on main still assert that an agent's own model wins over `agents.defaults.subagents.model`. The PR is therefore not implemented on main and appears to address a real current behavior mismatch with the documented sub-agent default model contract.
