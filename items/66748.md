Keep this issue open. Current main still unconditionally requests `exec-event` heartbeats from the bash exec completion paths, and the same-author implementation PR that says it fixes this issue is still open and unmerged. Per maintainer cleanup rules, the issue should remain open while that PR is an implementation candidate.
