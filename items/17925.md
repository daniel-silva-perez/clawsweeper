## Summary

Keep open. Current `main` has Gemini as a managed `web_search` provider and OpenAI/Codex-native web-search injection, but I found no native model-request passthrough for Google Gemini models and no Z.AI web-search passthrough/provider implementation at all.

## Evidence

- **Native search docs are limited to OpenAI/Codex:** `docs/tools/web.md` documents native web search only for direct OpenAI Responses models and optional Codex models; it does not describe native Gemini or Z.AI passthrough.
  - file: [docs/tools/web.md:117](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/docs/tools/web.md#L117)
  - command: `nl -ba docs/tools/web.md | sed -n '115,170p'`
  - sha: [a59d1bd46d9f](https://github.com/openclaw/openclaw/commit/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3)
- **Google registers Gemini as a standalone web_search provider:** The Google plugin separately registers `createGeminiWebSearchProvider()` in the plugin entry, while the model provider registration only sets the normal Google stream function and hooks.
  - file: [extensions/google/index.ts:161](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/extensions/google/index.ts#L161)
  - command: `nl -ba extensions/google/index.ts | sed -n '130,165p'; nl -ba extensions/google/provider-registration.ts | sed -n '1,120p'`
  - sha: [a59d1bd46d9f](https://github.com/openclaw/openclaw/commit/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3)
- **Gemini grounding exists only inside the managed web_search provider path:** `extensions/google/src/gemini-web-search-provider.runtime.ts` sends `tools: [{ google_search: {} }]` from the standalone web-search provider implementation, not from the Google model transport path.
  - file: [extensions/google/src/gemini-web-search-provider.runtime.ts:64](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/extensions/google/src/gemini-web-search-provider.runtime.ts#L64)

  - file: [extensions/zai/index.ts:237](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/extensions/zai/index.ts#L237)
