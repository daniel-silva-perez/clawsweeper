## Summary

Keep open. Current `main` has partial Slack Block Kit support: raw authored blocks can already be sent, and shared presentation renders a limited Slack Block Kit subset. But the broader umbrella request is not fully done on `main`: Slack markdown-table `block` mode is still explicitly downgraded back to `code`, and the shared presentation path is not arbitrary Block Kit JSON.
