## Summary

Current main already covers two parts of the request (`openclaw config validate` exists, and config writes preserve `${ENV}` references instead of persisting resolved secrets), but the remaining UX request is still unimplemented: bare `openclaw config` is intentionally wired to launch the interactive configure wizard rather than show help. That makes `implemented_on_main` too weak for a conservative close.
