## Summary

Keep open. Current main exposes only `agents.defaults.systemPromptOverride` / `agents.list[].systemPromptOverride`, which replaces the assembled system prompt. I found no additive `agents.defaults.systemPrompt` or `rules` config surface matching the request for config-injected rules layered after bootstrap files.
