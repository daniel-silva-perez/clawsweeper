## Summary

Keep open. Current main does not implement the requested plugin hook API: `before_model_call` and `after_model_call` are absent from the plugin hook name union, hook registry list, handler map, hook runner, and tests. Main does have internal diagnostic model-call events and an OpenTelemetry plugin path, but that is not the same public plugin hook surface requested by this PR. The PR discussion also shows useful review iteration and remaining maintainer-scope API placement questions, so this is not a conservative auto-close candidate.
