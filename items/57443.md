## Summary

Keep open. Current `main` does not implement the reported fix: tool schema cleaning is still limited to Gemini or explicit `model.compat.unsupportedToolSchemaKeywords`, and the bundled BytePlus/Volcengine providers do not set that compat, so `exec`-style schemas can still carry `patternProperties` to Ark-family `openai-completions` endpoints.
