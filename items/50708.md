## Summary

Keep open. Current `main` still has the reported user-visible behavior: the shared runtime formatter always includes `lastExitStatus` and `lastExitReason`, and the daemon status command prints that formatter output directly. The PR’s formatter-only approach also matches current code structure, because other diagnostics still consume the raw runtime exit fields.

## Evidence
