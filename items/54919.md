## Summary

Keep #54919 open. Current main appears to have implemented the `agents.defaults.thinkingDefault` part, but the requested global `agents.defaults.reasoningDefault` behavior is still missing from config schema/types and runtime resolution. Related PRs #39115 and #41809 track the remaining reasoning-default work, but that work is not merged into current main, so closing this issue would be premature.

## Best Possible Solution

Keep the issue open until `agents.defaults.reasoningDefault` is either intentionally rejected by maintainers or implemented consistently across config types, zod/generated schema, config docs/help, runtime precedence, session/status/UI surfaces as needed, and regression tests. The existing `thinkingDefault` support should be kept, and one of the related PRs can become the canonical implementation after rebasing and clarifying precedence against per-agent defaults.
