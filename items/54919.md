## Summary

Keep this issue open. Current main implements the global `agents.defaults.thinkingDefault` half, but the issue also asks for `agents.defaults.reasoningDefault`, and current main still supports reasoning defaults only per agent/session. The related open PRs are implementation candidates or partial channel fixes, not shipped main behavior.

## Best Possible Solution

Keep this issue open until maintainers either implement and document global `agents.defaults.reasoningDefault` across the config type, strict schema, generated schema/help, directive resolution, status/session surfaces, channel behavior where needed, and regression tests, or explicitly decide reasoning visibility should remain per-agent/session-only and remove misleading `agents.defaults.reasoningDefault` exposure from agent tool/config surfaces. Review #39115 as the likely implementation candidate; treat #71817 as a partial per-channel/per-agent fix only.
