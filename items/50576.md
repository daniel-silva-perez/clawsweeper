## Summary

Current main already implements the reported behavior: `/elevated full` forces `security=full` plus `ask=off`, skips the gateway approval path, and skips the YOLO script-preflight block. The changelog also records both the obfuscation-gate removal and the `security=full`/`ask=off` preflight fix, and the latest release tag contains those changes.
