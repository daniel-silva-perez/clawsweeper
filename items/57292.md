## Summary

Keep open. Current `main` still strips the Anthropic `context-1m` beta for OAuth-style `sk-ant-oat*` tokens, logs the warning the reporter quoted, and documents that fallback as expected behavior. There is no current override that would let Max-plan OAuth users force the 1M beta through.
