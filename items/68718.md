## Summary

Keep this PR open for maintainer-directed rework. Current main has the detached-task runtime seam and plugin registration contract that this work should build on, but it does not ship the durable SQLite queue/minions executor. The PR is recent, has active maintainer guidance, and still contains salvageable durability/recovery work, while the current diff is too large and risky to land as-is.

## Best Possible Solution

Do not land the current PR as-is and do not auto-close it yet. The best path is a smaller follow-up or reworked branch that keeps Tasks/TaskFlow as the canonical ledger, implements durable queue/recovery behavior as an optional detached runtime plugin using api.registerDetachedTaskRuntime, splits unrelated install-security-scan changes, and adds focused tests for ownership, recovery, cancel-tree, and migration behavior against the current seam.
