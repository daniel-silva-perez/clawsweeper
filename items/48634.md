## Summary

Current `main` already keeps the webchat agent/session picker backed by the global agent list instead of collapsing to the active session, and the same implementation is present in released tag `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
