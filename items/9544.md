## Summary

Keep open. Current `main` still uses Baileys `useMultiFileAuthState` and a disk-backed JSON auth directory for WhatsApp; recent work hardens `creds.json` writes and recovery, but it does not implement the requested move to a custom SQL-backed auth state.

## Evidence
