## Summary

Keep open. Current `main` does not implement Anthropic native web search as a `tools.web.search.provider` option. The hosted native-search path in code is OpenAI/Codex-only, and the bundled Anthropic plugin does not register any `webSearchProviders` contract.

## Evidence

- **Web-search docs only expose existing managed providers plus OpenAI/Codex native search:** The web-search docs list Brave, DuckDuckGo, Exa, Firecrawl, Gemini, Grok, Kimi, MiniMax, Ollama, Perplexity, SearXNG, and Tavily as providers, then document only `Native OpenAI web search` and `Native Codex web search`. There is no Anthropic native-search provider/config section.
  - file: [docs/tools/web.md:57](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/docs/tools/web.md#L57)
  - command: `nl -ba docs/tools/web.md | sed -n '40,170p'`
  - sha: [45146913007d](https://github.com/openclaw/openclaw/commit/45146913007da723b4f06aba6a66cf4a695974c0)
- **Config surface has native-search settings only for OpenAI Codex:** `tools.web.search` defines generic provider selection plus an `openaiCodex` nested config block. There is no parallel `anthropic` native-search config block for tool version, domains, user location, or max uses.
  - file: [src/config/types.tools.ts:492](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/src/config/types.tools.ts#L492)
  - command: `nl -ba src/config/types.tools.ts | sed -n '492,535p'`
  - sha: [45146913007d](https://github.com/openclaw/openclaw/commit/45146913007da723b4f06aba6a66cf4a695974c0)
- **Native hosted web_search activation code is Codex-specific:** The native-search eligibility check only returns true for `openai-codex` or `openai-codex-responses`, and the payload patcher builds/injects a `web_search` tool from that Codex-specific config. No Anthropic path exists here.
  - file: [src/agents/codex-native-web-search.ts:34](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/src/agents/codex-native-web-search.ts#L34)

A repo-wide manifest scan shows web-search-provider contracts on Brave, DuckDuckGo, Exa, Firecrawl, Google, MiniMax, Moonshot, Ollama, Perplexity, SearXNG, Tavily, and xAI, but not Anthropic.
