## Summary

Keep open. Current main still has the old hot-mode behavior: restart-required config changes in `gateway.reload.mode="hot"` are warned and skipped, and the proposed `fallbackToRestart` config key is not present in source, schema, tests, or docs. The PR is small, relevant gateway behavior work from a non-maintainer author, with no protected labels and no close reason applying.
