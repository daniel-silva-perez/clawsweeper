## Summary

Keep open. Current main does not implement the requested `before_prompt_build` or `before_agent_start` abort response contract. There is an adjacent `before_agent_reply` claiming hook, but current embedded-runner coverage explicitly limits it to cron triggers and asserts it is not invoked for normal user runs, so it does not satisfy the issue’s stated embedded-runner guardrail use case.
