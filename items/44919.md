## Summary

Keep open. Current `main` contains fresh mitigations for stale auth-store writes, subagent/main-agent OAuth mirroring, and failed subagent announce cleanup, but I did not find strong evidence that Anthropic Claude CLI credentials are now synchronized from `~/.claude/.credentials.json` into agent auth state. In fact, the current Anthropic CLI migration still writes access/refresh tokens into `auth-profiles.json`, while the external CLI bootstrap/sync path is only wired for MiniMax, not Claude CLI.

## Evidence

That is still a second persisted credential store, not a pure external-source read-through.
