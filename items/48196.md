## Summary

Keep open. Current `main` still injects the exact group-chat instruction to avoid the message tool while separately injecting a media hint that prefers the message tool, and there is still no channel-plugin seam to override that group-chat context. I found no evidence on `main` that the conflict described in #48196 has been reconciled.
