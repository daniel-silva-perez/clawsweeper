## Summary

Keep open. Current `main` does not show that arbitrary host `/tmp` paths are allowed for the sandboxed image tool, so this is not safely closable as `implemented_on_main` or `cannot_reproduce`. The current codebase instead allows only the secured preferred OpenClaw tmp root and explicitly rejects other host tmp paths, while preserving alias/symlink protections.
