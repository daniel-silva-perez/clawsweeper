## Summary

Keep open. Current `main` does not implement WebChat/Control UI file uploads for PDF/Office documents. The existing attachment path is image-only in the UI, the client request payload, and the gateway attachment parser, so #48700 is still an unmet feature request rather than something already fixed or safe to close.

## Evidence
