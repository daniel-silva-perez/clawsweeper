## Summary

Keep open. Current `main` already has the Slack thread mention toggle under `channels.slack.thread.requireExplicitMention`, but this PR still carries useful work that is not on `main`: `mergeSlackAccountConfig` still shallow-merges the nested `thread` object, so an account-level `thread` override can drop the global thread mention setting in multi-account Slack configs.
