## Summary

Keep open. Current `main` still awaits `before_agent_start` handlers sequentially with no timeout or watchdog, and the agent run paths await that promise directly. I found no shipped `plugins.hookTimeoutMs` config, diagnostics, or regression test covering a hung legacy hook.

## Evidence

- **Hook runner still blocks on awaited legacy hook:** `runModifyingHook` logs `running ... sequential` and then does a plain `await` on each handler inside the loop. There is no timeout wrapper, abort signal, or fallback path, so a never-resolving `before_agent_start` promise will stall the runner indefinitely.
  - file: [src/plugins/hooks.ts:347](https://github.com/openclaw/openclaw/blob/4013c658537ee2051c44186f197b0710e84c9049/src/plugins/hooks.ts#L347)
  - command: `nl -ba src/plugins/hooks.ts | sed -n '260,760p'`
  - sha: [4013c658537e](https://github.com/openclaw/openclaw/commit/4013c658537ee2051c44186f197b0710e84c9049)
- **Model-resolve path still waits on `runBeforeAgentStart`:** The agent setup path calls `await hookRunner.runBeforeAgentStart(...)` before continuing model/provider override resolution. The surrounding `catch` only handles rejected promises; it cannot recover from a promise that never settles.
  - file: [src/agents/pi-embedded-runner/run/setup.ts:74](https://github.com/openclaw/openclaw/blob/4013c658537ee2051c44186f197b0710e84c9049/src/agents/pi-embedded-runner/run/setup.ts#L74)
  - command: `nl -ba src/agents/pi-embedded-runner/run/setup.ts | sed -n '60,100p'`
  - sha: [4013c658537e](https://github.com/openclaw/openclaw/commit/4013c658537ee2051c44186f197b0710e84c9049)
- **Prompt-build path still waits on the same legacy hook:** Prompt preparation also awaits `runBeforeAgentStart(...)` when no precomputed result is present, again with only a rejection handler. A hung hook therefore still blocks prompt assembly and downstream execution.
  - file: [src/agents/pi-embedded-runner/run/attempt.prompt-helpers.ts:57](https://github.com/openclaw/openclaw/blob/4013c658537ee2051c44186f197b0710e84c9049/src/agents/pi-embedded-runner/run/attempt.prompt-helpers.ts#L57)
