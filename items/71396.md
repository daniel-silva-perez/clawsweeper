## Summary

Keep this PR open. Current main still forwards inbound Feishu mention targets into outbound reply dispatch, and those targets are still converted into real Feishu @mention markup on first plain-text/card chunks and streaming-card closes. The PR is recent, focused, and directly addresses linked bug #70065; it needs maintainer review, not conservative cleanup closure.

## Best Possible Solution

Keep this PR open for Feishu plugin maintainer review. The best path is to land or revise a focused Feishu plugin fix that stops unconditional automatic mention propagation for fallback, error, ack, plain-text, card, and streaming-card replies; keeps mention context bounded and sanitized for the agent; and either preserves a safe explicit-mention conversion path with tests or deliberately documents that agents must not rely on automatic mention forwarding.
