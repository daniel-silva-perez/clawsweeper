## Summary

Keep #70002 open. Current main still lacks the proposed upstream-repository guard in both docs workflows, and both jobs still use the upstream-only `OPENCLAW_DOCS_SYNC_TOKEN` path. The PR is a narrow two-line CI fix, not implemented on main and not superseded by the provided related items.

## Best Possible Solution

Keep the PR open for maintainer infra review. The best fix is to merge this exact job-level `github.repository == 'openclaw/openclaw'` guard or land an equivalent upstream-only guard in `.github/workflows/docs-sync-publish.yml` and `.github/workflows/docs-translate-trigger-release.yml`.
