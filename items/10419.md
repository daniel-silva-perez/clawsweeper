## Summary

Current `main` does not contain an OVHcloud/OVH AI Endpoints provider integration, docs page, or extension. OpenClaw does already support generic OpenAI-compatible custom providers, so OVHcloud may be manually configurable today, but that is not the same as the specific bundled-provider feature this issue requests. The request is coherent and still in scope for OpenClaw core/provider work, so it should stay open.
