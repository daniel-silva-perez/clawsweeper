## Summary

Keep open. Current main ships the env/file/exec SecretRef foundation and documents 1Password through an exec recipe, but native keyring/Keychain, native 1Password, and the plugin SecretRef provider seam are still not on main. Same-author PR #72548 is open as the concrete implementation candidate, so this issue should not be closed before that PR is reviewed, landed, or declined.

## Best Possible Solution

Review #72548 as the active implementation candidate. If it lands, close this as implemented or split any native 1Password follow-up that maintainers still want bundled. If it is declined, document the product decision and narrow this issue to the remaining accepted native provider or plugin API gap.
