## Summary

Keep open. Current `main` does not contain any Apertis provider implementation or Apertis docs, and this PR is still docs-only. The proposed docs/config examples also conflict with the current custom-provider schema and CLI, so there is no strong basis to close it as implemented, obsolete, or incoherent.

## Evidence
