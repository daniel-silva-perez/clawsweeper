## Summary

Current main already preserves provider-native dotted model IDs for copilot-proxy. Model-ref normalization only rewrites specific providers such as Anthropic, while fallback/runtime model resolution matches the configured or discovered exact model ID, so `copilot-proxy/claude-haiku-4.5` is not converted to `claude-haiku-4-5`. The shipped release `v2026.4.22` already contains this behavior.

## Evidence
