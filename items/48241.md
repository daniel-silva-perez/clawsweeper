## Summary

Keep open. Current `main` still preserves and broadcasts buffered assistant text when a run is aborted instead of discarding it, so there is not strong evidence that #48241 is already fixed on `main`. The issue’s linked fix also appears to still be an open PR rather than something merged and shipped.
