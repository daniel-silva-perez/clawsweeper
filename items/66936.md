## Summary

Keep open. Current main fixes the unresolved-SecretRef crash and the plain `agents list --json` plugin-preload path, but the whole report is not implemented: human `agents list` still loads plugins, `agents add` still inherits broad `agents` plugin loading, and CLI shutdown still only has memory-runtime cleanup rather than a generic external-plugin teardown path.

## Best Possible Solution

Keep the issue open but narrow it to the remaining CLI exit-hang problem. The best fix is to keep config-only `agents` paths, especially non-interactive `agents add` and human `agents list` where possible, off broad runtime plugin loading; if those paths genuinely need plugins, add a short-lived CLI plugin-runtime teardown contract that external plugins can implement. Treat the SecretRef fallback, bind/delete no-plugin overrides, and plain JSON list fast path as resolved subparts.
