## Summary

Current `main` no longer matches the reported obstructive top-banner behavior: WebChat now shows a small `New messages` pill above the composer and only when the user is scrolled away from the bottom. But the requested preference/config option to disable or customize that indicator does not exist in the current WebChat config surface, so the feature request itself is still unimplemented and should stay open.
