## Summary

Keep open. Current `main` has landed several parts of the Mattermost parity umbrella: threaded replies, reactions, typing indicators, interactive buttons, outbound media upload, and some markdown-formatting fixes. But the umbrella request is not fully implemented: I found no current Mattermost plugin/docs support for read receipts or presence, so the issue is still tracking real remaining parity work.
