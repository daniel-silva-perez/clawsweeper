## Summary

Keep open. Current `main` still surfaces generic context-overflow copy and does not include the requested session/model/token specifics. The code paths and regression tests both confirm the issue remains unimplemented on `754acc4478fe361db1e5303fd60f589d52beea7a`.

## Evidence

- **Embedded runner overflow payload is still generic:** The terminal overflow response is hard-coded to `Context overflow: prompt too large for the model. Try /reset (or /new) to start a fresh session, or use a larger-context model.` It records `sessionId` and model metadata in `meta`, but does not surface those specifics to the user-facing text.
  - file: [src/agents/pi-embedded-runner/run.ts:1216](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/agents/pi-embedded-runner/run.ts#L1216)
