## Summary

Keep this PR open. Current main still rejects empty `aws-sdk` auth in the media image and audio/video paths, while this PR is a focused implementation candidate for the Bedrock credential-chain regression tracked by #72031. The latest PR context also addresses the earlier review gap where empty keys were filtered before API-key rotation.

## Best Possible Solution

Keep this PR open for maintainer review and CI. The best path is to validate or iterate on the focused implementation that carries the `aws-sdk` missing-key allowance through media image, MiniMax fallback, and audio/video execution, including the empty-key rotation contract, then merge it to close #72031 if checks pass. PR #72066 is closed unmerged and should not supersede this work.
