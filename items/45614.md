## Summary

Current `main` does not implement a Serper web-search provider, so this is not `implemented_on_main`. There is a strong external-plugin path for web-search providers, but the evidence is not strong enough to close as `clawhub` because the issue was explicitly treated as a plausible core contribution and the thread references PR `#56511`, which could still represent useful follow-up work.
