## Summary

Keep open. Current `main` does not contain the pending-tool-call bridge described by PR #48512, so this is not implemented-on-main, and there is not enough evidence to close as obsolete or non-reproducible. The PR is also not safe to close as cleanup because its branch is massively polluted with unrelated changes, but the targeted runner bugfix still appears useful.
