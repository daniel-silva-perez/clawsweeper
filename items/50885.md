## Summary

Current `main` does not contain this PR's Windows UTF-8 cmd-wrapper changes. The live code still wraps `.cmd`/`.bat` commands with `cmd.exe /c ...` without the proposed `chcp 65001>nul &&` prefix, and `runExec` still does not expose the encoding override this PR adds. There is not strong evidence to close this PR as already implemented or obsolete.
