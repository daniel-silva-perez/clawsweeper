## Summary

Current `main` does not implement a `channels.feishu.tools.bitable` toggle. The Feishu config schema, runtime tool config type/defaults, and merged account tool resolver only cover `doc`, `chat`, `wiki`, `drive`, `perm`, and `scopes`, while `registerFull` still unconditionally invokes Bitable tool registration and `registerFeishuBitableTools` does not check any config flag.
