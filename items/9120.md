## Summary

Keep open. Current `main` has adjacent batching controls (`messages.inbound` debounce and `messages.queue.mode: "collect"`), but it does not implement the requested continuation-marker workflow, timeout, or continuation-specific typing behavior. This is still a distinct enhancement request.

## Evidence

- **Inbound batching exists, but only as time-window debounce:** Docs describe `messages.inbound` as batching rapid consecutive messages from the same sender into one turn, scoped by debounce timing rather than a semantic continuation marker.
  - file: [docs/concepts/messages.md:37](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/docs/concepts/messages.md#L37)
  - command: `nl -ba docs/concepts/messages.md | sed -n '24,70p'`
- **Queue collect only coalesces followups while a run is already active:** Queue docs say `collect` coalesces queued messages into a single followup turn; the tests show the first message still starts a run and only later same-destination messages are merged into a collected followup.
  - file: [src/auto-reply/reply/queue.collect.test.ts:57](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/auto-reply/reply/queue.collect.test.ts#L57)
  - command: `nl -ba src/auto-reply/reply/queue.collect.test.ts | sed -n '1,120p'`
- **Config schema has no continuation-marker fields:** `MessagesConfig` exposes `queue` and `inbound`, but there is no `continuationMarker`, `continuationTimeout`, or similar marker-driven buffering option in the typed config surface.
  - file: [src/config/types.messages.ts:13](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/src/config/types.messages.ts#L13)
  - command: `nl -ba src/config/types.messages.ts | sed -n '1,140p'`
- **Typing modes do not include continuation-specific suppression:** Typing indicator docs list only `never`, `instant`, `thinking`, and `message`; the issue's proposed `never-on-continuation` behavior is not present.
  - file: [docs/concepts/typing-indicators.md:24](https://github.com/openclaw/openclaw/blob/754acc4478fe361db1e5303fd60f589d52beea7a/docs/concepts/typing-indicators.md#L24)
  - command: `nl -ba docs/concepts/typing-indicators.md | sed -n '1,90p'`
- **Requested marker names are absent from code and docs:** Search for `continuationMarker`, `continuationTimeout`, `never-on-continuation`, and `(cont)` returned no matches under `src`, `docs`, `packages`, or `extensions`.
  - command: `rg -n "continuationMarker|continuationTimeout|never-on-continuation|\(cont\)" src docs packages extensions`
