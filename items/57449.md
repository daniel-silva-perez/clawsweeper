## Summary

Keep open. Current `main` does not implement this PR's CLI output changes: `openclaw devices list` still shows a single `Device` column for pending rows and no `Device ID`/`Client`/`OS`/`Last active` columns, and `devices remove` still does not point users back to the list output for the device id.
