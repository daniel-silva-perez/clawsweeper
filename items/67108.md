## Summary

Keep this PR open. Current main still lacks the proposed concurrent temp-dir repair tolerance, the linked race report remains open, and the patch is a narrow implementation candidate rather than obsolete cleanup.

## Best Possible Solution

Keep this PR open for maintainer review and CI. The best path is to land or refine a narrow fix in src/infra/tmp-openclaw-dir.ts with regression coverage in src/infra/tmp-openclaw-dir.test.ts, ideally confirming whether both chmod-throw and already-secure re-stat races from #66867 are covered.
