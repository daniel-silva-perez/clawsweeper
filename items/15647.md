## Summary

Current `main` does not implement Anthropic native `web_search` as a bundled OpenClaw web-search provider. The Anthropic plugin only declares the core model provider and media understanding, and current bundled web-search provider coverage/config tests omit any `anthropic` option.

## Evidence

- **Anthropic manifest has no web search contract:** `extensions/anthropic/openclaw.plugin.json` declares only `mediaUnderstandingProviders` under `contracts`; there is no `webSearchProviders` entry.
  - file: [extensions/anthropic/openclaw.plugin.json:51](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/extensions/anthropic/openclaw.plugin.json#L51)
  - command: `nl -ba extensions/anthropic/openclaw.plugin.json | sed -n '1,120p'`
  - sha: [a59d1bd46d9f](https://github.com/openclaw/openclaw/commit/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3)
- **Anthropic runtime registration does not register a web-search provider:** `registerAnthropicPlugin` registers the CLI backend, the Anthropic model provider, and media understanding only.
  - file: [extensions/anthropic/register.runtime.ts:615](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/extensions/anthropic/register.runtime.ts#L615)
  - command: `nl -ba extensions/anthropic/register.runtime.ts | sed -n '600,628p'`
  - sha: [a59d1bd46d9f](https://github.com/openclaw/openclaw/commit/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3)
- **Bundled web-search provider coverage omits Anthropic:** The current web-search provider config test enumerates bundled providers as `brave`, `firecrawl`, `gemini`, `grok`, `kimi`, `minimax`, `perplexity`, `searxng`, and `tavily`; there is no `anthropic` provider in that list.
  - file: [src/config/config.web-search-provider.test.ts:39](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/src/config/config.web-search-provider.test.ts#L39)
  - command: `nl -ba src/config/config.web-search-provider.test.ts | sed -n '30,120p'`
  - sha: [a59d1bd46d9f](https://github.com/openclaw/openclaw/commit/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3)
- **Web-search provider resolution only works from registered bundled providers:** `resolveWebSearchProviderId` resolves from `resolvePluginWebSearchProviders(... origin: "bundled")`, so Anthropic would need a registered bundled web-search provider to satisfy this issue.
  - file: [src/web-search/runtime.ts:115](https://github.com/openclaw/openclaw/blob/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3/src/web-search/runtime.ts#L115)
  - command: `nl -ba src/web-search/runtime.ts | sed -n '108,160p'`
  - sha: [a59d1bd46d9f](https://github.com/openclaw/openclaw/commit/a59d1bd46d9fc0bd2d9194b3f17db2561efc51e3)
