## Summary

Keep open. Current main does not expose a generic plugin-runtime outbound delivery API, so the PR is not implemented or obsolete. The proposed change still addresses a real gap for push/webhook-style plugins, but the branch appears stale and would need at least a loader reflection-key update before merge.
