## Summary

Keep open. Current main still does not implement half-block QR/block-art detection or styling for web chat/sidebar markdown. The PR appears to target a real UI gap, but it is unmerged and likely needs porting to the current markdown-it renderer before maintainers can land it.

## Best Possible Solution

Keep the PR/open item active, but port the idea to current `ui/src/ui/markdown.ts` markdown-it fence/code_block rendering and the raw chat/sidebar paths. Add a small exported detector, preserve QR whitespace when auto-fencing tool output, add `.half-block-art` CSS for chat and sidebar including inner `pre code` inheritance and scroll behavior, and add targeted tests around detector thresholds plus rendered class application.
