## Summary

Keep open. Current `main` has a bundled Google speech provider, but it is a Gemini TTS implementation, not the requested Google Cloud Text-to-Speech integration with WaveNet/Polyglot and Telegram-friendly OGG_OPUS output. The docs explicitly say the shipped Google provider is not the separate Cloud Text-to-Speech API path, and the provider currently returns WAV/PCM rather than native Opus voice-note output.
