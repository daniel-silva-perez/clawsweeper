## Summary

Keep open. Current `main` still has the gateway SSE contract bug this PR fixes: the tool-call terminal path builds a response with `status: "incomplete"` but emits `response.completed`, and the matching schema/docs/test coverage for `response.incomplete` are still absent.

## Evidence
