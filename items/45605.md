## Summary

Keep open. Current `main` does not implement this PR's approach: `/status` is still intentionally kept on the agent path, and flipping it to `executeLocal: true` would break the command because the local slash-command executor has no `status` handler. The PR also includes an unrelated README bot-artifact change, so there is no strong close reason from the allowed set.
