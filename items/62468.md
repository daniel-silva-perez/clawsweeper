## Summary

Keep open. Current main has partial Feishu target resolution through the shared directory fallback, but it still treats only prefixed/`ou_`/`oc_` values as Feishu IDs and the live Feishu lookup matches `open_id` plus display `name`, not a separate raw username/`user_id` field. That leaves the reported `Unknown target "jaydenli"` path plausible on current main rather than strongly obsolete.
