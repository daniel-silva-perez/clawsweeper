## Summary

Current main partially satisfies the request with per-agent `thinkingDefault` and flat `agents.list[].params`, but the requested per-agent per-model `agents.list[].models` map and adaptive `thinkingStrategy` rules/escalation/AGENTS.md policy surface are still absent. The adjacent #32496 and #44126 items do not supersede the remaining work.

## Best Possible Solution

Keep this issue open as the canonical product/API discussion, or split it into narrower follow-ups. Preserve the shipped `agents.list[].thinkingDefault` and flat `agents.list[].params` behavior, then decide whether core should add `agents.list[].models` for per-agent per-model params and a `thinkingStrategy` API for message rules, failure escalation, and AGENTS.md-sourced policy. Coordinate broad provider parameter forwarding with #32496 and adaptive thinking controls with #44126.
