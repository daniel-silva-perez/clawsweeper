## Summary

Keep open. Current main still only covers the narrow all-media-failed case with a generic warning; it does not implement this PR's dropped-media contract, per-item sanitized notices, mixed-media preservation, catastrophic-normalization feedback, outbound JSON/delivery propagation, or transcript mirror behavior. The related #70109 candidate is closed and unmerged, so it does not supersede this active PR.

## Best Possible Solution

Keep this PR open. The best path is a maintainer-reviewed, possibly scoped implementation that preserves sanitized per-item media drop details through reply media normalization, catastrophic fallback, route/followup filtering, outbound JSON/delivery, and transcript mirroring, while fixing the current code/reason drift and retaining the privacy and delivery-safety fixes already discussed in review.
