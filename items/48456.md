## Summary

Current `main` partially addresses the request by adding `agents.defaults.skills` inheritance, but it does not implement the issue’s proposed merge semantics. On `main`, an explicit `agents.list[].skills` list replaces inherited defaults instead of union-merging with them, so the core ask of “shared defaults + agent-specific additions” is still not implemented.

## Evidence
