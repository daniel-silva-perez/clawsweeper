## Summary

Current `main` does not have the requested declarative `agents.defaults.bootstrapFiles` / per-agent `bootstrapFiles` config. There is a supported hook-based extension path for extra bootstrap context, including arbitrary filenames via `agent:bootstrap` hooks, but that is a workaround/alternative design rather than the config key this issue asks for.

## Evidence

- **No bootstrapFiles config key in agent defaults schema:** `src/config/zod-schema.agent-defaults.ts` defines `skipBootstrap`, `contextInjection`, `bootstrapMaxChars`, `bootstrapTotalMaxChars`, and `bootstrapPromptTruncationWarning`, but no `bootstrapFiles` field.
  - file: [src/config/zod-schema.agent-defaults.ts:85](https://github.com/openclaw/openclaw/blob/ebd7f19a3b20001960474ec9ba9cc424e45f7b23/src/config/zod-schema.agent-defaults.ts#L85)

  - file: [docs/gateway/config-agents.md:60](https://github.com/openclaw/openclaw/blob/ebd7f19a3b20001960474ec9ba9cc424e45f7b23/docs/gateway/config-agents.md#L60)
