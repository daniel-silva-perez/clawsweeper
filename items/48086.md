## Summary

Current `main` already covers the reported Gemini 3.x 404 paths: it rewrites the obsolete Flash 3.1 preview ID to the working `gemini-3-flash-preview`, resolves `google/gemini-3.1-pro-preview` through the Google provider forward-compat path, and normalizes bare Google API roots to `/v1beta`. Those fixes are present in the latest shipped release `v2026.4.22`.
