## Summary

Keep open. Current `main` does not implement this PR's feature set: compaction mode is still limited to `default | safeguard`, the embedded runner only resolves those two modes, Pi auto-compaction is only disabled for engine-owned compaction, and the current pre-prompt overflow path still routes into compaction/truncation rather than `warn`/`error`/`none` behavior.
