## Summary

Keep open. Current main still hardcodes the bundled memory-core flush output to `memory/YYYY-MM-DD.md`, and the user config surface for `agents.defaults.compaction.memoryFlush` has no `pathTemplate` or equivalent field. There is a plugin-level `flushPlanResolver` seam for custom memory plugins, but that is not the requested built-in configurable write path.
