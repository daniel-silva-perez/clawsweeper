## Summary

Keep open. Current `main` has internal support for auto-expanding tool output, but the Control UI never exposes a user setting and the chat render path hard-codes that feature off.

## Evidence

- **Chat render hard-codes auto-expand off:** `renderChat(...)` passes `autoExpandToolCalls: false`, so webchat never enables default-expanded tool output on current `main`.
