## Summary

Keep open. Current `main` has added meaningful memory infrastructure since the report was filed, including automated dreaming/promotion, opt-in session transcript search, pluggable context engines, and alternate memory backends. But the issue’s two concrete core complaints are still unresolved on `main`: `MEMORY.md` is still part of normal bootstrap injection, and `memoryFlush` is still a pre-compaction mechanism rather than a shutdown/restart safeguard.
