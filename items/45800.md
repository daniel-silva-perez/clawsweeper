## Summary

Keep open. Current `main` still renders a `+` new-session button in the chat input toolbar, wires it directly to `/new`, and does not add any confirmation step. The latest release tag shown in this checkout (`v2026.4.22`) has the same behavior, so there is no strong evidence the request is already implemented or obsolete.
