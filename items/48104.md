## Summary

Keep open. Current main has stronger refusal/error classification and new diagnostic-event plumbing, but I found no implementation that bridges operator authorization/trusted-admin state into the model execution path for authorized infra tasks like SSH diagnostics. The current prompting remains generic, and the docs only distinguish pairing/policy/approval failures plus generic refusals.
