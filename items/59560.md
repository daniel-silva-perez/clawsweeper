## Summary

Keep open. This is a recent, coherent WhatsApp safety feature request, not stale, not maintainer-authored, and not protected-label gated. Current public WhatsApp docs describe only global/account-level DM and group controls, so the requested per-contact inbound/outbound/autonomous policy still appears to be a plausible unimplemented core WhatsApp plugin enhancement.
