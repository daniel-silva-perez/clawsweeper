## Summary

Keep open. Current `main` still logs duplicate-plugin warnings for bundled+installed Feishu configs and does not provide a doctor/self-repair path for that specific state. There is partial mitigation: newer onboarding prefers the bundled Feishu source and docs now describe Feishu as bundled, so fresh setups are less likely to create the problem.
