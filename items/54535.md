## Summary

Keep open. Current `main` still sends `openclaw browser navigate` through the shared `browser.request` -> `/navigate` path, and managed profiles still execute Playwright `navigateViaPlaywright` backed by `chromium.connectOverCDP`. I found no strong evidence on `main` that this Chrome 146 crash/regression was fixed, so there is not enough proof to close conservatively.
