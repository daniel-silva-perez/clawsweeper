## Summary

Keep open. Current `main` still does not implement a global cron-scoped tool deny config equivalent to `tools.cron.tools.deny`. The repo only exposes `tools.subagents.tools.*` for subagents, plus cron-specific per-job `toolsAllow` and some delivery-time message-tool gating.

- Closing as implemented would overstate current behavior and remove a still-missing configuration surface described in the issue.
