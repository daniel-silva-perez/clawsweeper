## Summary

Keep open. Current `main` has several related mitigations for context overflow and compaction, but I did not find strong evidence that issue #50065 is fully resolved for direct sessions. The code now does pre-prompt overflow checks and provider-aware context-window lookup, yet the documented Pi threshold path still compacts after a successful turn, and the issue has corroborating field reports from April 2026 after earlier related fixes shipped.
