## Summary

Keep open. Current `main` still documents the Android app as not publicly released, only documents self-build flows, and the public repo workflows/build scripts stop at local debug builds or signed local `.aab` generation. I found no evidence in the checked-out repo that official prebuilt Android APKs are published to GitHub releases.
