## Summary

Keep open. Current `main` still drops deleted agents out of Control UI usage even when `deleteFiles=false`: `agents.delete` always purges that agent's session-store entries before the `deleteFiles` branch, and `sessions.usage` only discovers transcripts for agents still present in config. That leaves transcript files on disk but removes them from the Usage dashboard's session/aggregate source.

## Evidence

  - file: [src/commands/agents.command-shared.ts:25](https://github.com/openclaw/openclaw/blob/ebd7f19a3b20001960474ec9ba9cc424e45f7b23/src/commands/agents.command-shared.ts#L25)
