## Summary

Keep open. Current `main` has a partial earlier mitigation for outbound replay safety, but the specific changes in PR #50850 are not implemented on `main`: `deliver.ts` still silently swallows `ackDelivery` failures, `delivery-queue-recovery.ts` still has no TTL/max-age expiry path, and queued delivery storage still does not persist the proposed `sessionId` field. That is not strong enough evidence to close this PR as already implemented or obsolete.
