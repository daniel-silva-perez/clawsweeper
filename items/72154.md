## Summary

Keep PR #72154 open. Current main still contains the broken Docker VM runtime URLs for `gog` and `goplaces`, and the PR is a focused docs-only fix with verified replacement asset URLs. Do not close it as superseded by #71796: that older overlapping PR also changes `AGENTS.md` to recommend `openclaw gateway restart --deep`, but current CLI/docs show `restart` only accepts `--json`. The best path is maintainer review or revision, likely folding in selective tar extraction/cleanup and deciding whether to also remove or correct the stale `wacli` Linux-container example.

## Best Possible Solution

Keep #72154 open for maintainer review. The best product/docs outcome is a focused Docker VM runtime docs fix that uses the working `gogcli` and `goplaces` release assets, extracts only the needed binaries or cleans `/tmp`, and either corrects or removes the stale Linux `wacli` example. If maintainers prefer #71796's broader Docker treatment, split or revise that PR's Docker-only portion rather than closing this narrower fix as obsolete.
