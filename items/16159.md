## Summary

Keep open. Current `main` does not implement coordinate-based browser clicks: the public act types/schema still expose element-targeted clicks only, request normalization still requires `ref` or `selector`, and both execution paths route clicks through element/UID-based helpers rather than x/y coordinates.

## Evidence
