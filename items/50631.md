## Summary

Keep open. Current `main` still allows the reported `tailscale.mode="serve"` + `gateway.auth.mode="none"` configuration, and the runtime auth path still accepts `auth.mode="none"` connections without credentials. I did not find evidence of the PR fix on `main` at `ebd7f19a3b20001960474ec9ba9cc424e45f7b23`.
