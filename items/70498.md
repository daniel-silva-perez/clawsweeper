## Summary

Keep open. Current main still does not document `ZALOUSER_PROFILE` or `ZCA_PROFILE` in the Zalo Personal channel docs, while the bundled plugin manifest, runtime account resolver, and tests confirm those env vars are real behavior. The prior PR #69643 is closed unmerged, and closed issue #65864 tracks the same docs gap but does not supersede this still-useful PR.

## Best Possible Solution

Keep PR #70498 open and either merge it or adapt the same docs section into `docs/channels/zalouser.md`, preserving the source-backed resolution order and multi-account caveat. No core runtime or plugin API change is needed.
