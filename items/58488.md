## Summary

Keep #58488 open. Current main has added ACP runtime model controls and an `env` field in the type surface, but the OpenClaw manager still does not pass model or env into `ensureSession`, the bundled acpx plugin has no per-agent env config, and pinned `acpx@0.6.0` still sends initial session model options only under `_meta.claudeCode.options.model`. The issue’s linked Kilo-side request remains relevant rather than obsolete.
