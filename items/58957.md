## Summary

Keep open. Current main has some context-overflow fallback handling, but the issue is recent and the discussion identifies still-plausible compaction dead zones. Source on main still skips preflight compaction for heartbeats and for sessions with fresh persisted token totals, so this is not high-confidence implemented-on-main cleanup.
