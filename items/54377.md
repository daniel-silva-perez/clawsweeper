## Summary

Keep open. Current `main` does not implement an explicit `npm:` override for `openclaw plugins install`; the CLI/docs still describe bare specs as `ClawHub`-first with npm fallback, and the npm-spec validator rejects `npm:<pkg>` as a protocol spec.
