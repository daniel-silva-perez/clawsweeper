## Summary

Keep open. Current `main` still contains the broken multi-line script placeholders in the export template, while the export generator still does exact literal `.replace("{{...}}", ...)` calls, so the reported HTML export bug remains plausible and is not implemented on `main` or in the latest release commit provided.
