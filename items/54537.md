## Summary

Keep open. Current `main` still unconditionally blocks sandbox Docker `network: "host"`, only exposes the existing `dangerouslyAllowContainerNamespaceJoin` escape hatch, and documents/tests that host networking remains blocked. This feature request is coherent and not implemented on `main`.
