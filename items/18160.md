## Summary

Keep open. Current `main` does not implement a cron `payload.kind: "exec"` path: the public cron schema only accepts `systemEvent` and `agentTurn`, cron tool/docs still describe isolated jobs as agent-turn based, job validation rejects non-`agentTurn` isolated/current/session jobs, and runtime execution routes detached jobs through `runIsolatedAgentJob` after requiring `payload.kind === "agentTurn"`. The provided GitHub context also shows the related implementation PR #51276 is still open, not landed.
