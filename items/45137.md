## Summary

Current `main` does not implement a Slack `mux` mode or mux receiver. The Slack config/types/docs still only support `socket` and `http`, and the runtime branches only between socket-mode and HTTP webhook handling, so issue #45137 remains an open feature request rather than cleanup-close material.
