## Summary

Keep open. Current `main` still unconditionally zeroes every assistant `usage` snapshot when compaction ends without retry, so the reported post-compaction context-token regression is not implemented away. The replay-history path already contains the narrower "only pre-compaction assistant usage is stale" logic, which makes the live compaction handler discrepancy concrete.
