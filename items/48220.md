## Summary

Current `main` does not provide strong evidence that #48220 is fixed or obsolete. The reported dashboard/CLI mismatch is still plausible because the dashboard context warning is rendered from gateway session rows, while CLI session output is formatted through a separate command path, and device-pair listing still applies caller-based filtering that can hide pending entries.
