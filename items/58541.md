## Summary

Keep open. Current main has partial mitigation for this exact Google Chat 401 path: it logs auth rejection reasons and warns when appPrincipal is missing or looks like an email. The issue discussion also confirms the reporter fixed their setup by using the numeric JWT sub value. However, the current Google Chat docs still do not document appPrincipal in the setup or config example, so the remaining user-facing docs request is still valid.
