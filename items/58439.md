## Summary

Keep PR #58439 open. Current main still does not forward `mediaLocalRoots` from the Mattermost message-tool `handleAction` send path into `sendMessageMattermost`, while the downstream Mattermost send API and the channel action context both support it. The PR remains a plausible, targeted fix; the cross-referenced PR #67796 is also open and does not establish that main already fixed this item.
