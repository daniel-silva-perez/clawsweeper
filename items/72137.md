## Summary

Keep PR #72137 open. Current main still lacks the missing-vs-empty-plain-object collapse in `diffConfigPaths`, so a normalized `plugins.entries.<id>.config: {}` branch can still be reported as a real config diff and then classified as a gateway restart. The PR remains a focused candidate fix for the #72061 skill-toggle restart bug, with the prior empty-array concern addressed in the provided latest diff.

## Best Possible Solution

Keep PR #72137 open for normal maintainer review and landing, or land an equivalent narrow fix. The best implementation is the current focused comparator change: collapse only `undefined` vs empty plain-object branches as no-op, preserve explicit empty-array diffs, and keep the regression tests in `src/gateway/config-reload.test.ts` for the full #72061 skill-toggle normalization scenario.
