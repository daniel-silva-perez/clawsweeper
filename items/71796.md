## Summary

Keep open. Current main still contains the broken Docker VM runtime URLs and the stale AGENTS gateway/Node wording, but PR #71796 is not merge-ready as-is because its AGENTS gateway replacement tells Mac users to append `--deep` to `openclaw gateway restart`, while the current CLI and docs show `--deep` is only available for `gateway status`. Related PR #72154 supersedes part of the Docker URL fix but does not cover the `wacli` Linux-asset issue that #71796 raises.

## Best Possible Solution

Keep the item open for maintainer/product cleanup rather than auto-closing. The best path is to either revise #71796 to remove the invalid `restart --deep` guidance, add the claimed Node 22.14+/Node 24 AGENTS clarification, and coordinate with #72154 for the Docker URL fix, or close one PR manually after folding the useful `wacli` Linux-asset clarification into the canonical Docker-doc PR.
