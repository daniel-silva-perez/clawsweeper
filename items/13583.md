## Summary

Keep open. Current main has plugin hooks for pre-LLM interception, tool-call blocking, outbound cancellation, and transcript-write blocking, but it does not implement the requested first-class pre-response enforcement layer or two-phase validator protocol. Closing as `clawhub` would be too speculative because the current public hook surface still lacks a built-in plan/validate/respond contract.
