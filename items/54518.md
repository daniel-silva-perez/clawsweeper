## Summary

Keep open. Current `main` at `8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf` still does not enforce per-agent workspace isolation by default: the docs explicitly say the workspace is only the default cwd, `tools.fs.workspaceOnly` remains opt-in, and the host read/write/edit implementations still allow paths outside the workspace when that flag is unset or false.

## Evidence
