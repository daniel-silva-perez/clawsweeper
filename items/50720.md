## Summary

Keep open. Current `main` does not implement `sessions_send` private delivery: the tool schema still lacks `deliveryMode`/`announce`, delivery metadata is still hard-coded to `mode: "announce"`, and the A2A flow still always proceeds to the external announce/send path.

## Evidence

- **Schema lacks requested parameters:** `sessions_send` on current `main` only exposes `sessionKey`, `label`, `agentId`, `message`, and `timeoutSeconds`; there is no `deliveryMode` or legacy `announce` parameter in the tool schema.
