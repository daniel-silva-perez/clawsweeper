## Summary

Keep open. Current main has partial UX improvements for configuration, including Quick Settings, schema-backed config editing, and ClawHub skill search/install in the Skills panel, but it still does not implement the broader WordPress-like guided dashboard and secure dashboard-based ClawHub plugin install/configuration experience requested here. The existing ClawSweeper comment reached the same conclusion, and current main still supports that conclusion.

## Best Possible Solution

Keep this open as a product/UI umbrella until maintainers either accept it as a roadmap item or split it into concrete issues. The likely implementation path is to extend Control UI with guided reconfiguration flows, secure dashboard plugin discovery/install/configuration backed by existing ClawHub/plugin install contracts, contextual help for complex config sections, and any generic plugin manifest/config-schema API needed for third-party plugins to expose safe dashboard configuration.
