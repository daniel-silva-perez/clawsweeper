## Summary

Keep open. Current main has meaningful Control UI configuration improvements, including Quick Settings, advanced schema-backed config editing, and ClawHub skill search/install in the Skills panel, but it does not implement the broader WordPress-like dashboard/plugin configuration experience requested here. The request is recent, labeled only as an enhancement, and neither provided related item is a canonical duplicate or completed implementation of this broader product ask.

## Best Possible Solution

Keep this open as a product/UI umbrella or have maintainers split it into concrete issues: guided dashboard reconfiguration without rerunning onboarding, a secure ClawHub plugin discovery/install/configuration flow in Control UI, contextual help for complex config sections, and any missing plugin manifest/schema API needed for third-party plugins to expose safe dashboard configuration.
