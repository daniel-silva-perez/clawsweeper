## Summary

Current `main` has a partial Control UI implementation for context awareness, but it does not fully implement issue #50071's request for an always-visible context-usage bar above the chat input. The shipped UI is a high-usage warning pill plus per-message `ctx` metadata, not a continuous progress bar.

## Evidence
