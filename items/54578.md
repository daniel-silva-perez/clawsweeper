## Summary

Current main does not implement an `openclaw tree` command or any equivalent hierarchical active-agent view. The shipped surfaces are flat session/task listings plus slash-command views like `/subagents list` and `/agents`, so the request remains open and still fits core CLI/web ergonomics rather than ClawHub.
