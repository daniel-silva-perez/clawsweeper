## Summary

Keep open. Current main still sends Feishu post markdown text unchanged, and I found no merged helper or regression coverage for preserving bare URLs with underscores. The PR is useful and plugin-local, but it still has unresolved review feedback about mutating existing Markdown links with nested brackets, so it is not a conservative cleanup close.

## Best Possible Solution

Keep this PR open for maintainer review. The likely product path is still a plugin-local fix in `extensions/feishu/src/send.ts`, backed by focused tests for bare URL underscores and for preserving existing Markdown links, image syntax, angle autolinks, inline code, and fenced code. Before landing, resolve the nested-bracket Markdown-link review or choose #41880 as the canonical narrower fix and close the other PR explicitly.
