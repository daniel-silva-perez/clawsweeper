## Summary

Keep open. Current main has partial related state, but it does not implement the requested per-session working-context capsule or expose working context in `/status`. The provided GitHub context also shows an open draft implementation candidate, #71135, which is explicitly scoped to a storage primitive with no runtime wiring yet.

## Best Possible Solution

Keep this issue open as the product/API tracker. Review #71135 as a possible first storage primitive, then land follow-up integration that updates the capsule from run/exec/subagent/ACP paths, exposes cwd/repo/branch/sandbox/temp-clone state in `/status`, and updates the workspace and compaction docs to distinguish semantic continuity from working-state continuity.
