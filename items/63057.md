## Summary

Keep PR #63057 open. Current main at b565e6e still contains the exact `options.thinkingBudgetTokens || 1024` fallback the PR targets, and the zero-budget path remains reachable through `adjustMaxTokensForThinking`; the PR is not implemented or superseded. It does need the regression tests already requested in review.

## Best Possible Solution

Keep the PR open and ask for, or add, focused regression tests in `src/agents/anthropic-transport-stream.test.ts`: one proving `thinkingBudgetTokens: 0` omits the legacy thinking block, and one preserving the intentional positive/default budget behavior. Then land this PR or an equivalent patch that replaces `|| 1024` with a nullish default plus a `budgetTokens > 0` guard in `buildAnthropicParams`.
