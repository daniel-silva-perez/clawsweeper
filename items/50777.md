## Summary

Current `main` still has the reported UI behavior path: the code-block copy button renders both `Copy` and `Copied!` spans in the HTML and relies on CSS alone to hide the `Copied!` span, so this PR is not implemented on `main` and should stay open for maintainer judgment.
