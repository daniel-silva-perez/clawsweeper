## Summary

Keep open. The issue is recent, has concrete runtime evidence, and includes a follow-up report after 2026.4.9. I did not find high-confidence evidence that current main or v2026.4.23 fixed this specific Linux long-runtime heartbeat stall, and the prior #52270 fix is explicitly the thing being reported as insufficient.
