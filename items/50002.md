## Summary

Keep open. Current `main` still does not implement a non-interactive Copilot onboarding token flag or any Copilot `runNonInteractive` auth path. The older 'silent fallthrough' part appears fixed: non-interactive `--auth-choice github-copilot` now errors explicitly instead of doing nothing.

## Evidence

- **Copilot manifest exposes only interactive device auth metadata:** `extensions/github-copilot/openclaw.plugin.json:11` defines the `github-copilot` choice, but there is no `optionKey`, `cliFlag`, or `cliOption`, so it cannot contribute a `--github-copilot-token`-style onboarding flag.
  - file: [extensions/github-copilot/openclaw.plugin.json:11](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/extensions/github-copilot/openclaw.plugin.json#L11)
  - command: `nl -ba extensions/github-copilot/openclaw.plugin.json | sed -n '1,120p'`
  - sha: [45146913007d](https://github.com/openclaw/openclaw/commit/45146913007da723b4f06aba6a66cf4a695974c0)
- **Copilot provider implements only interactive auth:** `extensions/github-copilot/index.ts:41` runs device login, checks `process.stdin.isTTY`, and returns early when no TTY is present. The registered auth method at `extensions/github-copilot/index.ts:95` provides `run`, but no non-interactive companion.
  - file: [extensions/github-copilot/index.ts:41](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/extensions/github-copilot/index.ts#L41)
  - command: `nl -ba extensions/github-copilot/index.ts | sed -n '35,120p'`
  - sha: [45146913007d](https://github.com/openclaw/openclaw/commit/45146913007da723b4f06aba6a66cf4a695974c0)
- **Onboard CLI only registers plugin-owned flags when manifests declare them:** `src/cli/program/register.onboard.ts:50` builds `ONBOARD_AUTH_FLAGS` from `resolveManifestProviderOnboardAuthFlags()`, and `src/cli/program/register.onboard.ts:104` adds only those declared flags. With no Copilot manifest flag metadata, current `main` cannot expose `--github-copilot-token`.
  - file: [src/cli/program/register.onboard.ts:50](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/src/cli/program/register.onboard.ts#L50)

That means the report's silent no-op behavior is obsolete, but the missing non-interactive Copilot path is not fixed.
  - file: [src/commands/onboard-non-interactive/local/auth-choice.plugin-providers.ts:131](https://github.com/openclaw/openclaw/blob/45146913007da723b4f06aba6a66cf4a695974c0/src/commands/onboard-non-interactive/local/auth-choice.plugin-providers.ts#L131)
