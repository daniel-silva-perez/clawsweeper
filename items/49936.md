## Summary

Keep open. Current `main` does not contain the two regression cases from PR #49936, and the closest existing tests are narrower. The PR still adds distinct value by covering webchat-visible internal agent turns after an `agent` RPC with `deliver: false`, including the resumed main-session path and the direct webchat-session path.
