## Summary

Current `main` already covers the requested behavior well enough: bare `/new` and `/reset` prepend runtime-selected recent daily memory into the first prompt, the workspace template tells agents to use that startup context first and delete `BOOTSTRAP.md` after first-run, and memory recall guidance covers `MEMORY.md`, daily notes, and indexed session transcripts.
