## Summary

Keep open. Current main still serves a Control UI CSP whose `connect-src` allows only same-origin and websocket connections, while the shipped custom-theme importer fetches tweakcn registry payloads directly from the browser. The PR is a narrow fix to a real current-main product bug, and the broader configurable-CSP issue does not supersede this specific regression fix.

## Best Possible Solution

Keep the PR open and land the narrow CSP allowlist plus regression coverage after normal maintainer/CI review. Treat broader configurable CSP work, such as #47565, as a separate product decision rather than a reason to close this focused bug fix.
