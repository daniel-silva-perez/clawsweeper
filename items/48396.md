## Summary

Keep open. Current `main` does not implement the Control UI session-list message preview, and the submitted PR still appears incomplete relative to current contracts: the web UI row type does not expose `lastMessagePreview`, the sessions controller does not request `includeLastMessage`, and the sessions table still renders only the timestamp in the Updated column.
