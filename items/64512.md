Keep this PR open. Current main still lacks Bedrock service-tier forwarding, while this PR remains the canonical implementation candidate for the already-closed feature request #63415. The open PR discussion shows active review and author follow-up, not an obsolete or duplicate change.

## Best Possible Solution

Keep this PR open as the canonical implementation candidate for #63415. Review or revise it into a focused Amazon Bedrock plugin change that resolves `serviceTier` and `service_tier`, injects top-level `{ serviceTier: { type } }` only for `model.api === "bedrock-converse-stream"`, preserves caller-provided payload values, warns on invalid values, adds focused regression coverage, updates the Bedrock provider docs, and refreshes Plugin SDK API artifacts if helper exports remain public.
