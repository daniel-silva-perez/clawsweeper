## Summary

Keep this PR open. Current main still does not implement Bedrock `service_tier` forwarding: the Amazon Bedrock plugin does not read `extraParams`, the shared Bedrock stream-wrapper module only exposes cache behavior, the SDK barrel does not export Bedrock service-tier helpers, and the Bedrock provider docs do not document the feature. The linked issue #63415 was already closed in favor of this PR, so the best maintainer path is to continue review here rather than close the implementation candidate.

## Best Possible Solution

Keep this PR open as the canonical implementation candidate for #63415. Review or revise it into a focused Bedrock plugin change that resolves `serviceTier`/`service_tier`, injects top-level `{ serviceTier: { type } }` only for `model.api === "bedrock-converse-stream"`, preserves caller payload values, warns on invalid values, adds focused tests, updates the Bedrock docs, and refreshes any required Plugin SDK API baseline if the helper exports remain public.
