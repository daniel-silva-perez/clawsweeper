## Summary

Current `main` does not already implement this PR’s stated behavior. The normal reply path and agent command path still persist unsupported thinking levels back into session state as a downgraded fallback, and the directive/model-switch path still remaps unsupported persisted thinking levels and acknowledges that remap. That means the underlying `xhigh`-preservation request is still open rather than implemented on `main`.
