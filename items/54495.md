## Summary

Keep open. Current `main` does not implement configurable image-generation timeouts via `AgentModelConfig`; it only supports per-call `timeoutMs` passed through the `image_generate` tool. The PR also appears to need follow-up because the tool config normalization path only preserves `primary` and `fallbacks`, so a config-level `timeoutMs` would be dropped there.
