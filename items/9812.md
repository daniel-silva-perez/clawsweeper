## Summary

Keep open. Current `main` does not implement agent-level `rateLimits` config or any matching runtime enforcement path. The per-agent config surface includes overrides like `model`, `params`, `heartbeat`, `sandbox`, and `tools`, but no `rateLimits`, and the agent config resolver does not read such a field.
