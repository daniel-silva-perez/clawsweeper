## Summary

Keep open. Current main does not implement `background: false` as a synchronous opt-out: `params.background === false` still follows the default timed backgrounding path. The PR is a plausible small feature with useful motivation and test coverage, and the provided review discussion indicates the earlier warning concern was addressed on the PR branch.
