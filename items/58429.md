## Summary

Keep open. Current main has related cleanup primitives, but it does not implement the requested default/configurable retention for completed run-mode subagent session-store entries. The default subagent spawn path still keeps completed run sessions unless the caller explicitly passes cleanup="delete", and archiveAfterMinutes only applies to delete-mode run records.
