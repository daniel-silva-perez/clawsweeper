## Summary

Current `main` has partial session-management support, but it does not implement the requested project-based sidebar shortcuts or project-scoped auto-loading behavior. The dashboard sidebar is still a static tab list, chat session switching lives in the chat header select, and workspace docs still describe `MEMORY.md` as loading only for the main private session rather than arbitrary named project sessions.
