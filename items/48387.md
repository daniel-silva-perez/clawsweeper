## Summary

Current `main` does not implement this change. Explicit WhatsApp outbound targets are still gated by `allowFrom`, the direct outbound path still calls the resolver with `mode: "explicit"`, and current tests do not cover the proposed bypass. This PR remains actionable and should stay open for maintainer review rather than being closed as already fixed or obsolete.
