## Summary

Keep open. Current main does not implement `openclaw backup restore`, and the PR is a coherent core-CLI feature request rather than an auto-close candidate. The posted OpenClaw Snapshot skill is a useful workaround, but core docs already present `openclaw backup create` as producing restorable snapshots before reset/uninstall, so this needs maintainer product/safety review rather than a `clawhub` close.
