## Summary

Current `main` does not implement the requested tool capability descriptors. Tool metadata in core is still limited to `ownerOnly` and `displaySummary`, plugin-side tool metadata only tracks `pluginId` and `optional`, and `before_tool_call` still exposes only `toolName` plus `params`, so policy plugins still need name-based matching.

## Evidence
