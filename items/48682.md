## Summary

Keep open. Current `main` still ignores `auth.order` for implicit GitHub Copilot discovery, although the implementation has moved since this PR was opened. The underlying bug remains, but the PR now targets an outdated core path and likely needs rebasing onto the current plugin-owned discovery flow.
