## Summary

Keep open. Current `main` does not implement the PR’s user-visible feature set: there is no `/sessions` command, `/session` still only handles `idle`/`max-age`, previous transcripts are still archived immediately on rollover instead of retained in switchable LRU history, and the `session.historyLimit` config surface is absent.
