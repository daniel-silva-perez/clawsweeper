## Summary

Keep open. Current `main` has partial Copilot vision support, but it does not register `github-copilot` as a bundled media-understanding provider the way the issue requests. The media-understanding registry only auto-adds config providers that already declare image-capable models, while the implicit Copilot provider config still writes `models: []`.
