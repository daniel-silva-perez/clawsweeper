## Summary

Keep open. Current `main` narrows the original "every CLI command" claim for some routes, but there is no strong evidence of a real fix for the repeated Feishu registration/log-noise path. Non-activating plugin snapshot loads still execute plugin `register(api)`, and the Feishu plugin still logs each surface registration during `registerFull`, so the narrower bug reported in comments remains plausible on current `main`.
