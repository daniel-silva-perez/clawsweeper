## Summary

Keep #71285 open. Current main still has no `memorySearch.ignorePaths` config, the builtin memory indexer still recursively indexes `memory/**/*.md` including `memory/dreaming/**`, the watcher only has hardcoded dependency-dir ignores, and QMD default memory collections still use `memory/**/*.md`. Related #66852 fixed dreaming promotion/self-ingestion, but not search-index exclusion; #68774 is an open narrower MEMORY.md promotion-leak PR and does not supersede this request.

## Best Possible Solution

Keep the issue open and implement a coherent exclusion contract. The likely path is an additive `agents.*.memorySearch.ignorePaths` (or equivalent) in schema, types, docs, resolver, builtin `listMemoryFiles()`, watcher config, and QMD collection handling, with tests for `memory/dreaming/**`, default no-ignore behavior if that remains the product choice, custom ignore patterns, stale row cleanup, and watch/index consistency.
