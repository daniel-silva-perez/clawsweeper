## Summary

Keep open. Current `main` does not implement this PR's behavior: bootstrap injection still follows the existing file order, `HARD_EXECUTION_RULES.md` is not a recognized bootstrap file, and the runtime does not add the proposed per-file skipped-file diagnostics.
