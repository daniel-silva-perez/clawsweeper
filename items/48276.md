## Close Comment

Closing from a Codex review. Current `main` now treats ClawHub as the canonical discovery surface for community plugins, so this docs-only listing PR is out of scope for the core repo. The current policy is explicit in `docs/plugins/community.md:14-16` and again in `docs/plugins/community.md:147-153`: do not open docs-only PRs just to add a plugin for discoverability; publish it on ClawHub instead. That matches the broader project direction in `VISION.md:54-61`, and `docs/tools/clawhub.md:10-14` defines ClawHub as the public registry for OpenClaw skills and plugins. Because of that scope change, this belongs on ClawHub rather than in `openclaw/openclaw` docs.
