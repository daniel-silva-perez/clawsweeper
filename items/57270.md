## Summary

Keep open. Current `main` still has the docs/schema mismatch: docs, generated schema metadata, and TS config types all advertise `tools.exec.approvalRunningNoticeMs`, but the runtime Zod validator for top-level `tools.exec` still omits it, so config load would still reject that key.
