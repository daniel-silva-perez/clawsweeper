## Summary

Keep open: current `main` does not implement the Teams-emoji special case. The inbound path still strips HTML tags to empty text for emoji-only `<img>` payloads, then `buildMSTeamsAttachmentPlaceholder()` counts HTML `<img src=...>` tags as generic images, which matches the bug report rather than superseding it.
