## Summary

Keep open. Current main still lacks Feishu reply-path mediaLocalRoots plumbing, while the PR addresses the real missing path from Feishu bot dispatch through createFeishuReplyDispatcher into sendMediaFeishu. Related open PRs/issues overlap, but none has landed or clearly supersedes this PR in main.

## Best Possible Solution

Keep this PR open or reconcile it into the chosen canonical Feishu fix. The best implementation is to derive getAgentScopedMediaLocalRoots at both Feishu bot dispatcher call sites, add a typed mediaLocalRoots field to CreateFeishuReplyDispatcherParams, forward it into sendMediaFeishu, and add focused tests proving reply attachments carry the allowed roots.
