## Summary

Keep open. Current `main` only fixes transcript-backed token fallback for `/status` and `session_status`; the `openclaw status` summary path still reads session-store totals directly and still renders missing totals as `unknown/<ctx> (?%)`, which matches the reported regression rather than proving it is fixed.
