## Summary

Keep open. The current public docs and issue discussion show an adjacent Mantle path now exists for bearer-token Bedrock access, but the specific requested surface, native `bedrock-converse-stream` proxy access with `Authorization: Bearer <api-key>` and no SigV4 signing, is still described as unsupported. This is a plausible feature request rather than a cleanup close.

## Evidence

- **Issue discussion confirms remaining gap:** A later comment says Mantle covers the official AWS/OpenAI-compatible path, but explicitly notes that a proxy speaking Bedrock Converse format with Bearer auth remains unsupported and would require a separate HTTP transport path that bypasses the AWS SDK.
- **Standard Bedrock provider docs still describe AWS SDK auth, not API-key auth:** The current Bedrock provider documentation says `amazon-bedrock` uses `bedrock-converse-stream` and AWS SDK credential-chain auth, not an API key.
  - file: [docs/providers/bedrock.md](https://github.com/openclaw/openclaw/blob/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75/docs/providers/bedrock.md)
  - command: `web search/open: https://docs.openclaw.ai/providers/bedrock`
  - sha: [d6ef1fcf24ee](https://github.com/openclaw/openclaw/commit/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75)
- **Mantle docs describe a separate provider and different API surface:** The Mantle docs describe `amazon-bedrock-mantle` as a separate provider using an OpenAI-compatible `/v1/chat/completions` surface with bearer-token auth. They also distinguish it from the standard Amazon Bedrock provider, which uses the native Bedrock API.
  - file: [docs/providers/bedrock-mantle.md](https://github.com/openclaw/openclaw/blob/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75/docs/providers/bedrock-mantle.md)

- **Release evidence for adjacent Mantle support:** The v2026.4.5 release notes mention bundled Mantle support for Amazon Bedrock, but that is adjacent coverage rather than proof that native Converse-format Bearer proxy auth was implemented.
  - file: [CHANGELOG.md](https://github.com/openclaw/openclaw/blob/d6ef1fcf24eea3a5985d653a3baf400e32aa1b75/CHANGELOG.md)
