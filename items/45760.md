## Summary

Keep open. Current `main` still excludes `MEMORY.md` from the minimal bootstrap allowlist for subagent/cron sessions, and the repo now documents/tests that long-term memory for those narrower contexts should be accessed via memory tools rather than auto-injected bootstrap.
