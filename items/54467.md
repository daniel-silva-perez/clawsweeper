## Summary

Current `main` does not implement a per-instance Bonjour service-name override. The bundled bonjour plugin always derives the advertised instance name from `machineDisplayName`, `OPENCLAW_MDNS_HOSTNAME` only affects the hostname fallback path, and there is no shipped config/plugin schema field to set a distinct mDNS instance name per gateway profile.

## Evidence

- **Bundled bonjour plugin always supplies an instance name:** The default bonjour plugin passes `formatBonjourInstanceName(ctx.machineDisplayName)` into `startGatewayBonjourAdvertiser`, so the normal runtime path never leaves `opts.instanceName` empty.
  - file: [extensions/bonjour/index.ts:23](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/bonjour/index.ts#L23)
  - command: `nl -ba extensions/bonjour/index.ts | sed -n '1,80p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Advertiser only uses OPENCLAW_MDNS_HOSTNAME as a fallback when instanceName is absent:** `startGatewayBonjourAdvertiser` reads `OPENCLAW_MDNS_HOSTNAME` into `hostname`, but then prefers `opts.instanceName` whenever it is a non-empty string and only falls back to `${hostname} (OpenClaw)` otherwise.
  - file: [extensions/bonjour/src/advertiser.ts:194](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/bonjour/src/advertiser.ts#L194)
  - command: `nl -ba extensions/bonjour/src/advertiser.ts | sed -n '188,215p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Shipped bonjour plugin exposes no config fields:** The bundled plugin manifest declares an empty `configSchema`, so there is no operator-settable bonjour config such as `instanceName` or similar on current main.
  - file: [extensions/bonjour/openclaw.plugin.json:1](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/bonjour/openclaw.plugin.json#L1)
  - command: `sed -n '1,160p' extensions/bonjour/openclaw.plugin.json`

  - file: [src/config/types.gateway.ts:24](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/config/types.gateway.ts#L24)
