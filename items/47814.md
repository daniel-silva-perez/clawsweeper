Keep this PR open. Current main still does not include the named-DM session key helpers, `activeNamedSession` store field, gateway store helpers, or named-DM docs proposed by the PR. The related predecessor #47403 is closed as superseded by this PR, not merged, so it is not a canonical shipped replacement.
