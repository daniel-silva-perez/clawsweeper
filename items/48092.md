## Summary

Current `main` does not already implement ModelScope support. The repo has no bundled `extensions/modelscope` plugin or `docs/providers/modelscope.md` page, the provider docs index does not list ModelScope, auth/env discovery is manifest-driven with no ModelScope manifest to contribute `MODELSCOPE_API_KEY`, and there is no matching ModelScope commit in the checked-out history. This PR is not closeable as already implemented or obsolete; it remains an unmerged feature that would need rebasing to current manifest-first provider architecture.
