## Summary

Keep open. Current `main` still uses the older keyword-fallback threshold in `extensions/memory-core/src/memory/manager.ts`, and the matching regression in `extensions/memory-core/src/memory/index.test.ts` is still present but skipped, so this PR's fix is not implemented on `main`.
