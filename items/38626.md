## Summary

Keep #38626 open. Current main has non-blocking subagent runs, scalar task tracking, terminal delivery, task cancellation, and subagent kill/steer controls, but it still does not implement the requested durable per-subagent lifecycle timeline, artifact-pointer completion summaries, first-failure/recommended-action diagnostics, or idempotent audit-logged cancel/steer/resume supervision controls. Closed #62478 is complementary background-spawn work, not a canonical replacement for this observability scope.

## Best Possible Solution

Keep this open unless a maintainer deliberately narrows the product scope. The best path is to add a durable per-run subagent/task event ledger for spawn, queued/running, tool start/end, warning/error, completion, and supervision actions; persist it in the subagent/task stores; expose it through /subagents info, the subagents tool, openclaw tasks show, and gateway/plugin runtime views; attach artifact pointers and terminal diagnostics; and make cancel, steer, and resume idempotent with auditable action records. Treat #62478 as complementary background-spawn work, not the canonical tracker for this observability request.
