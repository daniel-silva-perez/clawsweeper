## Summary

Keep #7424 open. Current main still has a global model catalog/allowlist under `agents.defaults.models` and per-agent primary/fallback model defaults, but no ordinary agent-scoped model allowlist field, schema, model-list filtering, or complete override enforcement. Related #9010 was closed as a duplicate of #7424, so this remains the canonical tracker.

## Best Possible Solution

Keep #7424 open as the canonical feature request. Add an agent-scoped model allowlist, likely under `agents.list[].model.allowlist` or `agents.list[].models`, and enforce it consistently across config types/schema/generated metadata, allowed-model resolution, `models.list` with agent context, `/model`, `sessions.patch`, OpenAI-compatible model overrides, Control UI model pickers, docs, and regression tests.
