## Summary

Keep open. Current main has not implemented this PR: `HelloOkSchema.auth` is still optional even though the gateway handler now always emits `auth`, and the generated Swift models plus one Swift test call site still reflect the optional contract. The PR is also paired with the same-author open issue #68160, and related open PR #68227 tracks the same remaining work, so maintainers should choose/land one implementation rather than auto-close this one.

## Best Possible Solution

Keep this PR open and have a maintainer resolve the duplicate with #68227 by choosing the better implementation path. The landed change should make `hello-ok.auth` required in the TypeBox schema, regenerate both Swift protocol models, update Swift call sites/tests, and close #68160 after merge.
