## Summary

Keep open. Current `main` still does not implement this PR's behavior: `/status` is explicitly kept off the local execution path, while the slash-menu Enter shortcut only auto-sends commands marked `executeLocal`, so the two-Enter UX described in #45569 still matches the checked-out code.
