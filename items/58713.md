## Summary

Keep open. Current main still routes `sessions_send` and A2A reply/announce turns through existing requester/target session keys, with persisted `inter_session` provenance, and the public config/types expose no ephemeral A2A option. The request is recent, concrete, author is not a maintainer, and it concerns core session-tool behavior rather than a ClawHub-only plugin surface.
