## Summary

Keep this PR open. Current main does not include the unified Amazon Polly/Transcribe/Nova Sonic plugin, and a maintainer explicitly left this active implementation candidate open with concrete blockers to resolve.

## Best Possible Solution

Keep this PR open as the active consolidated Amazon voice/speech integration candidate. The contributor should rebase onto current main, add provider docs/changelog/labeler metadata, fix disabled Transcribe behavior, align Nova Sonic with AWS bidirectional event schemas and OpenClaw realtime audio contracts, address the remaining Polly/Nova/audio review comments, and return with green changed-surface CI. If maintainers decide AWS voice/speech should ship outside the core bundle, make that an explicit ClawHub/plugin placement decision rather than an automated cleanup close.
