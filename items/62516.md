## Summary

Keep open. Current `main` still treats `tools.profile` as a base filter applied before later allowlists, the documented `coding` profile still excludes `browser` and generic plugin tools, and the only current exception is `bundle-mcp`. That means the reported `tools.profile: "coding"` + `agents.list[].tools.allow` case remains consistent with current code rather than fixed on `main`.

## Evidence
