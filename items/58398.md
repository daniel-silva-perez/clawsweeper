## Summary

Keep open. Current main has adjacent compaction/pruning pieces, including tool-result pruning and split/chunk summarization, but the issue asks for a broader Claude-style compaction architecture. Several requested items are not implemented in the searched main surface, and the provided timeline says the related implementation PR #58455 is still open.
