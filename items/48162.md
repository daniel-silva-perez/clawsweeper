## Summary

Keep open. Current `main` has overlapping cache-hit formatting changes and regression tests, but it still mixes cumulative `inputTokens` with last-call `cacheRead`/`cacheWrite`, so the denominator problem called out on the PR is not clearly resolved on `main`.

## Evidence
