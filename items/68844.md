## Summary

Keep this PR open. Current main still implements and tests newline-mode chunking as one blank-line-separated paragraph per outbound message, while this PR is a focused implementation candidate for packing adjacent paragraphs up to the configured limit. The related #42667 is closed and unmerged, so it does not supersede this narrower PR.
