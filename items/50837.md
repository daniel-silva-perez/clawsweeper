## Summary

Keep open. Current `main` does not implement the PR‚Äôs `publicMode` feature or its per-request owner override semantics. The code and docs still describe `/v1/chat/completions` and `/v1/responses` as full operator/owner surfaces under shared-secret auth, and the system prompt still includes the local runtime/workspace details this PR aims to redact.
