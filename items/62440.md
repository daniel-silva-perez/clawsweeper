## Summary

Keep open. Current `main` still sends the local `openclaw agent --local --json` envelope through `runtime.log(...)`, while CLI JSON mode routes console output to `stderr`. The gateway path already uses `writeRuntimeJson(...)` to `stdout`, and the relevant files are unchanged from `v2026.4.23`, so there is no evidence this issue has been fixed on `main` or in the latest release.
