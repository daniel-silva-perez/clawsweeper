## Summary

Current `main` already implements Anthropic Claude 4.6 adaptive thinking and effort mapping. The Anthropic provider advertises `adaptive` for Claude 4.6 and defaults that family to `adaptive`, the Anthropic transport emits `thinking: { type: "adaptive" }` plus `output_config.effort`, and regression tests cover the Opus 4.6 payload shape.
