## Summary

Keep open. Current `main` still has the stale `lastAssistant` fallback path that can replay prior assistant text on an aborted interrupt-mode run, so this is not implemented on `main`. The submitted PR also appears incomplete because its proposed abort-time early return would bypass current timeout payload synthesis and post-compaction bookkeeping.
