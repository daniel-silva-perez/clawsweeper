## Summary

Keep open. Current `main` does not implement the gateway fix described in PR #48714. The resolver path was refactored, but bare persisted runtime or override model names still fall back to the resolved default provider instead of first checking for a unique configured-provider match, and the current tests still encode the old fallback behavior.
