## Summary

Keep open. Current main has a narrower browser-local tweakcn custom theme slot, but it does not ship Matrix or Cyberpunk as first-class Control UI themes, and that custom slot cannot express the requested digital-rain/glitch/scanline or glassmorphism effects. The linked Matrix PR #66387 was closed unmerged, so the proposal still needs maintainer product/design judgment.

## Best Possible Solution

Leave this open for maintainer product/design judgment. A complete core implementation would add Matrix and Cyberpunk as first-class Control UI theme families in ui/src/ui/theme.ts, ui/src/styles/base.css, the full and quick settings selectors, and focused tests, with explicit accessibility and reduced-motion policy. If maintainers prefer external distribution, define or document a supported custom-theme/plugin surface that can cover more than safe color/font tokens before closing this as external work.
