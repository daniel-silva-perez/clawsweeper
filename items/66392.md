## Summary

Keep this open. Current main still has only Claw, Knot, Dash, and a single browser-local tweakcn custom slot; it does not ship Matrix or Cyberpunk as first-class Control UI themes, and the current custom slot intentionally cannot express the requested digital-rain, glitch, scanline, or glassmorphism effects. The linked Matrix PR #66387 was closed unmerged, so this remains a product/design decision rather than cleanup-close material.

## Best Possible Solution

Leave this open for maintainer product/design judgment. If accepted as core UI work, add Matrix and Cyberpunk as first-class Control UI theme families across ui/src/ui/theme.ts, ui/src/styles/base.css, the full and quick settings selectors, and focused tests, with explicit accessibility and prefers-reduced-motion behavior. If maintainers prefer external distribution, first define or document a supported custom-theme/plugin surface that can cover more than safe color/font tokens, including animation/effect policy.
