## Summary

Keep open. Current `main` does not include the PR’s two proposed fixes, and the PR’s own follow-up says its diagnosis is incomplete. The existing code still rejects device-less non-Control-UI `node` sessions, still does not skip backend self-pairing for `authMethod: "none"`, and several cited internal callers still use plain `callGateway(...)` operator/CLI defaults rather than the `node`/backend paths this PR targets.
