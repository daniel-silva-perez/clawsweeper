## Summary

Keep this PR open. Current main still lacks `api.runtime.usage` or an equivalent public plugin SDK path for model pricing resolution, while the internal pricing helpers already exist and are tested. The PR is an active, focused additive runtime API proposal with a concrete ContextClaw plugin use case and recent author follow-up, so it needs maintainer API review rather than cleanup closure.

## Best Possible Solution

Keep this PR open for maintainer review. If maintainers accept the API shape, the best path is a narrow additive `api.runtime.usage` implementation with a rebase to current main, public SDK runtime docs, a small runtime-shape test, regenerated SDK API baseline, and changed-gate proof before merge.
