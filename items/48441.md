## Summary

Keep open. Current main still routes active subagent completion announces into the requester session as the full internal task-completion prompt, and the webchat/TUI history filters only assistant commentary/control-token entries, not inter-session user/internal-context messages. I did not find a main-branch implementation of the requested hidden/internal display boundary for these announces.
