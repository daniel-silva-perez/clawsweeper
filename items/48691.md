## Summary

Keep open. Current main has updater work that safely targets Homebrew-owned npm prefixes, but the status/healthcheck path still does not detect Homebrew as a distinct install method or emit brew-specific update guidance. The reported cleanup is not clearly implemented on `main`.

## Evidence
