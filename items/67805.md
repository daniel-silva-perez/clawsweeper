## Summary

Keep this PR open. Current main still returns empty mcporter search results as final results and does not retry the same query through direct qmd, while this PR is a focused implementation candidate with regression coverage and active author follow-up on the review findings.

## Best Possible Solution

Keep this PR open and review the updated branch. The best fix is in `extensions/memory-core/src/memory/qmd-manager.ts`: when mcporter returns zero results, retry via direct `qmd query`, guard that retry so a fallback failure preserves the valid empty mcporter result, set a distinct debug fallback reason, and keep the regression tests in `extensions/memory-core/src/memory/qmd-manager.test.ts`. If the author's later command-surface consistency concern is required, handle it as part of review or split it into a follow-up rather than closing this focused fix.
