## Summary

Keep open. Current `main` and the latest release still ship the Render blueprint with `plan: starter`, a mounted `/data` disk, and `/data`-backed state/workspace paths. Repo docs explicitly say switching to Render free tier removes persistent storage and causes OpenClaw state to reset on deploy. This PR removes the disk while leaving `OPENCLAW_STATE_DIR` and `OPENCLAW_WORKSPACE_DIR` under `/data`, so there is strong evidence the proposed change is still unsafe rather than already implemented or obsolete.
