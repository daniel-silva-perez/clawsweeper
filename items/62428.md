## Summary

Keep open. Current `main` still does not include several substantive parts of this PR's exec v2 follow-through: mutable `safeBinTrustedDirs` filtering, formal `cat`/`ls` rejection in `safeBins`, interpreter-script path binding for allow-always/allowlist flows, and the matching docs/tests. Some narrower coverage already exists on `main`, but not enough to treat this PR as implemented or obsolete.

## Evidence
