## Summary

Keep open. Current `main` still implements a single JSON5-backed `openclaw.json` config path; there is no YAML loader or `.yaml`/`.yml` discovery path in the config code. The issue is clear, recent, and requests an in-core feature, so none of the allowed close reasons fit.
