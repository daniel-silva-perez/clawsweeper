## Summary

Keep open. Current `main` still eagerly includes configured channel plugins in the gateway startup set, the setup-runtime fast path only applies when a plugin explicitly opts into `startupDeferConfiguredChannelFullLoadUntilAfterListen`, WhatsApp does not opt in, and the gateway's deferred reload path still runs before `startListening()`. I did not find strong evidence that #48644 is already fixed on `main`.

## Evidence
