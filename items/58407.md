## Summary

Keep open. The issue is a concrete, recent Discord feature request from a non-maintainer, and current main has adjacent thread context behavior but does not appear to implement automatic parent-channel pinned-message injection. Discord channel behavior is also within OpenClaw’s current major messaging-channel scope, so this should not be closed as ClawHub-only without maintainer/product judgment.
