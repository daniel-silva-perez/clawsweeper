## Summary

Keep open. Current `main` does not contain the PR's core client-side skill security work: no skill capability metadata or scan status in the skill model, no frontmatter parsing for capabilities, no before-tool-call community-skill enforcement gate, no community-skill capability mapping in security constants, and no CLI/UI/docs surfaces matching the proposed runtime enforcement/trust-tier feature set.
