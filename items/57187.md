## Summary

Current main already fixes the reported OAuth/context1m regression through the newer provider-stream path: resolved runtime API keys are re-injected into provider-owned stream functions before Anthropic beta handling, and released Anthropic wrapper/tests/docs already cover stripping `context-1m` for `sk-ant-oat*` auth. This PR's older-path patch is obsolete.
