## Summary

Keep open. Current `main` has adjacent inspection surfaces for background work (`openclaw tasks`, `/tasks`, `/subagents info`, `/export-trajectory`), but I found no implemented `openclaw timeline` or `/timeline` command and no user-facing interactive timeline view for agent lifecycle events.

## Evidence

- **CLI exposes tasks, not timeline:** The CLI registers `openclaw tasks` and its subcommands (`list`, `audit`, `maintenance`, `show`, etc.), but there is no `timeline` command in the same command set.
  - file: [src/cli/program/register.status-health-sessions.ts:226](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/cli/program/register.status-health-sessions.ts#L226)

- **Chat commands expose trajectory export and subagents, not /timeline:** The command registry includes `/export-trajectory` (`/trajectory`) and `/subagents`, but no `/timeline` command.
  - file: [src/auto-reply/commands-registry.shared.ts:255](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/auto-reply/commands-registry.shared.ts#L255)
