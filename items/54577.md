## Summary

Keep this issue open. Current main still lacks the requested `openclaw timeline` CLI command, `/timeline` chat command, Control UI/TUI activity timeline, and JSON-exportable cross-agent lifecycle timeline. Existing trajectory export, subagent status, usage timelines, generic gateway event logs, and session lifecycle metadata are adjacent observability pieces but do not satisfy the requested product surface. Related #60814 is an open draft Control UI dashboard candidate and remains only a partial implementation path.

## Best Possible Solution

Keep the issue open until maintainers scope and ship the agent activity timeline product. The best path is to either broaden and land #60814 with the missing CLI command, `/timeline` command, TUI/Control UI surface, lifecycle event persistence/export contract, filters, tests, and docs, or split a narrower core/gateway activity-timeline implementation over the existing session lifecycle, subagent, task, agent-event, and trajectory data.
