## Summary

Keep open. Current `main` does not implement content-based agent routing in bindings: binding match types, schema help, CLI/docs, and the routing resolver only support channel/account/peer/guild/team/roles. Existing message-pattern support is limited to group-chat mention gating, which is separate from agent selection and does not provide binding-level `content` matching or `stripMatch`.
