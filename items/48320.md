## Summary

Keep open. Current `main` still does not implement the PR's fix: `src/plugin-sdk/windows-spawn.ts` omits `windowsHide: true` on the `direct` return and on the `shell-fallback` return, and the current test suite still expects `windowsHide` to be `undefined` for shell fallback. This is not implemented on `main` yet.
