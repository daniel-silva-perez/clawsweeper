## Summary

Keep open. Current `main` still routes cron reminder text through the heartbeat model path and explicitly asks the model to relay the reminder to the user, so the reported "ChatGPT as string formatter" behavior is not implemented away on `main`. There is some dedupe/skip logic, but not a core change that removes the model formatting step for scheduled reminders.
