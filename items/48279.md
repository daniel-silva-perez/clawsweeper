## Summary

Keep open. Current `main` still implements `unstable_listSessions` instead of `listSessions`, while `package.json` now pins `@agentclientprotocol/sdk` to `0.19.0`. A repo-wide search found no `listSessions` handler in `src/acp`, so the reported ACP `session/list` break does not appear implemented on `main`.
