## Summary

Keep open. Current `main` does not contain this sanitization, and the local code still routes raw `senderName` through group tool-policy matching. The PR discussion also identified a plausible `toolsBySender` regression while the claimed OpenAI `message.name` root cause is still not proven in the current OpenClaw transport code. For conservative maintainer cleanup, that is not strong enough to close a draft PR.
