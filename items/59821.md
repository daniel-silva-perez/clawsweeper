## Summary

Keep open. The PR is still useful because current main still has the reported behavior: empty `allowBundled` is normalized to `undefined`, and `isBundledSkillAllowed` treats both undefined and empty arrays as unrestricted. The PR is also contributor-authored, open, unmerged, and has no protected labels requiring a different cleanup path.
