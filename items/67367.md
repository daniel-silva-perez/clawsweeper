## Summary

Current OpenClaw main does not expose the zeroclaw config/model described in the report. The cited surfaces (`runtime.kind`, `prompt_injection_mode`, `SKILL.toml`) are absent, current skills are `SKILL.md`-based, current main explicitly injects `skillsPrompt` into the agent system prompt, and current exec/sandbox behavior is host-first when sandboxing is off rather than silently forcing Docker.
