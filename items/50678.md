## Summary

Keep open. Current `main` still hardcodes the subagent prompt text, wires it into spawned runs via `extraSystemPrompt`, and exposes no `agents.defaults.subagents.*promptAppend`-style config. The only documented extension point for prompt mutation is a plugin hook, so the requested operator-facing config is not implemented on `main`.
