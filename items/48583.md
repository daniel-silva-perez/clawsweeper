## Summary

Keep open. Current `main` does not already implement the parent-run subagent cleanup claimed by PR #48583, so this is not `implemented_on_main`. But the branch as described is also not safe to auto-close as obsolete: its cleanup condition would delete any `spawnMode === "run"` child, which conflicts with current `main` semantics where run-mode subagents default to `cleanup: "keep"` and are intentionally retained unless explicitly delete-mode.

## Evidence
