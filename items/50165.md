## Summary

Keep open. Current main added background-task visibility, but the subagent status surfaces still treat a run as done once `endedAt` is set (except for pending descendant subagents), and the lifecycle code finalizes the subagent task before announce cleanup/delivery completes. That does not strongly support closing a report about subagents appearing completed before the user-relevant delegated work is actually finished.
