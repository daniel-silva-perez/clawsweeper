## Summary

Keep open. Current main does not implement the PR's core fix: stale `pluginDebugEntries` are still refreshed from disk for verbose replies, there is no turn-start clear helper, and `updateSessionStoreEntry()` has no preserve-activity option. The PR context is also positive: the bot review described the approach as sound, and the only P2 mutation note was addressed in PR commit `fb267ad48e22299ae851a6629341e8b77f11d53c`.

## Best Possible Solution

Keep #71249 open for maintainer review or merge after normal validation.
