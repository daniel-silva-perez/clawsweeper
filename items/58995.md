## Summary

Keep open. Current main still has the ASCII-only slug normalization that this PR targets, so the request is not implemented on main. The PR is also a plausible bug fix for linked issue #58932, but review comments identify a real follow-up concern around Unicode combining marks, so it should remain open for maintainer/contributor action rather than be closed by cleanup.
