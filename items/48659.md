## Summary

Keep open. Current `main` does not implement this PR end-to-end: the new `msteams-channel-archive` plugin is absent, Teams reply policy still defaults channel replies to `top-level` when `requireMention=false`, the DM recent-channel focus sidecar/readable-label persistence is not present, the webhook still uses `express.json()` without the JSON-escape repair path, and the plugin hook surface still has no `channel_deleted` hook for archive cleanup.
