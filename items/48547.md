## Summary

Current main already routes compaction through the safeguard pipeline by default, which summarizes only prepared conversation messages, hard-caps the stored summary at 16,000 chars, and re-injects only a bounded fresh AGENTS.md excerpt instead of letting model-generated summaries absorb the full system prompt.
