## Summary

Keep open. Current `main` still injects `SOUL.md`, `IDENTITY.md`, `USER.md`, and `MEMORY.md` as ordinary entries under `# Project Context`, with only a one-line SOUL-specific advisory. I found no shipped `context.elevation` config or equivalent prompt-section behavior on `main`, so this is not implemented and there is not strong evidence to close it under another cleanup reason.
