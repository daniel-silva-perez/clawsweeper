## Summary

Keep open. The issue is recent, author association is NONE, and it has no protected labels, but the request is not clearly implemented on current main. Current docs show some related fallback/session reconciliation behavior exists, yet observability is still described as per-attempt provider/model/reason summaries rather than closed-loop effective_model/effective_provider/effective_thinking telemetry for sub-agent pin fidelity.
