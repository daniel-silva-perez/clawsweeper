## Summary

Current `main` does not implement this PR's behavior. Gemini web search still only uses scoped `webSearch.apiKey` or `GEMINI_API_KEY`, still hardcodes the Google base URL, and still exposes no `webSearch.baseUrl` metadata/docs. The PR remains actionable rather than closeable.

## Evidence
