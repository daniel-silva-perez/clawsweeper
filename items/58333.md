## Summary

Keep PR #58333 open. Current main still has the broad descendant chevron selectors described by the PR, and the config renderer still creates nested details/chevron DOM where an open parent can rotate collapsed child chevrons. The PR is not obsolete, but it should address the existing review comments before merge.

Then run the appropriate UI changed gate before merge.
