## Summary

Current `main` still exposes `agents.defaults.timeoutSeconds` in the agent-defaults schema, but it does not have a matching human-readable label in `FIELD_LABELS`, and the generated schema metadata for that field still lacks a `title`. The PR addresses a real, still-open gap; there is not strong evidence to close it.
