## Summary

Keep open. Current `main` still has the CLI/UI split described in #48483: the gateway `models.list` handler serves the full gateway catalog unless an agent allowlist trims it, while the Control UI dropdown consumes that RPC directly. The CLI `openclaw models list` default path still builds rows from configured entries instead of the gateway catalog, so the mismatch remains on `4013c658537ee2051c44186f197b0710e84c9049`.

## Evidence
