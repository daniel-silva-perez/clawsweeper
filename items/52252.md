## Summary

Keep open. Current main still only rewrites Homebrew Cellar Node paths; it does not rewrite Volta `tools/image/node/<version>/bin/node` paths to a stable shim. PR #52252 remains a plausible fix for the linked open issue #52184, but the provided review threads identify real coverage gaps around nondefault `VOLTA_HOME` and Windows Volta layouts, so it should be revised or superseded rather than auto-closed.

## Best Possible Solution

Keep #52252 open or replace it with a narrower maintainer-approved fix. The implementation should extend `resolveStableNodePath` to rewrite Volta image-node paths to the stable shim for default macOS/Linux homes, custom `VOLTA_HOME`, and Windows Volta layouts where supported, then add regression coverage in `src/infra/stable-node-path.test.ts` and daemon runtime-path tests before landing.
