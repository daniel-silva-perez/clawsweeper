## Summary

Keep #72166 open. Current `main` still lacks `cron.enabled` and `agents.defaults.compaction.mode` in the agent-facing gateway config mutation allowlist, while both are real config fields. Related PR #72215 is a focused implementation candidate, but it is still open and not merged, so the request remains valid rather than implemented or obsolete.

## Best Possible Solution

Keep the issue open as the canonical request until maintainers merge #72215 or an equivalent scoped change. If accepted, add exactly `cron.enabled` and `agents.defaults.compaction.mode` to `ALLOWED_GATEWAY_CONFIG_PATHS` with positive guard coverage. If rejected, document that direct operator-admin `config.patch` or `openclaw config set` is the supported programmatic path while the agent-facing gateway tool remains narrow.
