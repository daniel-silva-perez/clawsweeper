## Summary

Keep open. Current `main` still uses `fs.promises.rename` in `rotateSessionFile`, current tests still expect `sessions.json` to disappear during rotation, and `loadSessionStore` still falls back to `{}` on read failure. That means the PR is not implemented on `main`, and the reported race remains plausible rather than obsolete.
