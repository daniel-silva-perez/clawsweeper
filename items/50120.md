## Summary

Keep open. Current `main` still uses custom global WebKit scrollbar styling in the Control UI and the chat pane remains a custom scroll container, so there is no strong evidence that the macOS Chrome scrollbar issue was fixed. The latest release `v2026.4.22` shows the same relevant CSS, and the existing browser tests cover auto-scroll behavior, not scrollbar visibility or macOS always-show-scrollbar behavior.
